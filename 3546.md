[Problem: 3546. 等和矩阵分割 I](https://leetcode.cn/problems/equal-sum-grid-partition-i/description/)

### 方法：前缀和 & 模拟

题意很清晰，就是对矩阵切一刀，让两部分的总和相同。

如何快速计算某几列/行的总和？拆分来看，连续的四列总和 **等价于**：分别对每一列求和，然后累加。也就是说，最好提前计算出每行/列的总和，因此联想 **前缀和**，具体可以看这篇 [题解](https://leetcode.cn/problems/minimum-operations-to-make-all-array-elements-equal/solutions/2842529/pai-xu-qian-zhui-he-er-fen-fei-chang-qin-biux/)。

1）首先，计算行/列的 **总和数组**：

- $rowSums$：第 $i$ 个元素是矩阵中第 $i$ 行的所有元素之和。
- $colSums$：第 $j$ 个元素是矩阵中第 $j$ 列的所有元素之和。

2）然后，利用总和数组，构建 **前缀和数组**：

- $rowPrefix[i]$ 表示前 $i+1$ 行的总和。
- $colPrefix[j]$ 表示前 $j+1$ 列的总和。

3）最后，进行 **总判断**：

- 矩阵总和 $S$ 必须是偶数。
- 如果存在 $rowPrefix[i] = S / 2$，且 $i$ 不是最后一行，则水平分割成功。
- 如果存在 $colPrefix[j] = S / 2$，且 $j$ 不是最后一列，则垂直分割成功。

代码如下，已附加注释：

```Python
# python
class Solution:
    def canPartitionGrid(self, grid: List[List[int]]) -> bool:
        m = len(grid)
        n = len(grid[0])
        # row_sums[i] 是第 i 行的和
        row_sums = [sum(row) for row in grid]
        # col_sums[j] 是第 j 列的和
        col_sums = [0] * n
        for r in range(m):
            for c in range(n):
                col_sums[c] += grid[r][c]
        
        # 矩阵总和
        total_sum = sum(row_sums)
        # 总和是奇数
        if total_sum % 2 != 0:
            return False
        target = total_sum // 2
        
        # 计算行前缀和
        # row_prefix[i] 表示从第 0 行到第 i 行的总和
        row_prefix = [0] * m
        row_prefix[0] = row_sums[0]
        for i in range(1, m):
            row_prefix[i] = row_prefix[i-1] + row_sums[i]
        
        # 检查
        for i in range(m - 1):
            if row_prefix[i] == target:
                return True
        
        # 计算列前缀
        # col_prefix[j] 表示从第 0 列到第 j 列的总和
        col_prefix = [0] * n
        col_prefix[0] = col_sums[0]
        for j in range(1, n):
            col_prefix[j] = col_prefix[j-1] + col_sums[j]
        
        # 检查
        for j in range(n - 1):
            if col_prefix[j] == target:
                return True
        
        return False
```

```Java
// java
class Solution {
    public boolean canPartitionGrid(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;
        
        // row_sums[i] 是第 i 行的和
        long[] row_sums = new long[m];
        for (int i = 0; i < m; i++) {
            long sum = 0;
            for (int val : grid[i]) {
                sum += val;
            }
            row_sums[i] = sum;
        }
        
        // col_sums[j] 是第 j 列的和
        long[] col_sums = new long[n];
        for (int r = 0; r < m; r++) {
            for (int c = 0; c < n; c++) {
                col_sums[c] += grid[r][c];
            }
        }
        
        // 矩阵总和
        long total_sum = 0;
        for (long val : row_sums) {
            total_sum += val;
        }
        
        // 总和是奇数
        if (total_sum % 2 != 0) {
            return false;
        }
        long target = total_sum / 2;
        
        // 计算行前缀和
        // row_prefix[i] 表示从第 0 行到第 i 行的总和
        long[] row_prefix = new long[m];
        row_prefix[0] = row_sums[0];
        for (int i = 1; i < m; i++) {
            row_prefix[i] = row_prefix[i - 1] + row_sums[i];
        }
        
        // 检查
        for (int i = 0; i < m - 1; i++) {
            if (row_prefix[i] == target) {
                return true;
            }
        }
        
        // 计算列前缀
        // col_prefix[j] 表示从第 0 列到第 j 列的总和
        long[] col_prefix = new long[n];
        col_prefix[0] = col_sums[0];
        for (int j = 1; j < n; j++) {
            col_prefix[j] = col_prefix[j - 1] + col_sums[j];
        }
        
        // 检查
        for (int j = 0; j < n - 1; j++) {
            if (col_prefix[j] == target) {
                return true;
            }
        }
        
        return false;
    }
}
```

- 时间复杂度： $O(m\times n)$，其中 $m,n$ 是矩阵 $grid$ 的行数、列数
- 空间复杂度： $O(m+n)$，表示四个数组 $sums,prefix$ 的大小

### 模拟

实际上，不需要创建前缀和数组，而是通过 **模拟** 大法，检查时再手动累加。

1. 遍历整个矩阵，计算矩阵总和 $S$。
2. 若 $S \% 2 \neq 0$，返回 $False$，反之目标 $target = S // 2$。
3. 逐行累加行的总和，只要累加到 $target$ 且不是最后一行，就成功。
4. 逐列累加列的总和，只要累加到 $target$ 且不是最后一列，就成功。

这种解法比上述慢慢累加前缀和的解法更快。

代码如下，已附加注释：

```Python
# python
class Solution:
    def canPartitionGrid(self, grid: List[List[int]]) -> bool:
        # 计算矩阵总和
        total_sum = sum(sum(row) for row in grid)
        
        # 总和是奇数
        if total_sum % 2 != 0:
            return False
        
        target = total_sum // 2
        
        # 检查水平分割
        # 遍历到倒数第二行
        row_target = target
        for i in range(len(grid) - 1):
            row_target -= sum(grid[i])
            if row_target == 0:
                return True
            if row_target < 0:  # 后续都是小于0
                break
        
        # 检查垂直分割
        # 遍历到倒数第二列
        col_target = target
        for col in zip(*grid):
            col_target -= sum(col)
            if col_target == 0:
                return True
            if col_target < 0:  # 优化
                break
        
        return False
```

```Java
// java
class Solution {
    public boolean canPartitionGrid(int[][] grid) {
        // 计算矩阵总和
        long total_sum = 0;
        for (int[] row : grid) {
            for (int val : row) {
                total_sum += val;
            }
        }
        
        // 总和是奇数
        if (total_sum % 2 != 0) {
            return false;
        }
        
        long target = total_sum / 2;
        
        // 检查水平分割
        // 遍历到倒数第二行
        long row_target = target;
        for (int i = 0; i < grid.length - 1; i++) {
            for (int val : grid[i]) {
                row_target -= val;
            }
            if (row_target == 0) {
                return true;
            }
            if (row_target < 0) {  // 后续都是小于0
                break;
            }
        }
        
        // 检查垂直分割
        // 遍历到倒数第二列
        long col_target = target;
        int n = grid[0].length;
        for (int j = 0; j < n - 1; j++) {
            for (int i = 0; i < grid.length; i++) {
                col_target -= grid[i][j];
            }
            if (col_target == 0) {
                return true;
            }
            if (col_target < 0) {  // 优化
                break;
            }
        }
        
        return false;
    }
}
```

- 时间复杂度： $O(m\times n)$
- 空间复杂度： $O(m)$，表示 $Python$ 创建切片的开销，而 $Java$ 为 $O(1)$
