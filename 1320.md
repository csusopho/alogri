[Problem: 1320. 二指输入的的最小距离](https://leetcode.cn/problems/minimum-distance-to-type-a-word-using-two-fingers/description/)

### 方法：记忆化 & DP

如何快速得知某个字母在平面中的坐标？

根据题目描述，可知 $A-Z$ 依次排列，每行 $6$ 个字母：

- 字母 $char$ 的索引 $dist = ord(char) - ord('A')$
- 行坐标 $r = index // 6$
- 列坐标 $c = index \% 6$

从坐标 $(r_1,c_1)$ 移动到坐标 $(r_2,c_2)$ 的成本是多少？

本质是曼哈顿距离： $dist(P_1, P_2) = |r_1 - r_2| + |c_1 - c_2|$。

由于我们必须按顺序输入 $word$ 中的每一个字符，**在准备输入第 $i$ 个字符时，其中一根手指必然停留在第 $i-1$ 个字符上。**

为了简化状态，不需要记录两根手指的位置，只需要记录：

- $idx$：当前正要输入的字符在 $word$ 中的下标。
- $otherPos$：**另一根** 不在 $word[idx-1]$ 上的手指目前所在的位置（可以是 $26$ 个字母之一，或者是一个特殊状态 “未放置”）。

因此，**定义** $dfs(idx, otherPos)$ 表示输入从 $word[idx]$ 到结尾的所有字符，另一根手指在 $otherPos$ 位置时，所需的总最小移动距离。

---

准备输入 $word[idx]$ 时，面临两个选择：

**选择 A：移动上一刻输入 $word[idx-1]$ 的那根手指。**

此时，移动距离是 $dist(word[idx-1], word[idx])$，这是上面曼哈顿的距离公式。移动后，另一根手指的位置 $otherPos$ 保持不变。下一步变为 $dfs(idx + 1, otherPos)$。转移方程为：

$$
dfs(idx, otherPos) = dist(word[idx-1], word[idx]) + dfs(idx+1, otherPos)
$$

**选择 B：移动处于 $otherPos$ 的那根手指。**

此时，移动距离是 $dist(otherPos, word[idx])$。移动后，原本在 $word[idx-1]$ 上的那根手指变成了 “另一根手指”。下一步变为 $dfs(idx + 1, word[idx-1])$。转移方程为：

$$
dfs(idx, otherPos) = dist(otherPos, word[idx]) + dfs(idx+1, word[idx-1])$$

**边界条件**：当 $idx = n$ 时，说明全部输入完成，返回距离 $0$。

**递归入口**：第一个字符 $word[0]$ 的输入是零代价的。输入完 $word[0]$ 后，一根手指在 $word[0]$，另一根在 “未放置” 状态。所以初始调用为 $dfs(1, None)$ 或者 $dfs(1,-1)$。

整个递归中有大量重复递归调用（递归参数相同）。由于 **递归函数没有副作用**，同样的入参无论计算多少次，算出来的结果都是一样的，因此可以用记忆化搜索来优化。

记忆化的 **本质** 就是减少分支数，避免重复搜索，降低时间复杂度。递归的参数只有两个，所以记忆化数组需要两维。 $Java$ 中使用数组记录， $Python$ 中直接使用 $@cache$ 装饰器即可。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minimumDistance(self, word: str) -> int:
        n = len(word)

        # 预计算坐标映射
        pos = {}
        for i in range(26):
            pos[i] = (i // 6, i % 6)

        # 移动手指，曼哈顿距离
        def get_dist(p1_idx, p2_idx):
            # 手指还没放在键盘上
            if p1_idx == -1 or p2_idx == -1:
                return 0
            r1, c1 = pos[p1_idx]
            r2, c2 = pos[p2_idx]
            return abs(r1 - r2) + abs(c1 - c2)

        # 转为索引列表
        nums = [ord(c) - ord('A') for c in word]

        @cache
        def dfs(idx, other_pos):
            # 边界
            if idx == n:
                return 0

            # 当前要输入的字符
            curr = nums[idx]
            # 第一根手指所在字符
            prev = nums[idx-1]

            # 移动上一根手指
            res1 = get_dist(prev, curr) + dfs(idx + 1, other_pos)
            # 移动另一根手指
            res2 = get_dist(other_pos, curr) + dfs(idx + 1, prev)
            return min(res1, res2)

        # 一根手指在nums[0]，另一根在 -1
        return dfs(1, -1)
```

```Java
// java
class Solution {
    private int[][] pos = new int[26][2];
    private int[] nums;
    private Integer[][] memo;
    private int n;

    // 移动手指，曼哈顿距离
    private int get_dist(int p1_idx, int p2_idx) {
        // 手指还没放在键盘上
        if (p1_idx == 26 || p2_idx == 26) {
            return 0;
        }
        return Math.abs(pos[p1_idx][0] - pos[p2_idx][0]) + Math.abs(pos[p1_idx][1] - pos[p2_idx][1]);
    }

    private int dfs(int idx, int other_pos) {
        // 边界
        if (idx == n) {
            return 0;
        }
        if (memo[idx][other_pos] != null) {
            return memo[idx][other_pos];
        }

        // 当前要输入的字符
        int curr = nums[idx];
        // 第一根手指所在字符
        int prev = nums[idx - 1];

        // 移动上一根手指
        int res1 = get_dist(prev, curr) + dfs(idx + 1, other_pos);
        // 移动另一根手指
        int res2 = get_dist(other_pos, curr) + dfs(idx + 1, prev);

        return memo[idx][other_pos] = Math.min(res1, res2);
    }

    public int minimumDistance(String word) {
        n = word.length();

        // 预计算坐标映射
        for (int i = 0; i < 26; i++) {
            pos[i][0] = i / 6;
            pos[i][1] = i % 6;
        }

        // 转为索引列表
        nums = new int[n];
        for (int i = 0; i < n; i++) {
            nums[i] = word.charAt(i) - 'A';
        }

        memo = new Integer[n][27];
        // 一根手指在nums[0]，另一根在 -1
        return dfs(1, 26);
    }
}
```

记忆化后，每个状态只会计算一次。时间复杂度 = 状态个数 $\times$ 单个状态的计算时间。参数一共两个，状态个数等于 $O(n\times |\Sigma|)$，单个状态的计算时间为 $O(1)$，所以总时间复杂度为 $O(n\times |\Sigma|)$。

- 时间复杂度： $O(n\times |\Sigma|)$，其中 $n$ 是串 $word$ 的长度，而 $|\Sigma|=26$ 表示字符集大小
- 空间复杂度： $O(n\times |\Sigma|)$，保存多少状态，就需要多少空间

### 动态规划

将递归 $1:1$ 转化为递推。

**定义** $dp[i][j]$ 表示：输入完 $word[0...i]$ 后，且一根手指在 $word[i]$，另一根手指在 $j$ 时的最小总距离。

从状态 $i-1$ 转移到 $i$，上一时刻的手指状态是：一根在 $word[i-1]$，另一根在 $j$。现在我们要输入 $curr = word[i]$。一共两种决策：

**决策 A：移动位于 $word[i-1]$ 的那根手指去 $curr$**

$$
dp[i] [j] = \min(dp[i] [j], \ dp[i-1] [j] + \text{dist}(word[i-1], word[i]))
$$

**决策 B：移动位于 $j$ 的那根手指去 $curr$**

$$
dp[i][word[i-1]] = \min(dp[i][word[i-1]], \ dp[i-1] [j] + \text{dist}(j, word[i]))
$$

**初始状态**：输入第一个字母 $word[0]$ 时，两根手指都可以从任意位置起步，代价为 $0$。因此 $dp[0][26] = 0$，其余 $dp[0][...]$ 初始化为无穷大 $+\infty$。

上面的距离函数中，未放置是 $-1$，这里为了下标对应，字符集总共是 $0-25$，所以未放置改为 $26$，同样取不到。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minimumDistance(self, word: str) -> int:
        n = len(word)

        # 预计算坐标
        def get_pos(idx):
            return (idx // 6, idx % 6)

        # 曼哈顿距离
        def get_dist(idx1, idx2):
            if idx1 == 26 or idx2 == 26:
                return 0
            r1, c1 = get_pos(idx1)
            r2, c2 = get_pos(idx2)
            return abs(r1 - r2) + abs(c1 - c2)

        # 转换单词为索引
        nums = [ord(c) - ord('A') for c in word]

        # dp[i][j] 表示输入完 nums[i] 后，另一根手指在 j 的的最小总距离
        dp = [[float('inf')] * 27 for _ in range(n)]

        # 输入完第 0 个字符，另一根手指还没放
        dp[0][26] = 0

        for i in range(1, n):
            prev = nums[i-1]
            curr = nums[i]

            for j in range(27):
                if dp[i-1][j] == float('inf'):
                    continue

                # 移动在 prev 的手指到 curr
                d_a = get_dist(prev, curr)
                dp[i][j] = min(dp[i][j], dp[i-1][j] + d_a)

                # 移动在 j 的手指到 curr
                d_b = get_dist(j, curr)
                dp[i][prev] = min(dp[i][prev], dp[i-1][j] + d_b)

        # “另一根手指” 的位置任意
        return min(dp[n-1])
```

```Java
// java
class Solution {
    // 预计算坐标
    private int get_pos_r(int idx) { return idx / 6; }
    private int get_pos_c(int idx) { return idx % 6; }

    // 曼哈顿距离
    private int get_dist(int idx1, int idx2) {
        if (idx1 == 26 || idx2 == 26) {
            return 0;
        }
        return Math.abs(get_pos_r(idx1) - get_pos_r(idx2)) + Math.abs(get_pos_c(idx1) - get_pos_c(idx2));
    }

    public int minimumDistance(String word) {
        int n = word.length();

        // 转换单词为索引
        int[] nums = new int[n];
        for (int i = 0; i < n; i++) {
            nums[i] = word.charAt(i) - 'A';
        }

        // dp[i][j] 表示输入完 nums[i] 后，另一根手指在 j 的的最小总距离
        int[][] dp = new int[n][27];
        for (int i = 0; i < n; i++) {
            Arrays.fill(dp[i], Integer.MAX_VALUE);
        }

        // 输入完第 0 个字符，另一根手指还没放
        dp[0][26] = 0;

        for (int i = 1; i < n; i++) {
            int prev = nums[i - 1];
            int curr = nums[i];

            for (int j = 0; j < 27; j++) {
                if (dp[i - 1][j] == Integer.MAX_VALUE) continue;

                // 移动在 prev 的手指到 curr
                int d_a = get_dist(prev, curr);
                dp[i][j] = Math.min(dp[i][j], dp[i - 1][j] + d_a);

                // 移动在 j 的手指到 curr
                int d_b = get_dist(j, curr);
                dp[i][prev] = Math.min(dp[i][prev], dp[i - 1][j] + d_b);
            }
        }

        // “另一根手指” 的位置任意
        int res = Integer.MAX_VALUE;
        for (int j = 0; j < 27; j++) {
            res = Math.min(res, dp[n - 1][j]);
        }
        return res;
    }
}
```

- 时间复杂度： $O(n\times |\Sigma|)$
- 空间复杂度： $O(n\times |\Sigma|)$
