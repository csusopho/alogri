[Problem: 2615. 等值距离和](https://leetcode.cn/problems/sum-of-distances/description/)

### 方法：前缀和

假设数组中数值 $v$ 出现的所有下标为 $p_0, p_1, p_2, \dots, p_{k-1}$（从小到大排序）。

对于下标 $p_i$，$arr[p_i]$ 的定义是：

$$
arr[p_i] = \sum_{j=0, j \neq i}^{k-1} |p_i - p_j|
$$

由于下标是有序的 $p_0 < p_1 < \dots < p_{k-1}$，我们可以去掉绝对值符号：

1. 对于所有 $j < i$ 的项，$p_i > p_j$，所以 $|p_i - p_j| = p_i - p_j$。
2. 对于所有 $j > i$ 的项，$p_i < p_j$，所以 $|p_i - p_j| = p_j - p_i$。

因此：

$$
arr[p_i] = \sum_{j=0}^{i-1} (p_i - p_j) + \sum_{j=i+1}^{k-1} (p_j - p_i)
$$

将上面的两个求和部分分别化简：

**前半部分（左侧距离之和）：**

$$
\sum_{j=0}^{i-1} (p_i - p_j) = (i \cdot p_i) - \sum_{j=0}^{i-1} p_j
$$

这里的 $\sum_{j=0}^{i-1} p_j$ 是前 $i$ 个下标的累加和（前缀和）。

**后半部分（右侧距离之和）：**

$$
\sum_{j=i+1}^{k-1} (p_j - p_i) = \left( \sum_{j=i+1}^{k-1} p_j \right) - (k - 1 - i) \cdot p_i
$$

这里的 $\sum_{j=i+1}^{k-1} p_j$ 是剩余下标的累加和（可以用 “总和 - 前缀和 - 当前值” 算出）。

**合并公式：** 设 $S$ 为数值 $v$ 的所有下标之和， $Pre$ 为前 $i$ 个下标之和（即 $\sum_{j=0}^{i-1} p_j$）：

$$
arr[p_i] = (i \cdot p_i - Pre) + [(S - Pre - p_i) - (k - 1 - i) \cdot p_i]
$$

因此，解题步骤为

- 首先进行预处理分组：使用哈希表（字典）记录每个数值出现的所有下标。
- 接着，遍历哈希表中的每一组下标。对每一组下标，利用前缀和公式计算。

代码如下，已附加注释：

```Python
# python
class Solution:
    def distance(self, nums: List[int]) -> List[int]:
        n = len(nums)
        res = [0] * n
        # 记录每个数值出现的所有下标
        groups = defaultdict(list)
        for i, val in enumerate(nums):
            groups[val].append(i)
        
        # 处理每一组
        for val in groups:
            indices = groups[val]
            k = len(indices)
            
            # 下标总和
            total_sum = sum(indices)
            # 当前遍历过的前缀和
            prefix_sum = 0
            
            # 遍历下标
            for i, p_i in enumerate(indices):
                # 左侧部分
                left_val = i * p_i - prefix_sum
                
                # 右侧部分
                right_sum = total_sum - prefix_sum - p_i
                right_count = k - 1 - i
                right_val = right_sum - right_count * p_i
                
                # 存入
                res[p_i] = left_val + right_val
                # 更新前缀和
                prefix_sum += p_i
        
        return res
```

```Java
// java
class Solution {
    public long[] distance(int[] nums) {
        int n = nums.length;
        long[] res = new long[n];
        // 记录每个数值出现的所有下标
        Map<Integer, List<Integer>> groups = new HashMap<>();
        for (int i = 0; i < n; i++) {
            groups.computeIfAbsent(nums[i], k -> new ArrayList<>()).add(i);
        }
        
        // 处理每一组
        for (List<Integer> indices : groups.values()) {
            int k = indices.size();
            
            // 下标总和
            long total_sum = 0;
            for (int idx : indices) {
                total_sum += idx;
            }
            // 当前遍历过的前缀和
            long prefix_sum = 0;
            
            // 遍历下标
            for (int i = 0; i < k; i++) {
                int p_i = indices.get(i);
                // 左侧部分
                long left_val = (long) i * p_i - prefix_sum;
                
                // 右侧部分
                long right_sum = total_sum - prefix_sum - p_i;
                int right_count = k - 1 - i;
                long right_val = right_sum - (long) right_count * p_i;
                
                // 存入
                res[p_i] = left_val + right_val;
                // 更新前缀和
                prefix_sum += p_i;
            }
        }
        
        return res;
    }
}
```

- 时间复杂度： $O(n)$，其中 $n$ 是数组 $queri$ 的长度，建立哈希表预处理一遍，计算距离时每个下标只会被访问一次
- 空间复杂度： $O(n)$，表示哈希表 $groups$ 大小，返回值 $res$ 一般不计入复杂度
