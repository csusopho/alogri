[Problem: 3653. 区间乘法查询后的异或 I](https://leetcode.cn/problems/xor-after-range-multiplication-queries-i/description/)

### 方法：模拟

一句话解释：对 $nums[li, ri]$ 内的元素，按照步长 $ki$，分别乘以 $vi$。

举个例子，比如 $q=[1,5,2,4]$，那就是对 $nums[1],nums[1+2],nums[1+2+2]$ 这三个元素分别乘以 $4$。

本题数据范围较小，完全可以 **模拟** 整个操作流程：

- 对于当前的查询，初始化索引 $idx = li$。
- 将 $nums[idx]$ 乘以 $vi$。
- 对结果取模 $10^9 + 7$。
- 将 $idx$ 增加步长 $ki$，直至 $idx>ri$ 才结束当前查询。

处理完所有查询后，遍历修改后的 $nums$ 数组，将所有元素进行按位异或 $XOR$ 运算。

代码如下，已附加注释：

```Python
# python
class Solution:
    def xorAfterQueries(self, nums: List[int], queries: List[List[int]]) -> int:
        MOD = 10**9 + 7

        for li, ri, ki, vi in queries:
            idx = li
            # 按照 ki 的步长在 [li, ri] 范围内循环
            while idx <= ri:
                # 更新
                nums[idx] = (nums[idx] * vi) % MOD
                # 跳跃
                idx += ki
        
        # 异或
        ans = 0
        for x in nums:
            ans ^= x
        return ans
```

```Java
// java
class Solution {
    public int xorAfterQueries(int[] nums, int[][] queries) {
        long MOD = 1000000007L;

        for (int[] query : queries) {
            int li = query[0];
            int ri = query[1];
            int ki = query[2];
            int vi = query[3];
            
            int idx = li;
            // 按照 ki 的步长在 [li, ri] 范围内循环
            while (idx <= ri) {
                // 更新
                nums[idx] = (int) (((long) nums[idx] * vi) % MOD);
                // 跳跃
                idx += ki;
            }
        }

        // 异或
        int ans = 0;
        for (int x : nums) {
            ans ^= x;
        }
        return ans;
    }
}
```

- 时间复杂度： $O(q\times \frac{n}{k})$，其中 $q$ 表示数组 $queries$ 的长度，这里的 $k$ 表示平均次数，最坏情况下，所有查询的步长都是 $1$，复杂度变为 $O(q \times n)$
- 空间复杂度： $O(1)$，仅使用常数个额外变量
