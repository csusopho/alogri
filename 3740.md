[Problem: 3740. 三个相等元素之间的最小距离 I](https://leetcode.cn/problems/minimum-distance-between-three-equal-elements-i/description/)

### 方法：暴力 & 哈希

题意很清晰，**暴力** 解法：穷举所有可能的三元组组合，计算每一个符合条件的三元组距离，并从中找出最小值。

- 第一层循环遍历下标 $i$（从 $0$ 到 $n-3$）
- 第二层循环遍历下标 $j$（从 $i+1$ 到 $n-2$）
- 第三层循环遍历下标 $k$（从 $j+1$ 到 $n-1$）

检查 $nums[i]、nums[j]、nums[k]$ 是否全部相等。如果相等，根据公式 `abs(i - j) + abs(j - k) + abs(k - i)` 计算当前距离。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minimumDistance(self, nums: List[int]) -> int:
        n = len(nums)
        ans = float('inf')
        found = False
        
        # 第一个数
        for i in range(n):
            # 第二个数
            for j in range(i + 1, n):
                # 第三个数
                for k in range(j + 1, n):
                    # 检查
                    if nums[i] == nums[j] == nums[k]:
                        found = True
                        # 公式
                        cur_dist = abs(i - j) + abs(j - k) + abs(k - i)
                        
                        if cur_dist < ans:
                            ans = cur_dist
        
        return ans if found else -1
```

```Java
// java
class Solution {
    public int minimumDistance(int[] nums) {
        int n = nums.length;
        int ans = Integer.MAX_VALUE;
        boolean found = false;
        
        // 第一个数
        for (int i = 0; i < n; i++) {
            // 第二个数
            for (int j = i + 1; j < n; j++) {
                // 第三个数
                for (int k = j + 1; k < n; k++) {
                    // 检查
                    if (nums[i] == nums[j] && nums[j] == nums[k]) {
                        found = true;
                        // 公式
                        int cur_dist = Math.abs(i - j) + Math.abs(j - k) + Math.abs(k - i);
                        
                        if (cur_dist < ans) {
                            ans = cur_dist;
                        }
                    }
                }
            }
        }
        
        return found ? ans : -1;
    }
}
```

从 $n$ 个元素中选择 $3$ 个元素，组合数为 $C(n, 3)$。根据数学公式可知，结果为 $\frac{n(n-1)(n-2)}{6}$，因此时间复杂度为 $O(n^3)$。

- 时间复杂度： $O(n^3)$，其中 $n$ 是数组 $nums$ 的长度
- 空间复杂度： $O(1)$，仅使用常数个额外变量

### 优化

假设三个不同的下标为 $i, j, k$，且不失一般性地假设其大小顺序为 $i < j < k$。

有效三元组的距离定义为：
$$D = |i - j| + |j - k| + |k - i|$$

由于 $i < j < k$，我们可以去掉绝对值符号：

1. $|i - j| = j - i$
2. $|j - k| = k - j$
3. $|k - i| = k - i$

将这三项相加：

$$
D = (j - i) + (k - j) + (k - i) = 2(k - i)
$$

**结论：** 有效三元组的距离只取决于 **最大下标** 与 **最小下标** 的差值的两倍。中间的下标 $j$ 只要存在于 $i$ 和 $k$ 之间即可，不影响最终距离。

为了使 $2(k - i)$ 最小，我们需要找到同一个数值中，**下标最接近** 的三个位置。对于同一个数字的下标序列（升序排列），最小的 $k-i$ 一定出现在 **相邻的** 三个下标中。

---

**分组**：遍历数组，使用哈希表存储每个数字出现的所有下标。键是数字，值是该数字出现下标的列表。

由于我们是从前往后遍历，所以哈希表中存储的下标一定升序！

遍历哈希表中的每一个下标列表：

- 如果某个数字出现的次数少于 $3$ 次，无法组成三元组。
- 如果出现次数 $\ge 3$，遍历该下标列表。对于每一个跨度为 $2$ 的下标对 $(p[a], p[a+2])$，计算距离 $2 \times (p[a+2] - p[a])$。

**更新**：记录并返回所有可能三元组中的最小距离。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minimumDistance(self, nums: List[int]) -> int:
        # 哈希，记录每个数字出现的所有下标
        pos_map = defaultdict(list)
        for index, num in enumerate(nums):
            pos_map[num].append(index)
        
        ans = float('inf')
        found = False
        
        for num in pos_map:
            indices = pos_map[num]
            # 必须次数满足
            if len(indices) >= 3:
                found = True
                # 检查所有相邻的三个下标
                for a in range(len(indices) - 2):
                    cur_dist = 2 * (indices[a+2] - indices[a])
                    if cur_dist < ans:
                        ans = cur_dist
        
        return int(ans) if found else -1
```

```Java
// java
class Solution {
    public int minimumDistance(int[] nums) {
        // 哈希，记录每个数字出现的所有下标
        Map<Integer, List<Integer>> pos_map = new HashMap<>();
        for (int index = 0; index < nums.length; index++) {
            pos_map.computeIfAbsent(nums[index], k -> new ArrayList<>()).add(index);
        }
        
        int ans = Integer.MAX_VALUE;
        boolean found = false;
        
        for (List<Integer> indices : pos_map.values()) {
            // 必须次数满足
            if (indices.size() >= 3) {
                found = true;
                // 检查所有相邻的三个下标
                for (int a = 0; a < indices.size() - 2; a++) {
                    int cur_dist = 2 * (indices.get(a + 2) - indices.get(a));
                    if (cur_dist < ans) {
                        ans = cur_dist;
                    }
                }
            }
        }
        
        return found ? ans : -1;
    }
}
```

- 时间复杂度： $O(n)$，遍历一次数组 $nums$，遍历哈希的所有下标
- 空间复杂度： $O(n)$，表示哈希表 $posMap$ 的大小
