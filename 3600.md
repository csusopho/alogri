[Problem: 3600. 升级后最大生成树稳定性](https://leetcode.cn/problems/maximize-spanning-tree-stability-with-upgrades/description/)

### 方法：二分+并查集+贪心

什么是生成树？所有 $n$ 个节点全在一个连通块中，且没有成环。

什么叫做最大化最小值？尽量让所有值都变大，这样的话最小值也会随之变大。

啰嗦一下题意：值 $must_i = 1$ 的边必须包含在生成树中，且 **不能** 升级；值 $must_i = 0$ 的边可以包含也可以不包含，且 **最多升级一次**，总升级次数不能超过 $k$。

一般来说，最大化一个“最小值”，这通常意味着 **二分搜索**。也就是说，尝试一个稳定性目标值 $X$，然后判断是否存在一种方案，满足所有约束且所有边强度均 $\ge X$。

为什么想到二分？**单调性**：如果稳定性 $X$ 可以实现，那么比 $X$ 小的稳定性也一定能实现。寻找 “最大稳定性” 的问题，转化为了多次 “是否存在可行解” 的判定问题。

---

首先，我们应该检查是否成环。使用 **并查集**，将所有 $must_i = 1$ 的边放在一个连通块中。如果成环，无法构成生成树，直接返回 $-1$。

其次，我们应该检查连通性。使用所有边，包含升级后的 $must_i = 0$ 边，检查是否能连通所有节点。若不能，返回 $-1$。

二分边界：

- 最小可能的稳定性是所有边中最小的 $s_i$。
- 最大可能的稳定性是所有边中最大的 $2s_i$。

更精确地，可能的答案 $X$ 只会出现在 $\{s_i\} \cup \{2s_i\}$ 这些集合中。因此，将这些值收集后 **排序**，作为二分的搜索空间。

给定目标稳定性 $X$，判断它是否可行：

1. 检查必须边：对于所有 $must_i = 1$ 的边，如果其原始强度 $s_i < X$，说明目标 $X$ 不可行（因为这些边必须选且不能升级）。
2. 构建并查集：先将所有 $must_i = 1$ 的边加入并查集。
3. 使用可选边：
    - **优先使用** 不需要升级的边：遍历 $must_i = 0$ 且 $s_i \ge X$ 的边。如果边连接了两个不同的分量，合并它们。
    - **再使用** 需要升级的边：遍历 $must_i = 0$ 且 $s_i < X$ 但 $2s_i \ge X$ 的边。如果能连接不同分量，则使用它，并记录升级次数 +1。
4. 最终判断：如果最终并查集的连通分量数量为 $1$，即所有节点已连通。且所使用的升级次数 $\le k$，则目标 $X$ 是可行的。

注意，上面操作中最核心的是 **贪心** 思路：先用免费的，再用收费的。

- **第一优先级：必须边** —— 这些边没得选，必须先放进去。如果它们自己就成环了，直接判定方案非法。
- **第二优先级：免费的可选边** —— 这些边不需要消耗升级次数就能满足强度。尽可能多地使用它们来合并连通分量。
- **第三优先级：收费的可选边** —— 只有在前两个优先级处理完后，节点还没连通，此时才动用 $k$ 次升级机会，选择最少数量的这种边来补位。

为什么不需要检查 “多连”？换句话说，为什么一直 $union$ 没事？

因为只要能 “连通”，就一定能 “修剪” 成树。在图论中，如果一个图是连通的，连通分量为 $1$，那么它一定包含至少一个生成树。同时，在之前的预处理中，我们已经排除了成环的可能。

代码中的 `if root_u != root_v` 就是后续的 “防环” 检查。它确保了只在必要时添加边，最终如果能连通，手里拿到的边集正好就是一个生成树。

使用最容易理解的闭区间二分，代码如下，已附加注释：

```Python
# python
class Solution:
    def maxStability(self, n: int, edges: List[List[int]], k: int) -> int:
        must_edges = []     # 必须连接的边
        opt_edges = []      # 可选
        candidates = set()  # 所有可能的强度
        
        # 分类边
        for u, v, s, m in edges:
            candidates.add(s)
            if m == 1:
                must_edges.append((u, v, s))
            else:
                opt_edges.append((u, v, s))
                candidates.add(2 * s)  # 可升级边的翻倍
        
        # 排序
        sorted_cand = sorted(list(candidates))
        
        # 检查强度x是否可行
        def check(X):
            # 重新初始化并查集
            parent = list(range(n))
            components = n
            
            def find(i):
                if parent[i] == i: return i
                parent[i] = find(parent[i])  # 路径压缩
                return parent[i]
            
            # 先处理必须边
            for u, v, s in must_edges:
                if s < X:  # 强度不够
                    return False
                root_u, root_v = find(u), find(v)
                if root_u != root_v:  # 放到生成树上
                    parent[root_u] = root_v
                    components -= 1
                else:  # 形成环
                    return False
            
            # 优先不用升级的可选边
            for u, v, s in opt_edges:
                if s >= X:
                    root_u, root_v = find(u), find(v)
                    if root_u != root_v:  # 放到生成树上
                        parent[root_u] = root_v
                        components -= 1

            # 再尝试升级边
            upgrades = 0
            for u, v, s in opt_edges:
                if s < X <= 2 * s:  # 原始不达标，升级后才达标
                    root_u, root_v = find(u), find(v)
                    if root_u != root_v:  # 合并
                        parent[root_u] = root_v
                        components -= 1
                        upgrades += 1
            
            # 连通且升级次数不超过 k
            return components == 1 and upgrades <= k

        # 二分
        ans = -1
        left, right = 0, len(sorted_cand) - 1

        while left <= right:
            mid = (left + right) // 2
            if check(sorted_cand[mid]):  # 可行，尝试更大值
                ans = sorted_cand[mid]
                left = mid + 1
            else:  # 不行，下调目标
                right = mid - 1
          
        return ans
```

```Java
// java
class Solution {
    public int maxStability(int n, int[][] edges, int k) {
        List<int[]> must_edges = new ArrayList<>();     // 必须连接的边
        List<int[]> opt_edges = new ArrayList<>();      // 可选
        Set<Long> candidates = new HashSet<>();         // 所有可能的强度
        
        // 分类边
        for (int[] edge : edges) {
            int u = edge[0], v = edge[1], s = edge[2], m = edge[3];
            candidates.add((long) s);
            if (m == 1) {
                must_edges.add(new int[]{u, v, s});
            } else {
                opt_edges.add(new int[]{u, v, s});
                candidates.add(2L * s);  // 可升级边的翻倍
            }
        }
        
        // 排序
        Long[] sorted_cand = candidates.toArray(new Long[0]);
        Arrays.sort(sorted_cand);
        
        // 二分
        int ans = -1;
        int left = 0, right = sorted_cand.length - 1;

        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (check(sorted_cand[mid], n, must_edges, opt_edges, k)) {  // 可行，尝试更大值
                ans = sorted_cand[mid].intValue();
                left = mid + 1;
            } else {  // 不行，下调目标
                right = mid - 1;
            }
        }
          
        return ans;
    }

    // 检查强度x是否可行
    private boolean check(long X, int n, List<int[]> must_edges, List<int[]> opt_edges, int k) {
        // 重新初始化并查集
        int[] parent = new int[n];
        for (int i = 0; i < n; i++) parent[i] = i;
        int components = n;
        
        // 先处理必须边
        for (int[] edge : must_edges) {
            int u = edge[0], v = edge[1], s = edge[2];
            if (s < X) {  // 强度不够
                return false;
            }
            int root_u = find(u, parent), root_v = find(v, parent);
            if (root_u != root_v) {  // 放到生成树上
                parent[root_u] = root_v;
                components--;
            } else {  // 形成环
                return false;
            }
        }
        
        // 优先不用升级的可选边
        for (int[] edge : opt_edges) {
            int u = edge[0], v = edge[1], s = edge[2];
            if (s >= X) {
                int root_u = find(u, parent), root_v = find(v, parent);
                if (root_u != root_v) {  // 放到生成树上
                    parent[root_u] = root_v;
                    components--;
                }
            }
        }

        // 再尝试升级边
        int upgrades = 0;
        for (int[] edge : opt_edges) {
            int u = edge[0], v = edge[1], s = edge[2];
            if (s < X && 2L * s >= X) {  // 原始不达标，升级后才达标
                int root_u = find(u, parent), root_v = find(v, parent);
                if (root_u != root_v) {  // 合并
                    parent[root_u] = root_v;
                    components--;
                    upgrades++;
                }
            }
        }
        
        // 连通且升级次数不超过 k
        return components == 1 && upgrades <= k;
    }

    private int find(int i, int[] parent) {
        if (parent[i] == i) return i;
        return parent[i] = find(parent[i], parent);  // 路径压缩
    }
}
```

排序候选值时间为 $O(E \log E)$，二分搜索执行 $\log(2E)$ 次 $check$ 函数。其中，每次 $check$ 需要遍历所有边并进行并查集操作，时间复杂度为 $O(E \cdot \alpha(n))$。

- 时间复杂度： $O(E \log E \cdot \alpha(n))$，其中 $E$ 是数组 $edges$ 的长度
- 空间复杂度： $O(n+E)$，用于存储并查集和边列表

### 优化

在 $check$ 函数中，按照 “必须边 > 免费边 > 升级边” 的顺序进行合并，这个不能改变。

我们可以增加预处理，将 “成环” 以及 “全连通” 这两步操作提到 $check$ 函数外进行预处理。

同时，由于本题的边强度 $s_i$ 非常小，只是 $10^5$ 范围，所以没必要使用 $candidates$ 排序，直接对数值进行二分，速度反而更快。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxStability(self, n: int, edges: List[List[int]], k: int) -> int:
        # 查找强度边界
        min_s, max_s = inf, 0
        for _, _, s, _ in edges:
            if s < min_s: min_s = s
            if s > max_s: max_s = s
        
        def find(i, parent):
            root = i
            while parent[root] != root:
                root = parent[root]
            while parent[i] != root:  # 路径压缩
                parent[i], i = root, parent[i]
            return root
        
        # 预检查，连接必须边，不能成环
        must_pa = list(range(n))
        for u, v, s, must in edges:
            if must:
                root_u, root_v = find(u, must_pa), find(v, must_pa)
                if root_u == root_v:
                    return -1  # 成环
                must_pa[root_u] = root_v
        
        # 预检查，所有边全连上，必须让图连通
        all_pa = list(range(n))
        need_cnt = n
        for u, v, s, must in edges:
            root_u, root_v = find(u, all_pa), find(v, all_pa)
            if root_u != root_v:
                all_pa[root_u] = root_v
                need_cnt -= 1
        if need_cnt > 1:
            return -1  # 图本身不连通

        # 判断目标稳定性 target 是否可行
        def check(target):
            parent = list(range(n))
            components = n  # 当前连通块数量
            
            # 先处理所有必须边
            for u, v, s, must in edges:
                if must:
                    if s < target:
                        return False  # 强度达不到
                    root_u, root_v = find(u, parent), find(v, parent)
                    parent[root_u] = root_v
                    components -= 1
            
            # 再贪心使用免费边
            for u, v, s, must in edges:
                if not must and s >= target:
                    root_u, root_v = find(u, parent), find(v, parent)
                    # 必须判断是否已经在同一个连通块
                    if root_u != root_v:
                        parent[root_u] = root_v
                        components -= 1
            
            if components == 1:
                return True  # 已连通
            
            # 再尝试升级边
            upgrades = k
            for u, v, s, must in edges:
                if components == 1 or upgrades == 0:
                    break
                if not must and s < target <= 2 * s:  # 原始不达标，升级后才达标
                    root_u, root_v = find(u, parent), find(v, parent)
                    if root_u != root_v:
                        parent[root_u] = root_v
                        components -= 1
                        upgrades -= 1
            
            # 连通且升级次数不超过 k
            return components == 1

        # 二分
        left, right = min_s, max_s * 2 + 1
        ans = 0

        while left < right:
            mid = (left + right) // 2
            if check(mid):  # 可行，尝试更大值
                ans = mid
                left = mid + 1
            else:  # 不行，下调目标
                right = mid
        
        return ans
```

```Java
// java
class Solution {
    public int maxStability(int n, int[][] edges, int k) {
        // 查找强度边界
        long min_s = Long.MAX_VALUE, max_s = 0;
        for (int[] edge : edges) {
            int s = edge[2];
            if (s < min_s) min_s = s;
            if (s > max_s) max_s = s;
        }
        
        // 预检查，连接必须边，不能成环
        int[] must_pa = new int[n];
        for (int i = 0; i < n; i++) must_pa[i] = i;
        for (int[] edge : edges) {
            if (edge[3] == 1) {
                int root_u = find(edge[0], must_pa), root_v = find(edge[1], must_pa);
                if (root_u == root_v) {
                    return -1;  // 成环
                }
                must_pa[root_u] = root_v;
            }
        }
        
        // 预检查，所有边全连上，必须让图连通
        int[] all_pa = new int[n];
        for (int i = 0; i < n; i++) all_pa[i] = i;
        int need_cnt = n;
        for (int[] edge : edges) {
            int root_u = find(edge[0], all_pa), root_v = find(edge[1], all_pa);
            if (root_u != root_v) {
                all_pa[root_u] = root_v;
                need_cnt--;
            }
        }
        if (need_cnt > 1) {
            return -1;  // 图本身不连通
        }

        // 二分
        long left = min_s, right = max_s * 2 + 1;
        int ans = 0;

        while (left < right) {
            long mid = left + (right - left) / 2;
            if (check(mid, n, edges, k)) {  // 可行，尝试更大值
                ans = (int) mid;
                left = mid + 1;
            } else {  // 不行，下调目标
                right = mid;
            }
        }
        
        return ans;
    }

    // 判断目标稳定性 target 是否可行
    private boolean check(long target, int n, int[][] edges, int k) {
        int[] parent = new int[n];
        for (int i = 0; i < n; i++) parent[i] = i;
        int components = n;  // 当前连通块数量
        
        // 先处理所有必须边
        for (int[] edge : edges) {
            if (edge[3] == 1) {
                if (edge[2] < target) {
                    return false;  // 强度达不到
                }
                int root_u = find(edge[0], parent), root_v = find(edge[1], parent);
                if (root_u != root_v) {
                    parent[root_u] = root_v;
                    components--;
                }
            }
        }
        
        // 再贪心使用免费边
        for (int[] edge : edges) {
            if (edge[3] == 0 && edge[2] >= target) {
                int root_u = find(edge[0], parent), root_v = find(edge[1], parent);
                // 必须判断是否已经在同一个连通块
                if (root_u != root_v) {
                    parent[root_u] = root_v;
                    components--;
                }
            }
        }
        
        if (components == 1) {
            return true;  // 已连通
        }
        
        // 再尝试升级边
        int upgrades = k;
        for (int[] edge : edges) {
            if (components == 1 || upgrades == 0) {
                break;
            }
            if (edge[3] == 0 && edge[2] < target && 2L * edge[2] >= target) {  // 原始不达标，升级后才达标
                int root_u = find(edge[0], parent), root_v = find(edge[1], parent);
                if (root_u != root_v) {
                    parent[root_u] = root_v;
                    components--;
                    upgrades--;
                }
            }
        }
        
        // 连通且升级次数不超过 k
        return components == 1;
    }

    private int find(int i, int[] parent) {
        int root = i;
        while (parent[root] != root) {
            root = parent[root];
        }
        while (parent[i] != root) {  // 路径压缩
            int next = parent[i];
            parent[i] = root;
            i = next;
        }
        return root;
    }
}
```

- 时间复杂度： $O(18\cdot E \cdot \alpha(n))$，二分次数减小
- 空间复杂度： $O(n)$，仅需存储并查集的父节点数组
