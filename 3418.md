[Problem: 3418. 机器人可以获得的最大金币数](https://leetcode.cn/problems/maximum-amount-of-money-robot-can-earn/description/)

### 方法：DP & 优化

说白了，还是从左上到右下的网格图问题。本题加了一个 “感化”，相当于该格子的金币为 $0$。不进行感化的话，该格子的金币为负数。

机器人只能向右或向下移动，这满足了动态规划的 **“无后效性”** 特征。唯一的特殊点在于 “最多感化 $2$ 个强盗”，这需要在 $DP$ 状态中额外增加一个维度来记录。

观察状态，一共需要记录机器人当前的位置，以及它已经使用了多少次 “感化” 能力。

**定义** $dp[i][j][k]$ 为：

- $i, j$：机器人当前所在的行和列。
- $k$：机器人 **已经使用** 感化能力的次数， $k \in \{0, 1, 2\}$。
- $dp[i][j][k]$ 的值：到达位置 $(i, j)$ 且感化了 $k$ 个强盗时，所能获得的 **最大金币数**。

---

机器人可以从左边 $(i, j-1)$ 或上边 $(i-1, j)$ 移动到当前位置 $(i, j)$。在每一个位置，机器人都有两种选择：

#### 情况 A：不感化当前的单元格 $(i, j)$

无论 $coins[i][j]$ 是正是负，直接加上它的值。

$$
dp[i][j][k] = \max(dp[i-1][j][k], dp[i][j-1][k]) + coins[i][j]
$$

#### 情况 B：感化当前的单元格 $(i, j)$

这种情况只有在 $k > 0$（即还有剩余感化次数）且 $coins[i][j] < 0$ 时才有意义如果感化了，当前格子的贡献计为 $0$。

$$
dp[i][j][k] = \max(dp[i][j][k], \max(dp[i-1][j][k-1], dp[i][j-1][k-1]) + 0)
$$

总的来看，对于每一个 $k \in \{0, 1, 2\}$：

1. **基础继承**：从上方或左方转移过来，累加 $coins[i][j]$。
2. **能力触发**：如果 $k > 0$ 且 $coins[i][j] < 0$，尝试将当前格子的损失抹零，并从 $k-1$ 状态转移。

由于金币数可以是负数，我们将 $DP$ 数组初始化为负无穷大 $-\infty$。

起点是 $(0,0)$，初始化 $dp[0][0][0] = coins[0][0]$。

- 如果 $coins[0][0] < 0$：机器人可以立刻感化起点，则 $dp[0][0][1/2] = 0$
- 如果 $coins[0][0] \ge 0$：机器人不需要感化起点，则 $dp[0][0][1/2] = coins[0][0]$

答案是到达右下角 $(m-1, n-1)$ 时，使用 $0、1、2$ 次能力的最高值，也即

$$
\max(dp[m-1][n-1][0], dp[m-1][n-1][1], dp[m-1][n-1][2])
$$

代码如下，已附加注释：

```Python
# python
class Solution:
    def maximumAmount(self, coins: List[List[int]]) -> int:
        m = len(coins)
        n = len(coins[0])
        # dp[i][j][k] 表示到达位置 (i,j) 且感化了 k 个强盗时，所能获得的最大金币数
        dp = [[[float('-inf')] * 3 for _ in range(n)] for _ in range(m)]
        
        # 起点
        cur_val = coins[0][0]
        # 不使用能力
        dp[0][0][0] = cur_val
        # 使用能力
        if cur_val < 0:
            dp[0][0][1] = 0
            dp[0][0][2] = 0
        else:
            dp[0][0][1] = cur_val
            dp[0][0][2] = cur_val
        
        # 开始填表
        for i in range(m):
            for j in range(n):
                if i == 0 and j == 0:
                    continue
                    
                for k in range(3):
                    # 从上方或左方过来的最大前置收益
                    res_from_top = dp[i-1][j][k] if i > 0 else float('-inf')
                    res_from_left = dp[i][j-1][k] if j > 0 else float('-inf')
                    prev_max = max(res_from_top, res_from_left)
                    
                    if prev_max == float('-inf'):
                        continue
                    
                    # 正常通过当前格子
                    dp[i][j][k] = max(dp[i][j][k], prev_max + coins[i][j])
                    
                    # 感化当前格子
                    if k > 0 and coins[i][j] < 0:
                        prev_res_from_top = dp[i-1][j][k-1] if i > 0 else float('-inf')
                        prev_res_from_left = dp[i][j-1][k-1] if j > 0 else float('-inf')
                        prev_k_max = max(prev_res_from_top, prev_res_from_left)
                        
                        if prev_k_max != float('-inf'):
                            # 感化后，当前格子的收益计为 0
                            dp[i][j][k] = max(dp[i][j][k], prev_k_max + 0)
        
        return max(dp[m-1][n-1])
```

```Java
// java
class Solution {
    public int maximumAmount(int[][] coins) {
        int m = coins.length;
        int n = coins[0].length;
        int INF = Integer.MIN_VALUE / 2;
        // dp[i][j][k] 表示到达位置 (i,j) 且感化了 k 个强盗时，所能获得的最大金币数
        int[][][] dp = new int[m][n][3];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                Arrays.fill(dp[i][j], INF);
            }
        }
        
        // 起点
        int cur_val = coins[0][0];
        // 不使用能力
        dp[0][0][0] = cur_val;
        // 使用能力
        if (cur_val < 0) {
            dp[0][0][1] = 0;
            dp[0][0][2] = 0;
        } else {
            dp[0][0][1] = cur_val;
            dp[0][0][2] = cur_val;
        }
        
        // 开始填表
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (i == 0 && j == 0) {
                    continue;
                }
                    
                for (int k = 0; k < 3; k++) {
                    // 从上方或左方过来的最大前置收益
                    int res_from_top = i > 0 ? dp[i-1][j][k] : INF;
                    int res_from_left = j > 0 ? dp[i][j-1][k] : INF;
                    int prev_max = Math.max(res_from_top, res_from_left);
                    
                    if (prev_max == INF) {
                        continue;
                    }
                    
                    // 正常通过当前格子
                    dp[i][j][k] = Math.max(dp[i][j][k], prev_max + coins[i][j]);
                    
                    // 感化当前格子
                    if (k > 0 && coins[i][j] < 0) {
                        int prev_res_from_top = i > 0 ? dp[i-1][j][k-1] : INF;
                        int prev_res_from_left = j > 0 ? dp[i][j-1][k-1] : INF;
                        int prev_k_max = Math.max(prev_res_from_top, prev_res_from_left);
                        
                        if (prev_k_max != INF) {
                            // 感化后，当前格子的收益计为 0
                            dp[i][j][k] = Math.max(dp[i][j][k], prev_k_max + 0);
                        }
                    }
                }
            }
        }
        
        return Math.max(dp[m-1][n-1][0], Math.max(dp[m-1][n-1][1], dp[m-1][n-1][2]));
    }
}
```

- 时间复杂度： $O(m\times n)$，其中 $m,n$ 是网格 $coins$ 的行数、列数
- 空间复杂度： $O(m\times n)$，表示数组 $dp$ 的大小

---

### 空间优化

观察上面的 $DP$ 观察，可以发现，**当前状态只依赖于 “上方” 和 “左方” 的值**。

对于 $dp[i][j][k]$ 来说，计算第 $i$ 行时，只需要第 $i-1$ 行的数据（上方）和第 $i$ 行中刚刚计算出的 $j-1$ 列的数据（左方）。因此，将 $m \times n \times 3$ 的数组压缩为 $n \times 3$。

**定义** $dp[j][k]$ 表示当前行第 $j$ 列，使用 $k$ 次能力的最大金币数。

**更新逻辑**：

- 在进入第 $j$ 列的计算之前， $dp[j][k]$ 存储的是 **上一行**（上方格子）的值。
- 而 $dp[j-1][k]$ 存储的是 **当前行**（左方格子）刚刚更新完的值。
- 使用一个临时变量来存储当前格子的新状态，计算完成后再覆盖 $dp[j]$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maximumAmount(self, coins: List[List[int]]) -> int:
        m, n = len(coins), len(coins[0])
        INF = float('-inf')
        # dp[j][k] 表示第 j 列在感化 k 次后的最大值
        dp = [[INF] * 3 for _ in range(n)]
        
        # 起点
        v0 = coins[0][0]
        # 不感化
        dp[0][0] = v0
        # 感化
        if v0 < 0:
            dp[0][1] = 0
            dp[0][2] = 0
        else:
            dp[0][1] = v0
            dp[0][2] = v0
        
        for i in range(m):
            for j in range(n):
                if i == 0 and j == 0:
                    continue
                
                # 当前格子的新状态
                cur_k_states = [INF] * 3
                
                for k in range(3):
                    # 从上方或左方过来的最大前置收益
                    res_from_top = dp[j][k] if i > 0 else INF
                    res_from_left = dp[j-1][k] if j > 0 else INF
                    prev_max = max(res_from_top, res_from_left)
                    
                    # 不感化
                    if prev_max != INF:
                        cur_k_states[k] = max(cur_k_states[k], prev_max + coins[i][j])
                    
                    # 感化
                    if k > 0 and coins[i][j] < 0:
                        res_from_top_k = dp[j][k-1] if i > 0 else INF
                        res_from_left_k = dp[j-1][k-1] if j > 0 else INF
                        prev_k_max = max(res_from_top_k, res_from_left_k)
                        
                        if prev_k_max != INF:
                            # 感化后当前格收益视为 0
                            cur_k_states[k] = max(cur_k_states[k], prev_k_max)
                
                # 更新 dp 数组的当前列
                dp[j] = cur_k_states
        
        return max(dp[n-1])
```

```Java
// java
class Solution {
    public int maximumAmount(int[][] coins) {
        int m = coins.length, n = coins[0].length;
        int INF = Integer.MIN_VALUE / 2;
        // dp[j][k] 表示第 j 列在感化 k 次后的最大值
        int[][] dp = new int[n][3];
        for (int j = 0; j < n; j++) {
            Arrays.fill(dp[j], INF);
        }
        
        // 起点
        int v0 = coins[0][0];
        // 不感化
        dp[0][0] = v0;
        // 感化
        if (v0 < 0) {
            dp[0][1] = 0;
            dp[0][2] = 0;
        } else {
            dp[0][1] = v0;
            dp[0][2] = v0;
        }
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (i == 0 && j == 0) {
                    continue;
                }
                
                // 当前格子的新状态
                int[] cur_k_states = new int[3];
                Arrays.fill(cur_k_states, INF);
                
                for (int k = 0; k < 3; k++) {
                    // 从上方或左方过来的最大前置收益
                    int res_from_top = i > 0 ? dp[j][k] : INF;
                    int res_from_left = j > 0 ? dp[j-1][k] : INF;
                    int prev_max = Math.max(res_from_top, res_from_left);
                    
                    // 不感化
                    if (prev_max != INF) {
                        cur_k_states[k] = Math.max(cur_k_states[k], prev_max + coins[i][j]);
                    }
                    
                    // 感化
                    if (k > 0 && coins[i][j] < 0) {
                        int res_from_top_k = i > 0 ? dp[j][k-1] : INF;
                        int res_from_left_k = j > 0 ? dp[j-1][k-1] : INF;
                        int prev_k_max = Math.max(res_from_top_k, res_from_left_k);
                        
                        if (prev_k_max != INF) {
                            // 感化后当前格收益视为 0
                            cur_k_states[k] = Math.max(cur_k_states[k], prev_k_max);
                        }
                    }
                }
                
                // 更新 dp 数组的当前列
                dp[j] = cur_k_states;
            }
        }
        
        return Math.max(dp[n-1][0], Math.max(dp[n-1][1], dp[n-1][2]));
    }
}
```

- 时间复杂度： $O(m\times n)$
- 空间复杂度： $O(n)$，空间优化
