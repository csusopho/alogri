[Problem: 1784. 检查二进制字符串字段](https://leetcode.cn/problems/check-if-binary-string-has-at-most-one-segment-of-ones/description/)

### 方法：模拟 & 优化

什么叫做连续的 $1$ 片段？从左到右遍历，如果遇到了字符 $1$，并且它 **是第一个字符** 或者它 **前面的字符是 0**，这就标志着一个 **新的 1 片段的开始**。

因此，我们只需要在遍历的过程中，统计这样的 “开头”出现了多少次。

- 如果统计出的片段数量 $count > 1$，说明存在多于一个的由 $1$ 组成的连续字段，直接返回 $False$。
- 如果遍历结束 $count\leq 1$，说明符合题意，返回 $True$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def checkOnesSegment(self, s: str) -> bool:
        cnt = 0
        
        for i in range(len(s)):
            # 当前字符是 1，并且是该片段的第一个字符
            if s[i] == '1' and (i == 0 or s[i-1] == '0'):
                cnt += 1
                # 片段数已经超过 1
                if cnt > 1:
                    return False
        
        return True
```

```Java
// java
class Solution {
    public boolean checkOnesSegment(String s) {
        int cnt = 0;
        
        for (int i = 0; i < s.length(); i++) {
            // 当前字符是 1，并且是该片段的第一个字符
            if (s.charAt(i) == '1' && (i == 0 || s.charAt(i - 1) == '0')) {
                cnt += 1;
                // 片段数已经超过 1
                if (cnt > 1) {
                    return false;
                }
            }
        }
        
        return true;
    }
}
```

- 时间复杂度： $O(n)$，其中 $n$ 是字符串 $s$ 的长度
- 空间复杂度： $O(1)$，仅使用常数个额外变量

### 寻找规律

顺着前面的推导再深入一步。

既然题目规定了 **“不含前导零”**，这就意味着：

- 字符串第一个字符必定是 $1$。
- 如果要出现 **第二个** $1$ 的片段，由于片段之间必须被 $0$ 隔开，那么在原本的 $0$ 后面，必然要出现一个新的 $1$。

换句话说，只要字符串里出现了 $0$ 紧接着 $1$ 的情况（即存在子串 $01$），就说明在中断之后又出现了新的 $1$，此时必然存在 **两个以上** 的 $1$ 片段！

**结论**：

- 如果存在子串 $01$ $\Rightarrow$ 说明有前置的 $0$ 切断了 $1$，并且后面又出现了新的 $1$ $\Rightarrow$ 至少有两个片段 $\Rightarrow$ $False$。
- 如果不存在子串 $01$ $\Rightarrow$ 所有的 $1$ 只能在遇到 $0$ 之前连在一起（或者全是 $1$） $\Rightarrow$ 最多只有一个片段 $\Rightarrow$ $True$。

那一行就能解决，代码如下：

```Python
class Solution:
    def checkOnesSegment(self, s: str) -> bool:
        return "01" not in s
```

```Java
// java
class Solution {
    public boolean checkOnesSegment(String s) {
        return !s.contains("01");
    }
}
```

- 时间复杂度： $O(n)$
- 空间复杂度： $O(1)$
