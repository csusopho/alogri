[Problem: 2169. 得到 0 的操作数](https://leetcode.cn/problems/count-operations-to-obtain-zero/description/)

### 方法：模拟

~~题干是人话？~~ 其实就一句话：让 **大** 数字减去 **小** 数字，直至某个数字变为 $0$。

模拟操作流程，代码如下：

```Python
# python
class Solution:
    def countOperations(self, num1: int, num2: int) -> int:
        # 辗转相减
        ans = 0
        while num1 > 0 and num2 > 0:
            if num1 >= num2:
                num1 -= num2
            else:
                num2 -= num1
            ans += 1
        return ans
```

```Java
// java
class Solution {
    public int countOperations(int num1, int num2) {
        // 辗转相减
        int ans = 0;
        while (num1 > 0 && num2 > 0) {
            if (num1 >= num2) {
                num1 -= num2;
            } else {
                num2 -= num1;
            }
            ans++;
        }
        return ans;
    }
}
```

- 时间复杂度： $O(max(num1,num2))$
- 空间复杂度： $O(1)$，仅使用常数个额外变量

### 辗转相除

实际上，最后剩下来的那个不为 $0$ 的 $num$ 就是两者的 **最大公约数**。

$GCD$ 有多种求解方式，最常见的两种就是「辗转相除」和「辗转相减」，上面的那种方式就是「辗转相减」，具体的原理/模版/推导，可以看这篇 [题解](https://leetcode.cn/problems/replace-non-coprime-numbers-in-array/solutions/3783419/shu-xue-san-chong-gcdmo-ban-zhan-mo-ni-j-ffav/)。

下面使用递推形式的 “辗转相除法”，注意 `%` 是用来取余，而 `/` 则是操作次数，本质是减法的优化。

直接套用公式，得到如下代码：

```Python
# python
class Solution:
    def countOperations(self, num1: int, num2: int) -> int:
        # 辗转相除
        ans = 0
        while num1 != 0 and num2 != 0:
            if num1 >= num2:
                ans += num1 // num2
                num1 %= num2
            else:
                ans += num2 // num1
                num2 %= num1
        return ans
```

```Java
// java
class Solution {
    public int countOperations(int num1, int num2) {
        // 辗转相除
        int ans = 0;
        while (num1 != 0 && num2 != 0) {
            if (num1 >= num2) {
                ans += num1 / num2;
                num1 %= num2;
            } else {
                ans += num2 / num1;
                num2 %= num1;
            }
        }
        return ans;
    }
}
```

- 时间复杂度： $O(log(min(num1,num2)))$
- 空间复杂度： $O(1)$
