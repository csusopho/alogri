[Problem: 3474. 字典序最小的生成字符串](https://leetcode.cn/problems/lexicographically-smallest-generated-string/description/)

### 方法：模拟+贪心

先解释一下示例 $1$，从前往后看：

- 由于 $str1[0]=T$，所以必须让 $word[0]=a$ 且 $word[1]=b$。
- 下一个 $str1[2]=T$，所以必须 $word[2]=a$ 且 $word[3]=b$。
- 此时，由于 $str1[3]=F$，所以必须让 $word[3,4]\not =ab$。上面已经规定 $word[3]=b$，已经让它不等于 $a$，则 $word[4]$ 任意填。
- 字典序需要最小，最小的字母当然是 $a$，所以答案是 $ababa$。

那么，本题的思路很明确：先处理 $T$，然后再贪心地填充 $a$，最后检查 $F$ 条件重新尝试。

#### 第一步：处理所有 “必须满足” 的条件

创建一个数组 $res$ 表示 $word$ 每一位，初始为 $None$。

遍历 $str1$，若 $str1[i]=T$，则将 $res[i...i+m-1]$ 赋值为 $str2$。

**冲突检测**：在填充过程中，如果某个位置已经被填入字符，且与当前要填入的 $str2$ 对应字符不同，说明无法生成满足条件的字符串，直接返回空字符串 $""$。

同时，记录哪些下标是被 $T$ 条件 **固定** 的，因为这些位置在后续优化字典序时不能被修改。使用数组 $fixed$ 进行记录。

#### 第二步：初始化贪心策略

为了让字典序最小，对于所有没有被 $T$ 固定的位置，应该尝试填入最小的字符 $a$。具体来说，遍历 $res$，如果 $res[j]$ 为 $None$，则令 $res[j]=a$。

#### 第三步：处理 “不能相等” 的条件

现在 $res$ 中填满了字符，需要检查是否满足所有 $str1[i] = F$ 的条件。如果在某个下标 $i$，串 $res[i...i+m-1]$ 恰好等于 $str2$（而 $str1[i]$ 是 $F$），那就必须修改这个窗口内的一个字符。

**贪心修改策略**：为了保持字典序最小，希望修改尽可能 **靠右**的字符，并且增加的幅度尽可能小。

为什么靠右？因为在字典序比较中，左侧字符的权重更高。修改右侧字符对字典序的影响更小，例如 $aab$ 比 $aba$ 小。

**做法**：在窗口 $[i, i+m-1]$ 中，从右向左找到第一个 **没有被固定** 的位置 $k$，将其字符增加，例如从 $a$ 变成 $b$。

一旦修改了字符 $res[k]$，为了保证字典序最小，应该将 $k$ 之后所有 **未固定** 的位置重置回 $a$。

如果某个 $F$ 窗口必须修改但该窗口内所有字符都是固定的，则说明无解，返回 `""`。

代码如下，已附加注释：

```Python
# python
class Solution:
    def generateString(self, str1: str, str2: str) -> str:
        n, m = len(str1), len(str2)
        L = n + m - 1
        res = [None] * L
        fixed = [False] * L  # 当前字符是否固定
        
        # 先处理固定字符
        for i in range(n):
            if str1[i] == 'T':
                for j in range(m):
                    target_char = str2[j]
                    if res[i + j] is not None and res[i + j] != target_char:
                        return ""  # 之前填充过，且字符冲突，无解
                    res[i + j] = target_char
                    fixed[i + j] = True
        
        # 未固定位置全填充 a
        for i in range(L):
            if res[i] is None:
                res[i] = 'a'
        
        # 检查 F 条件
        i = 0
        while i < n:
            if str1[i] == 'F':
                # 检查当前窗口是否等于 str2
                is_match = True
                for j in range(m):
                    if res[i + j] != str2[j]:
                        is_match = False
                        break
                
                # 全相等，必须修改窗口内的一个非固定字符
                if is_match:
                    # 从右往左找第一个非固定位置
                    found_k = -1
                    for k in range(i + m - 1, i - 1, -1):
                        if not fixed[k] and res[k] < 'z':
                            found_k = k
                            break
                    
                    if found_k == -1:
                        return ""  # 窗口无法修改
                    
                    # 递增该位字符
                    res[found_k] = chr(ord(res[found_k]) + 1)
                    
                    # 将该位之后的所有非固定位置重置为 a
                    for j in range(found_k + 1, L):
                        if not fixed[j]:
                            res[j] = 'a'
            # 匹配下一位
            i += 1
        
        return "".join(res)
```

```Java
// java
public class Solution {
    public String generateString(String str1, String str2) {
        int n = str1.length();
        int m = str2.length();
        int L = n + m - 1;
        char[] res = new char[L];
        boolean[] fixed = new boolean[L]; // 当前字符是否固定
        boolean[] filled = new boolean[L]; // 标记该位置是否已填充字符
        
        // 先处理固定字符
        for (int i = 0; i < n; i++) {
            if (str1.charAt(i) == 'T') {
                for (int j = 0; j < m; j++) {
                    char targetChar = str2.charAt(j);
                    int pos = i + j;
                    if (filled[pos] && res[pos] != targetChar) {
                        return ""; // 之前填充过，且字符冲突，无解
                    }
                    res[pos] = targetChar;
                    fixed[pos] = true;
                    filled[pos] = true;
                }
            }
        }

        // 未固定位置全填充 a
        for (int i = 0; i < L; i++) {
            if (!filled[i]) {
                res[i] = 'a';
            }
        }

        // 检查 F 条件
        int i = 0;
        while (i < n) {
            if (str1.charAt(i) == 'F') {
                // 检查当前窗口是否等于 str2
                boolean isMatch = true;
                for (int j = 0; j < m; j++) {
                    if (res[i + j] != str2.charAt(j)) {
                        isMatch = false;
                        break;
                    }
                }

                // 全相等，必须修改窗口内的一个非固定字符
                if (isMatch) {
                    // 从右往左找第一个非固定位置
                    int foundK = -1;
                    for (int k = i + m - 1; k >= i; k--) {
                        if (!fixed[k] && res[k] < 'z') {
                            foundK = k;
                            break;
                        }
                    }

                    if (foundK == -1) {
                        return ""; // 窗口无法修改
                    }

                    // 递增该位字符
                    res[foundK] = (char) (res[foundK] + 1);

                    // 将该位之后的所有非固定位置重置为 a
                    for (int j = foundK + 1; j < L; j++) {
                        if (!fixed[j]) {
                            res[j] = 'a';
                        }
                    }
                }
            }
            // 匹配下一位
            i++;
        }

        return new String(res);
    }
}
```

- 时间复杂度： $O(n\times m)$，其中 $n,m$ 是串 $str1,str2$ 的长度
- 空间复杂度： $O(n+m)$，存储结果字符数组 $res$ 和固定标记位 $fixed$
