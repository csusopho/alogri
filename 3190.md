[Problem: 3190. 使所有元素都可以被 3 整除的最少操作数](https://leetcode.cn/problems/find-minimum-operations-to-make-all-elements-divisible-by-three/description/)

### 方法：数学&一行流

数字 $1$ 想变为 $3$ 的倍数，只需要一次 $+1$ 操作；数字 $2$ 想变为 $3$ 的倍数，只需要一次 $-1$ 操作。所以对于任意数字，对应 **零/一次** 操作。

**做法**：判断数字 $num$ 对 $3$ 取模 $MOD$ 的结果是否大于 $0$。

代码如下：

```Python
# python
class Solution:
    def minimumOperations(self, nums: List[int]) -> int:
        res = 0
        for num in nums:
            if num % 3:
                res += 1
        return res
```

```Java
// java
class Solution {
    public int minimumOperations(int[] nums) {
        int ans = 0;
        for (int x : nums) {
            if (x % 3 > 0) {
                ans += 1;
            }
        }
        return ans;
    }
}
```

- 时间复杂度： $O(n)$，其中 $n$ 是数组 $nums$ 的长度
- 空间复杂度： $O(1)$，仅使用常数个额外变量

### 一行流

$\textbf{仅供整活使用，一般不推荐}$，了解语法即可。代码如下：

```Python
# python
class Solution:
    def minimumOperations(self, nums: List[int]) -> int:
        return sum(1 for i in nums if i % 3 > 0)
```

```Java
// java
class Solution {
    public int minimumOperations(int[] nums) {
        return Arrays.stream(nums).map(n -> n % 3 == 0 ? 0 : 1).sum();
    }
}
```

- 时间复杂度： $O(n)$
- 空间复杂度： $O(1)$
