[Problem: 2069. 模拟行走机器人 II](https://leetcode.cn/problems/walking-robot-simulation-ii/description/)

### 方法：模拟

首先，本题最好不要一步步模拟机器人的移动，限制 $num$ 为 $10^5$ 量级，非常容易超时。即使采用取模优化，也得绕行一圈，只有个别代码能通过。

机器人永远只在网格的 **最外圈边界** 移动。它不会进入网格内部。

网格的周长，即机器人绕一圈回到原点所需的步数，是固定的。

宽度为 $width$，高度为 $height$。则底边步数为 $width - 1$，右边步数为 $height - 1$，顶边步数为 $width - 1$，左边步数为 $height - 1$。因此总周长

$$
L = 2 \times (width - 1) + 2 \times (height - 1)
$$

**取模优化**：由于机器人绕圈运动，可以对总移动步数进行 $num \% L$ 操作。

举个例子，假设总周长是 $6$，一共移动 $26$ 步，相当于绕了 $4$ 圈还多 $2$ 步。

---

这里有一个微妙的陷阱：**当机器人在拐角处停止时，它的方向是什么？**

根据规则：“下一步超出了边界，机器人会逆时针转 $90$ 度”。这意味着：

- 如果机器人在边上（非拐角），方向就是边的方向
- 如果机器人在拐角，且 **刚好走完** 指令步数，它的方向应该是它 **进入** 这个格子的方向

**本题最大陷阱（原点 0,0）：**

- 刚初始化时，位置 $(0,0)$，方向为 $East$
- 移动了 $L$ 的整数倍步数后回到 $(0,0)$，此时它实际上是从上方 “掉” 下来的，方向应为 $\textbf{South！}$ 千万小心，样例 $140/142$ 直接针对这一坑点。

代码如下，已附加注释：

```Python
# python
class Robot:
    def __init__(self, width: int, height: int):
        # 边界属性
        self.width = width
        self.height = height
        
        # 边界最大坐标值
        self.max_x = width - 1
        self.max_y = height - 1
        
        # 总周长
        self.perimeter = 2 * self.max_x + 2 * self.max_y
        
        # 相对于原点顺着边界走的 “路程”
        self.position_offset = 0
        
        # 标记是否执行过移动
        self.has_moved = False

    def step(self, num: int) -> None:
        self.has_moved = True
        # 对周长取模
        self.position_offset = (self.position_offset + num) % self.perimeter

    def getPos(self) -> list[int]:
        d = self.position_offset
        mx, my = self.max_x, self.max_y
        
        # 在底边
        if 0 <= d <= mx:
            return [d, 0]
        # 在右边
        if mx < d <= mx + my:
            return [mx, d - mx]
        # 在顶边
        if mx + my < d <= 2 * mx + my:
            return [mx - (d - (mx + my)), my]
        # 在左边
        return [0, my - (d - (2 * mx + my))]

    def getDir(self) -> str:
        d = self.position_offset
        mx, my = self.max_x, self.max_y
        
        # 没移动过
        if not self.has_moved:
            return "East"
        
        # 回到了原点，头朝下！
        if d == 0:
            return "South"
        
        # 判断
        if 1 <= d <= mx:
            return "East"
        elif mx < d <= mx + my:
            return "North"
        elif mx + my < d <= 2 * mx + my:
            return "West"
        else:
            return "South"
```

```Java
// java
class Robot {
    private int width;
    private int height;
    private int maxX;
    private int maxY;
    private int perimeter;
    private int positionOffset;
    private boolean hasMoved;

    public Robot(int width, int height) {
        // 边界属性
        this.width = width;
        this.height = height;
        
        // 边界最大坐标值
        this.maxX = width - 1;
        this.maxY = height - 1;
        
        // 总周长
        this.perimeter = 2 * this.maxX + 2 * this.maxY;
        
        // 相对于原点顺着边界走的 “路程”
        this.positionOffset = 0;
        
        // 标记是否执行过移动
        this.hasMoved = false;
    }

    public void step(int num) {
        this.hasMoved = true;
        // 对周长取模
        this.positionOffset = (this.positionOffset + num) % this.perimeter;
    }

    public int[] getPos() {
        int d = this.positionOffset;
        int mx = this.maxX;
        int my = this.maxY;
        
        // 在底边
        if (0 <= d && d <= mx) {
            return new int[]{d, 0};
        }
        // 在右边
        if (mx < d && d <= mx + my) {
            return new int[]{mx, d - mx};
        }
        // 在顶边
        if (mx + my < d && d <= 2 * mx + my) {
            return new int[]{mx - (d - (mx + my)), my};
        }
        // 在左边
        return new int[]{0, my - (d - (2 * mx + my))};
    }

    public String getDir() {
        int d = this.positionOffset;
        int mx = this.maxX;
        int my = this.maxY;
        
        // 没移动过
        if (!this.hasMoved) {
            return "East";
        }
        
        // 回到了原点，头朝下！
        if (d == 0) {
            return "South";
        }
        
        // 判断
        if (1 <= d && d <= mx) {
            return "East";
        } else if (mx < d && d <= mx + my) {
            return "North";
        } else if (mx + my < d && d <= 2 * mx + my) {
            return "West";
        } else {
            return "South";
        }
    }
}
```

- 时间复杂度：四个方法均为 $O(1)$，仅包含数学/逻辑运算
- 空间复杂度： $O(1)$，仅使用常数个额外变量
