[Problem: 2078. 两栋颜色不同且距离最远的房子](https://leetcode.cn/problems/two-furthest-houses-with-different-colors/description/)

### 方法：贪心

颜色不同的任意两栋房子，只计算一次距离，两种颜色不必相邻！

那么，只有以下两种情况，并取其中的最大值：

1. **固定第一栋房子**：从最后一栋房子开始向左遍历，找到第一栋颜色与 $colors[0]$ 不同的房子，记录距离。
2. **固定最后一栋房子**：从第一栋房子开始向右遍历，找到第一栋颜色与 $colors[n-1]$ 不同的房子，记录距离。

为什么这个 **贪心策略** 是正确的？

假设最优解的两栋房子分别是 $i$ 和 $j$（ $i < j$），且它们都不在边界上。

- 如果 $colors[i] \neq colors[n-1]$，那么距离 $(n-1) - i$ 显然比 $j - i$ 更大。
- 如果 $colors[j] \neq colors[0]$，那么距离 $j - 0$ 显然比 $j - i$ 更大。
- 如果 $colors[i] = colors[n-1]$ 且 $colors[j] = colors[0]$，由于题目保证一定存在两栋不同颜色的房子，通过简单的逻辑推导可以得出，最大距离的一端必然可以延伸到 $0$ 或 $n-1$。

因此，最大距离至少有一端是 **数组的边界**。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxDistance(self, colors: List[int]) -> int:
        n = len(colors)
        
        # 固定左端点，从右往左找
        res1 = 0
        for j in range(n - 1, 0, -1):
            if colors[j] != colors[0]:
                res1 = j
                break
        
        # 固定右端点，从左往右找
        res2 = 0
        for i in range(0, n - 1):
            if colors[i] != colors[n - 1]:
                res2 = (n - 1) - i
                break
        
        # 返回两者的最大值
        return max(res1, res2)
```

```Java
// java
class Solution {
    public int maxDistance(int[] colors) {
        int n = colors.length;
        
        // 固定左端点，从右往左找
        int res1 = 0;
        for (int j = n - 1; j > 0; j--) {
            if (colors[j] != colors[0]) {
                res1 = j;
                break;
            }
        }
        
        // 固定右端点，从左往右找
        int res2 = 0;
        for (int i = 0; i < n - 1; i++) {
            if (colors[i] != colors[n - 1]) {
                res2 = (n - 1) - i;
                break;
            }
        }
        
        // 返回两者的最大值
        return Math.max(res1, res2);
    }
}
```

- 时间复杂度： $O(n)$，其中 $n$ 为数组 $colors$ 的长度
- 空间复杂度： $O(1)$，仅使用常数个额外变量
