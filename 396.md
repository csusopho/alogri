[Problem: 396. 旋转函数](https://leetcode.cn/problems/rotate-function/description/)

### 方法：数学

假设数组 $nums$ 的长度为 $n$，元素总和为 $Sum$。

根据题意 $F(0)$ 的定义：

$$
F(0) = 0 \cdot nums[0] + 1 \cdot nums[1] + 2 \cdot nums[2] + \dots + (n-1) \cdot nums[n-1]
$$

顺时针旋转 $1$ 位，原数组最后一个元素移动到开头，数组变为：$[nums[n-1], nums[0], nums[1], \dots, nums[n-2]]$，此时 $F[1]$ 的定义：

$$
F(1) = 0 \cdot nums[n-1] + 1 \cdot nums[0] + 2 \cdot nums[1] + \dots + (n-1) \cdot nums[n-2]
$$

将 $F(1)$ 减去 $F(0)$，接着合并同类项：

$$
F(1) - F(0) = (1-0)nums[0] + (2-1)nums[1] + \dots + ((n-1)-(n-2))nums[n-2] - (n-1)nums[n-1]
$$

也即

$$
F(1) - F(0) = nums[0] + nums[1] + \dots + nums[n-2] - (n-1)nums[n-1]
$$

为了凑出总和 $Sum = \sum_{i=0}^{n-1}nums[i]$，我们在末尾加上并减去一个 $nums[n-1]$：

$$
F(1) - F(0) = (nums[0] + nums[1] + \dots + nums[n-2] + nums[n-1]) - nums[n-1] - (n-1)nums[n-1]
$$

因此，得出 **递推公式**：

$$
F(1) = F(0) + Sum - n \cdot nums[n-1]
$$

同理，对于 $F(k)$，它是 $F(k-1)$ 旋转一步得到的，新移到首位的元素是原数组中的 $nums[n-k]$。**推广到通用递推公式**：

$$
F(k) = F(k-1) + Sum - n \cdot nums[n-k]
$$

**总结**：首先计算数组所有元素的总和 $Sum$；然后，计算初始的旋转函数值 $F(0)$，记为 $curF$；最后，从 $k=1$ 遍历到 $n-1$，利用递推公式，更新 $curF$ 以及 $ans$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxRotateFunction(self, nums: List[int]) -> int:
        n = len(nums)
        total_sum = sum(nums)
        
        # F(0) = 0*nums[0] + 1*nums[1] + ... + (n-1)*nums[n-1]
        cur_f = 0
        for i in range(n):
            cur_f += i * nums[i]
        
        ans = cur_f
        
        # 迭代计算
        # 递推公式: F(k) = F(k-1) + Sum - n * nums[n-k]
        for k in range(1, n):
            cur_f = cur_f + total_sum - n * nums[n - k]
            if cur_f > ans:
                ans = cur_f
        
        return ans
```

```Java
// java
class Solution {
    public int maxRotateFunction(int[] nums) {
        int n = nums.length;
        long totalSum = 0;
        for (int num : nums) {
            totalSum += num;
        }
        
        // F(0) = 0*nums[0] + 1*nums[1] + ... + (n-1)*nums[n-1]
        long curF = 0;
        for (int i = 0; i < n; i++) {
            curF += (long) i * nums[i];
        }
        
        long ans = curF;
        
        // 迭代计算
        // 递推公式: F(k) = F(k-1) + Sum - n * nums[n-k]
        for (int k = 1; k < n; k++) {
            curF = curF + totalSum - (long) n * nums[n - k];
            if (curF > ans) {
                ans = curF;
            }
        }
        
        return (int) ans;
    }
}
```

- 时间复杂度： $O(n)$，其中 $n$ 是数组 $nums$ 的长度，三次遍历
- 空间复杂度： $O(1)$，仅使用常数个额外变量
