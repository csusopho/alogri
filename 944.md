[Problem: 944. 删列造序](https://leetcode.cn/problems/delete-columns-to-make-sorted/description/)

### 方法：模拟&排序

非严格递增：递增，且允许相邻元素相等，比如 $a,a,b,c$。如果前一个元素 > 后一个元素，可以看作一个断点，这个序列就不是 “···” 的列。

删除不是 “···” 的列，等价于保留是 “···” 的列。

因此，用人话表述本题题意：**统计含有断点的列数**。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        res = 0
        for i in range(len(strs[0])):
            for j in range(len(strs) - 1):
                if strs[j][i] > strs[j + 1][i]:  # 检查当前列是否单调不减
                    res += 1
                    break  # 这一列有问题，结束循环
        return res
```

```Java
// java
class Solution {
    public int minDeletionSize(String[] strs) {
        int res = 0;
        for (int i = 0; i < strs[0].length(); i++) {
            for (int j = 0; j < strs.length - 1; j++) {
                if (strs[j].charAt(i) > strs[j + 1].charAt(i)) {  // 检查当前列是否单调不减
                    res += 1;
                    break;  // 这一列有问题，结束循环
                }
            }
        }
        return res;
    }
}
```

- 时间复杂度： $O(n\times l)$，其中 $n$ 是数组 $strs$ 的长度，每个串长为 $l$
- 空间复杂度： $O(1)$，仅使用常数个额外变量

### 排序

将某一列的元素全部存储，看作一个序列。

非严格递减，等价于 **从小到大排序**。也即，如果排序后的序列值，与排序前的序列值不同，说明存在断点，答案 $+1$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        ans = 0
        for col in zip(*strs):
            # 排序后应该顺序一致
            if list(col) != sorted(col):
                ans += 1
        return ans
```

```Java
// java
class Solution {
    public int minDeletionSize(String[] strs) {
        int ans = 0;
        int n = strs.length;
        int m = strs[0].length();
        for (int j = 0; j < m; j++) {
            char[] col = new char[n];
            for (int i = 0; i < n; i++) {
                col[i] = strs[i].charAt(j);
            }
            // 排序后应该顺序一致
            char[] sortedCol = col.clone();
            Arrays.sort(sortedCol);
            if (!Arrays.equals(col, sortedCol)) {
                ans += 1;
            }
        }
        return ans;
    }
}
```

- 时间复杂度： $O(lnlogn)$
- 空间复杂度： $O(n)$，每一列含有 $n$ 个元素
