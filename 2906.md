[Problem: 2906. 构造乘积矩阵](https://leetcode.cn/problems/construct-product-matrix/description/)

### 方法：前后缀 & 优化

题意很简单，但是数据范围较大，我们不能直接模拟，否则超时。

题目要求对 $12345$ 取余，且 $12345$ 并不是一个质数 $12345 = 3 \times 5 \times 823$，**不能** 简单地计算所有元素的总乘积再除以当前元素。

1. **处理 0**：如果矩阵中有一个 $0$，除了 $0$ 那个位置，其他位置乘积为 $0$。如果有两个 $0$，所有位置全为 $0$。除法需要写很多 $if$ 分支来处理。
2. **取模限制**：在模运算中， $\frac{a}{b} \pmod m$ 需要计算 $b$ 的模逆元。只有当 $\gcd(b, m) = 1$ 时才存在逆元。这里的 $m=12345$ 是合数，很多数（如 $3$ 的倍数或 $5$ 的倍数）没有逆元，所以除法逻辑会失效。

对于任何一个位置 $(i, j)$，我们要计算的是除了它之外的所有元素的乘积。我们可以把矩阵想象成一个排好序的 **一维序列**。

对于序列中的某一个位置，它的 “除自身外乘积” = (该位置之前所有元素的乘积) $\times$ (该位置之后所有元素的乘积)。

也就是说，我们从左到右，从上到下，将 $grid$ 看作一个长度为 $n \times m$ 的一维数组。

- **前缀乘积**：存储在当前位置之前，按正向扫描顺序遇到的所有元素的乘积。
- **后缀乘积**：存储在当前位置之后，按反向扫描顺序遇到的所有元素的乘积。

定义前缀乘积数组，其中 $pre[k]$ 表示前 $k$ 个元素 $nums[0...k-1]$ 的乘积；以及后缀乘积数组，其中 $suf[k]$ 表示第 $k$ 个元素之后所有元素 $nums[k+1...L-1]$ 的乘积。

这里的 $nums$ 就是将 $grid$ 拉直后的一维数组。对于任意位置 $k$ 来说，其除自身外的乘积为：

$$
res[k] = (pre[k] \times suf[k]) \pmod{12345}
$$

最后将该一维结果重新填回 $n \times m$ 的矩阵中。

代码如下，已附加注释：

```Python
# python
class Solution:
    def constructProductMatrix(self, grid: List[List[int]]) -> List[List[int]]:
        n = len(grid)
        m = len(grid[0])
        L = n * m
        MOD = 12345
        
        # 将二维矩阵拉平为一维数组
        nums = []
        for i in range(n):
            for j in range(m):
                nums.append(grid[i][j])
        
        # 前缀乘积数组 pre
        pre = [1] * L
        for k in range(1, L):
            pre[k] = (pre[k-1] * nums[k-1]) % MOD
        
        # 后缀乘积数组 suf
        suf = [1] * L
        for k in range(L - 2, -1, -1):
            suf[k] = (suf[k+1] * nums[k+1]) % MOD
        
        # 计算
        ans = [[0] * m for _ in range(n)]
        for i in range(n):
            for j in range(m):
                # 映射回一维索引
                k = i * m + j
                # 当前位置结果 = 左边所有数的积 * 右边所有数的积
                ans[i][j] = (pre[k] * suf[k]) % MOD
        
        return ans
```

```Java
// java
class Solution {
    public int[][] constructProductMatrix(int[][] grid) {
        int n = grid.length;
        int m = grid[0].length;
        int L = n * m;
        int MOD = 12345;
        
        // 将二维矩阵拉平为一维数组
        int[] nums = new int[L];
        int idx = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                nums[idx++] = grid[i][j];
            }
        }
        
        // 前缀乘积数组 pre
        int[] pre = new int[L];
        pre[0] = 1;
        for (int k = 1; k < L; k++) {
            pre[k] = (int) (((long) pre[k - 1] * (nums[k - 1] % MOD)) % MOD);
        }
        
        // 后缀乘积数组 suf
        int[] suf = new int[L];
        suf[L - 1] = 1;
        for (int k = L - 2; k >= 0; k--) {
            suf[k] = (int) (((long) suf[k + 1] * (nums[k + 1] % MOD)) % MOD);
        }
        
        // 计算
        int[][] ans = new int[n][m];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                // 映射回一维索引
                int k = i * m + j;
                // 当前位置结果 = 左边所有数的积 * 右边所有数的积
                ans[i][j] = (int) (((long) pre[k] * suf[k]) % MOD);
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m\times n)$，其中 $m,n$ 是矩阵 $grid$ 的行数、列数，四次遍历
- 空间复杂度： $O(m\times n)$，表示数组 $nums,pre,suf$ 的大小

### 优化

将前缀积数组优化，仅使用 **一个变量** $pre$ 记录当前位置 **之前所有** 元素的乘积，初始为 $1$。

- 按照 $(0,0)$ 到 $(n-1, m-1)$ 的顺序遍历。
- 令 $res[i][j] = pre$。
- 更新前缀 $pre = (pre * grid[i][j]) % 12345$。

然后 **倒序遍历**，这样就无需开辟后缀数组。使用一个变量 $suf$ 记录当前位置 **之后所有** 元素的乘积，初始为 $1$。

- 按照 $(n-1, m-1)$ 到 $(0,0)$ 的反向顺序遍历。
- 令 $res[i][j] = (res[i][j] * suf) % 12345$，此时 $res[i][j]$ 是前缀乘后缀。
- 更新后缀 $suf = (suf * grid[i][j]) % 12345$。

如此一来，只需要一次前缀填充，以及一次后缀填充。并且无需开辟额外空间。

代码如下，已附加注释：

```Python
# python
class Solution:
    def constructProductMatrix(self, grid: List[List[int]]) -> List[List[int]]:
        n = len(grid)
        m = len(grid[0])
        MOD = 12345
        ans = [[0] * m for _ in range(n)]
        
        # 计算每个位置的前缀乘积
        pre = 1
        for i in range(n):
            for j in range(m):
                ans[i][j] = pre
                # 边乘边取余，防止溢出
                pre = (pre * grid[i][j]) % MOD
        
        # 乘以每个位置的后缀乘积
        suf = 1
        for i in range(n - 1, -1, -1):
            for j in range(m - 1, -1, -1):
                # 结果 = 当前的前缀乘积 * 后面的后缀乘积
                ans[i][j] = (ans[i][j] * suf) % MOD
                # 更新后缀乘积
                suf = (suf * grid[i][j]) % MOD
        
        return ans
```

```Java
// java
class Solution {
    public int[][] constructProductMatrix(int[][] grid) {
        int n = grid.length;
        int m = grid[0].length;
        int MOD = 12345;
        int[][] ans = new int[n][m];
        
        // 计算每个位置的前缀乘积
        long pre = 1;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                ans[i][j] = (int) pre;
                // 边乘边取余，防止溢出
                pre = (pre * (grid[i][j] % MOD)) % MOD;
            }
        }
        
        // 乘以每个位置的后缀乘积
        long suf = 1;
        for (int i = n - 1; i >= 0; i--) {
            for (int j = m - 1; j >= 0; j--) {
                // 结果 = 当前的前缀乘积 * 后面的后缀乘积
                ans[i][j] = (int) ((ans[i][j] * suf) % MOD);
                // 更新后缀乘积
                suf = (suf * (grid[i][j] % MOD)) % MOD;
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m\times n)$
- 空间复杂度： $O(1)$，返回值 $ans$ 一般不计入复杂度
