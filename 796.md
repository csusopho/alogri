[Problem: 796. 旋转字符串](https://leetcode.cn/problems/rotate-string/description/)

### 方法：模拟

题意很清晰，就是不断将字符串 $s$ 的 **第一个** 字符移到 **末尾**，并在每一次移动后检查它是否与 $goal$ 相等。

长度为 $n$ 的字符串，在旋转 $n$ 次后会回到原状，因此只需要尝试 $n$ 次即可。

代码如下，已附加注释：

```Python
# python
class Solution:
    def rotateString(self, s: str, goal: str) -> bool:
        # 长度必须相等
        if len(s) != len(goal):
            return False
        
        # 本身就相等
        if s == goal:
            return True
        
        # 模拟旋转过程
        current = s
        for _ in range(len(s)):
            # 旋转
            current = current[1:] + current[0]
            # 检查
            if current == goal:
                return True
        
        return False
```

```Java
// java
class Solution {
    public boolean rotateString(String s, String goal) {
        // 长度必须相等
        if (s.length() != goal.length()) {
            return false;
        }
        
        // 本身就相等
        if (s.equals(goal)) {
            return true;
        }
        
        // 模拟旋转过程
        String current = s;
        for (int i = 0; i < s.length(); i++) {
            // 旋转
            current = current.substring(1) + current.charAt(0);
            // 检查
            if (current.equals(goal)) {
                return true;
            }
        }
        
        return false;
    }
}
```

- 时间复杂度： $O(n^2)$，其中 $n$ 为串 $s$ 的长度
- 空间复杂度： $O(n)$，表示临时字符串 $current$ 的大小

实际上，本题有更巧妙的方法，利用旋转字符串的 **特性**：

如果 $goal$ 是由 $s$ 旋转而来的，那么 $goal$ 一定在 $s + s$ 这个合并后的字符串中。

因此，拼接一次后 $调库find$ 或者 $KMP$ 都行。
