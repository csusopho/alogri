[Problem: 3661. 可以被机器人摧毁的最大墙壁数目](https://leetcode.cn/problems/maximum-walls-destroyed-by-robots/description/)

### 方法：DP+二分/双指针

本题所有数值固定，只有机器人的射击不确定。每个机器人只有一颗子弹，可以选择向左或向右射击。

子弹碰到机器人停止，说明机器人 $i$ 的子弹最多只能影响到它 **左边** 的机器人 $i-1$ 和它 **右边** 的机器人 $i+1$ 之间的区域。

因此，我们可以将整条直线划分为若干不相交的区间：

1. **左侧边界**：第一个机器人左边的所有墙壁（只有第一个机器人向左射击能碰到）
2. **右侧边界**：最后一个机器人右边的所有墙壁（只有最后一个机器人向右射击能碰到）
3. **机器人位**：每个机器人所在位置脚下的墙壁
4. **间隙**：机器人 $i-1$ 和 $i$ 之间的开区间 $(P_{i-1}, P_i)$。

对于任何一个间隙 $(P_{i-1}, P_i)$：

- 只有机器人 $i-1$ 向右射击 **或** 机器人 $i$ 向左射击能够摧毁这里的墙壁。
- 如果在同一个间隙内，左边机器人向右射，右边机器人向左射，那么这个间隙被摧毁的墙壁数量是两束子弹路径的 **并集**。

因此，在预处理中，我们将机器人按位置排序；将墙壁按位置排序。统计每个机器人位置 $P_i$ 脚下的墙壁数量 $Wat[i]$。以及第一个机器人左侧射程内的墙壁数 $ExtraL$ 和最后一个机器人右侧射程内的墙壁数 $ExtraR$。

对于每个间隙 $(P_{i-1}, P_i)$，计算出：

- $Lonly[i]$：机器人 $i$ 向左射击能击中的该间隙内的墙壁数。
- $Ronly[i]$：机器人 $i-1$ 向右射击能击中的该间隙内的墙壁数。
- $Union[i]$：两者同时发生时，即 $i-1$ 向右 $i$ 向左，该间隙内被击中的墙壁总数。

---

注意，机器人 $i$ 的决定（向左还是向右），只影响它与机器人 $i-1$ 之间的那个间隙，以及它与机器人 $i+1$ 之间的那个间隙，**局部影响**！

一旦机器人 $i$ 的方向选定了，它对机器人 $i+2, i+3 \dots$ **没有任何** 直接影响。

当一个问题的全局最优解，可以分解为一系列 “只与邻居有关” 的局部决策时，就想到使用 $DP$。

**贪心尝试**：让每个机器人选能打掉更多墙的方向，不行吗？

举个反例，假设机器人 $A$ 和 $B$ 之间有很多墙：

- 如果 $A$ 向右射，能打掉 $10$ 面墙。
- 但如果 $A$ 向右射，由于 $A$ 的子弹占用了空间，导致 $B$ **必须** 向右射（因为 $B$ 向左射会和 $A$ 重合，没收益），而 $B$ 向右射只能打掉 $2$ 面墙。
- 反之，如果 $A$ 牺牲自己向左射打掉 $6$ 面，那么 $B$ 就可以放心地向左射打掉 $10$ 面。

当前最优不代表全局最优。需要一种方法，**能够保留之前的多种可能性，等看到后面的时候再做最终决定**。这就是 $DP$ 的本质。

---

那么 $DP$ 状态如何定义？我只需要知道第 $i-1$ 个机器人是 **向左** 还是 **向右**。

- 如果 $i-1$ 向左，它不占用 $i-1$ 和 $i$ 之间的空间，第 $i$ 个机器人向左射时能拿满。
- 如果 $i-1$ 向右，它占用了空间，第 $i$ 个机器人向左射时，两人会有重叠。

因此，状态只与坐标与选择的方向有关，也即 `dp[i][方向]`。

**定义** $dp[i][state]$ 为前 $i$ 个机器人能摧毁的最大墙壁数。其中 $state$ 为 $0$ 表示机器人 $i$ 向左射击，为 $1$ 表示机器人 $i$ 向右射击。

在计算机器人 $i$ 的选择时，只需要看机器人 $i-1$ 的状态：

1）如果 $i$ **向左射**：

- 若 $i-1$ 向左：间隙 $(P_{i-1}, P_i)$ 贡献 $Lonly[i]$。
- 若 $i-1$ 向右：间隙 $(P_{i-1}, P_i)$ 贡献 $Union[i]$。

2）如果 $i$ **向右射**：

- 若 $i-1$ 向左：间隙 $(P_{i-1}, P_i)$ 贡献 $0$。
- 若 $i-1$ 向右：间隙 $(P_{i-1}, P_i)$ 贡献 $Ronly[i]$。

总的来看，我们是将 “无限直线” 分解为 “独立间隙”，巧妙地处理了机器人阻挡子弹的逻辑，接着利用 $DP$ 解决了机器人决策之间的冲突。

如何计算 $Lonly$ 等数组？**最暴力** 的方法：使用二分查找。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxWalls(self, robots: List[int], distance: List[int], walls: List[int]) -> int:
        n = len(robots)
        # 绑在一起按位置排序
        pts = sorted(zip(robots, distance))
        walls.sort()
        
        # 统计机器人脚下的墙壁
        W_at = [0] * n
        for i in range(n):
            W_at[i] = bisect.bisect_right(walls, pts[i][0]) - bisect.bisect_left(walls, pts[i][0])
        
        # 第一个机器人向左的墙壁数
        idx_l_start = bisect.bisect_left(walls, pts[0][0] - pts[0][1])
        idx_l_end = bisect.bisect_left(walls, pts[0][0])
        ExtraL = max(0, idx_l_end - idx_l_start)
        
        # 最后一个机器人向右的墙壁数
        idx_r_start = bisect.bisect_right(walls, pts[-1][0])
        idx_r_end = bisect.bisect_right(walls, pts[-1][0] + pts[-1][1])
        ExtraR = max(0, idx_r_end - idx_r_start)
        
        # 计算每个间隙的贡献
        L_only = [0] * n
        R_only = [0] * n
        Union = [0] * n
        
        for i in range(1, n):
            p_prev, d_prev = pts[i-1]
            p_curr, d_curr = pts[i]
            
            # 提取间隙内的墙壁数
            w_start = bisect.bisect_right(walls, p_prev)
            w_end = bisect.bisect_left(walls, p_curr)
            gap_w = walls[w_start : w_end]
            
            if gap_w:
                # i-1向右射击覆盖的墙
                r_cnt = bisect.bisect_right(gap_w, p_prev + d_prev)
                R_only[i] = r_cnt
                # i向左射击覆盖的墙
                l_cnt = len(gap_w) - bisect.bisect_left(gap_w, p_curr - d_curr)
                L_only[i] = l_cnt
                # 两者覆盖墙的并集 = 总数 - 两边都没射到的中间部分
                neither_start = bisect.bisect_right(gap_w, p_prev + d_prev)
                neither_end = bisect.bisect_left(gap_w, p_curr - d_curr)
                Union[i] = len(gap_w) - max(0, neither_end - neither_start)
        
        # dp[i][0/1]: 机器人 i 向左/右射时，前 i 个机器人摧毁的最大墙壁数
        dp = [[0, 0] for _ in range(n)]
        
        # 初始
        dp[0][0] = W_at[0] + ExtraL
        dp[0][1] = W_at[0]
        
        for i in range(1, n):
            # i-1向左射 + i向左射
            # i-1向右射 + i向左射
            dp[i][0] = max(dp[i-1][0] + L_only[i], 
                        dp[i-1][1] + Union[i]) + W_at[i]
            
            # i-1向左射 + i向右射
            # i-1向右射 + i向右射
            dp[i][1] = max(dp[i-1][0] + 0, 
                        dp[i-1][1] + R_only[i]) + W_at[i]
        
        # 最后一步可能向右射击
        return max(dp[-1][0], dp[-1][1] + ExtraR)
```

```Java
// java
class Solution {
    public int maxWalls(int[] robots, int[] distance, int[] walls) {
        int n = robots.length;
        // 绑在一起按位置排序
        int[][] pts = new int[n][2];
        for (int i = 0; i < n; i++) {
            pts[i][0] = robots[i];
            pts[i][1] = distance[i];
        }
        Arrays.sort(pts, (a, b) -> Integer.compare(a[0], b[0]));
        Arrays.sort(walls);
        
        // 统计机器人脚下的墙壁
        int[] W_at = new int[n];
        for (int i = 0; i < n; i++) {
            W_at[i] = upperBound(walls, pts[i][0]) - lowerBound(walls, pts[i][0]);
        }
        
        // 第一个机器人向左的墙壁数
        int idx_l_start = lowerBound(walls, pts[0][0] - pts[0][1]);
        int idx_l_end = lowerBound(walls, pts[0][0]);
        int ExtraL = Math.max(0, idx_l_end - idx_l_start);
        
        // 最后一个机器人向右的墙壁数
        int idx_r_start = upperBound(walls, pts[n - 1][0]);
        int idx_r_end = upperBound(walls, pts[n - 1][0] + pts[n - 1][1]);
        int ExtraR = Math.max(0, idx_r_end - idx_r_start);
        
        // 计算每个间隙的贡献
        int[] L_only = new int[n];
        int[] R_only = new int[n];
        int[] Union = new int[n];
        
        for (int i = 1; i < n; i++) {
            int p_prev = pts[i - 1][0], d_prev = pts[i - 1][1];
            int p_curr = pts[i][0], d_curr = pts[i][1];
            
            // 提取间隙内的墙壁范围
            int w_start = upperBound(walls, p_prev);
            int w_end = lowerBound(walls, p_curr);
            
            if (w_start < w_end) {
                int gap_len = w_end - w_start;
                // i-1向右射击覆盖的墙
                int r_cnt = upperBound(walls, p_prev + d_prev, w_start, w_end) - w_start;
                R_only[i] = r_cnt;
                // i向左射击覆盖的墙
                int l_cnt = w_end - lowerBound(walls, p_curr - d_curr, w_start, w_end);
                L_only[i] = l_cnt;
                // 两者覆盖墙的并集 = 总数 - 两边都没射到的中间部分
                int neither_start = upperBound(walls, p_prev + d_prev, w_start, w_end);
                int neither_end = lowerBound(walls, p_curr - d_curr, w_start, w_end);
                Union[i] = gap_len - Math.max(0, neither_end - neither_start);
            }
        }
        
        // dp[i][0/1]: 机器人 i 向左/右射时，前 i 个机器人摧毁的最大墙壁数
        int[][] dp = new int[n][2];
        
        // 初始
        dp[0][0] = W_at[0] + ExtraL;
        dp[0][1] = W_at[0];
        
        for (int i = 1; i < n; i++) {
            // i-1向左射 + i向左射
            // i-1向右射 + i向左射
            dp[i][0] = Math.max(dp[i - 1][0] + L_only[i], 
                               dp[i - 1][1] + Union[i]) + W_at[i];
            
            // i-1向左射 + i向右射
            // i-1向右射 + i向右射
            dp[i][1] = Math.max(dp[i - 1][0] + 0, 
                               dp[i - 1][1] + R_only[i]) + W_at[i];
        }
        
        // 最后一步可能向右射击
        return Math.max(dp[n - 1][0], dp[n - 1][1] + ExtraR);
    }

    private int lowerBound(int[] arr, int target) {
        return lowerBound(arr, target, 0, arr.length);
    }

    private int lowerBound(int[] arr, int target, int left, int right) {
        while (left < right) {
            int mid = left + (right - left) / 2;
            if (arr[mid] >= target) right = mid;
            else left = mid + 1;
        }
        return left;
    }

    private int upperBound(int[] arr, int target) {
        return upperBound(arr, target, 0, arr.length);
    }

    private int upperBound(int[] arr, int target, int left, int right) {
        while (left < right) {
            int mid = left + (right - left) / 2;
            if (arr[mid] > target) right = mid;
            else left = mid + 1;
        }
        return left;
    }
}
```

- 时间复杂度： $O(mlogm+nlogn+nlogm)$，其中 $n,m$ 分别是数组 $robots,walls$ 长度，快速排序，对 $n$ 个机器人进行二分查找墙壁
- 空间复杂度： $O(n+m)$，表示列表 $pts$ 以及数组 $dp$ 等的大小

### 双指针

在一维直线上，我们就像 **拿着一把尺子从左往右扫**。在扫的过程中，顺便把每个机器人之间的墙壁 “分袋装好”，然后直接计算每个袋子里有多少墙能被扫到。

因此，通俗来说，就是把墙壁分成一个个 “区域”。

- 区域 $0$：第 $1$ 个机器人左边的墙。
- 区域 $i$：第 $i$ 个和第 $i+1$ 个机器人之间的墙。
- 区域 $N$：最后一个机器人右边的墙。
- **特殊位**：正好在机器人脚底下的墙。

那么，没必要用二分在每段区间都去墙壁数组里 “搜” 一遍，而是像吃豆子一样，随着机器人的推进，一个接一个地处理墙壁。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxWalls(self, robots: List[int], distance: List[int], walls: List[int]) -> int:
        # 绑定在一起排序
        pts = sorted(zip(robots, distance))
        walls.sort()
        n = len(pts)
        m = len(walls)
        wall_ptr = 0
        
        # 第一个机器人左边的墙
        first_robot_pos, first_dist = pts[0]
        extra_l = 0
        while wall_ptr < m and walls[wall_ptr] < first_robot_pos:
            if walls[wall_ptr] >= first_robot_pos - first_dist:
                extra_l += 1
            wall_ptr += 1
        
        # dp[i][0/1] 表示第 i 个机器人向左/右射时，累计摧毁的最大墙数
        dp = [[0, 0] for _ in range(n)]
        
        # 第一个机器人，看脚下有几面墙
        at_walls = 0
        while wall_ptr < m and walls[wall_ptr] == first_robot_pos:
            at_walls += 1
            wall_ptr += 1
        # 初始化
        dp[0][0] = extra_l + at_walls
        dp[0][1] = at_walls
        
        for i in range(1, n):
            prev_p, prev_d = pts[i-1]
            curr_p, curr_d = pts[i]
            
            # 统计当前间隙 (prev_p, curr_p) 里的墙
            gap_walls = []
            while wall_ptr < m and walls[wall_ptr] < curr_p:
                gap_walls.append(walls[wall_ptr])
                wall_ptr += 1
            
            # 统计当前机器人 curr_p 脚下的墙
            at_walls = 0
            while wall_ptr < m and walls[wall_ptr] == curr_p:
                at_walls += 1
                wall_ptr += 1
            
            # 在这个间隙里
            count_r = 0  # 左边机器人往右射中数
            count_l = 0  # 右边机器人往左射中数
            count_union = 0 # 两人对着中间射中数
            
            for w in gap_walls:
                hit_by_prev = (w <= prev_p + prev_d)
                hit_by_curr = (w >= curr_p - curr_d)
                if hit_by_prev: count_r += 1
                if hit_by_curr: count_l += 1
                if hit_by_prev or hit_by_curr: count_union += 1
            
            # 当前机器人 i 向左射
            dp[i][0] = max(dp[i-1][0] + count_l, dp[i-1][1] + count_union) + at_walls
            # 当前机器人 i 向右射：
            dp[i][1] = max(dp[i-1][0] + 0, dp[i-1][1] + count_r) + at_walls
        
        # 处理最后一个机器人右边的墙
        last_robot_pos, last_dist = pts[-1]
        extra_r = 0
        while wall_ptr < m:
            if walls[wall_ptr] <= last_robot_pos + last_dist:
                extra_r += 1
            wall_ptr += 1
        
        return max(dp[-1][0], dp[-1][1] + extra_r)
```

```Java
// java
class Solution {
    public int maxWalls(int[] robots, int[] distance, int[] walls) {
        int n = robots.length;
        // 绑定在一起排序
        int[][] pts = new int[n][2];
        for (int i = 0; i < n; i++) {
            pts[i][0] = robots[i];
            pts[i][1] = distance[i];
        }
        Arrays.sort(pts, (a, b) -> Integer.compare(a[0], b[0]));
        Arrays.sort(walls);
        int m = walls.length;
        int wall_ptr = 0;
        
        // 第一个机器人左边的墙
        int first_robot_pos = pts[0][0], first_dist = pts[0][1];
        int extra_l = 0;
        while (wall_ptr < m && walls[wall_ptr] < first_robot_pos) {
            if (walls[wall_ptr] >= first_robot_pos - first_dist) {
                extra_l += 1;
            }
            wall_ptr += 1;
        }
        
        // dp[i][0/1] 表示第 i 个机器人向左/右射时，累计摧毁的最大墙数
        int[][] dp = new int[n][2];
        
        // 第一个机器人，看脚下有几面墙
        int at_walls = 0;
        while (wall_ptr < m && walls[wall_ptr] == first_robot_pos) {
            at_walls += 1;
            wall_ptr += 1;
        }
        // 初始化
        dp[0][0] = extra_l + at_walls;
        dp[0][1] = at_walls;
        
        for (int i = 1; i < n; i++) {
            int prev_p = pts[i - 1][0], prev_d = pts[i - 1][1];
            int curr_p = pts[i][0], curr_d = pts[i][1];
            
            // 统计当前间隙 (prev_p, curr_p) 里的墙
            List<Integer> gap_walls = new ArrayList<>();
            while (wall_ptr < m && walls[wall_ptr] < curr_p) {
                gap_walls.add(walls[wall_ptr]);
                wall_ptr += 1;
            }
            
            // 统计当前机器人 curr_p 脚下的墙
            at_walls = 0;
            while (wall_ptr < m && walls[wall_ptr] == curr_p) {
                at_walls += 1;
                wall_ptr += 1;
            }
            
            // 在这个间隙里
            int count_r = 0;  // 左边机器人往右射中数
            int count_l = 0;  // 右边机器人往左射中数
            int count_union = 0; // 两人对着中间射中数
            
            for (int w : gap_walls) {
                boolean hit_by_prev = (w <= prev_p + prev_d);
                boolean hit_by_curr = (w >= curr_p - curr_d);
                if (hit_by_prev) count_r += 1;
                if (hit_by_curr) count_l += 1;
                if (hit_by_prev || hit_by_curr) count_union += 1;
            }
            
            // 当前机器人 i 向左射
            dp[i][0] = Math.max(dp[i - 1][0] + count_l, dp[i - 1][1] + count_union) + at_walls;
            // 当前机器人 i 向右射：
            dp[i][1] = Math.max(dp[i - 1][0] + 0, dp[i - 1][1] + count_r) + at_walls;
        }
        
        // 处理最后一个机器人右边的墙
        int last_robot_pos = pts[n - 1][0], last_dist = pts[n - 1][1];
        int extra_r = 0;
        while (wall_ptr < m) {
            if (walls[wall_ptr] <= last_robot_pos + last_dist) {
                extra_r += 1;
            }
            wall_ptr += 1;
        }
        
        return Math.max(dp[n - 1][0], dp[n - 1][1] + extra_r);
    }
}
```

- 空间复杂度： $O(n+m)$，同理
