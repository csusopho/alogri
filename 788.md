[Problem: 788. 旋转数字](https://leetcode.cn/problems/rotated-digits/description/)

### 方法：模拟

一个数 $X$ 是 “好数”，必须满足两个条件：

1）**旋转后有效**：数 $X$ 中的每一位数字在旋转 $180$ 度后必须仍然是有效的数字。

- 有效的数字有： $0, 1, 8$（旋转后还是自己）和 $2, 5, 6, 9$（旋转后变成另一个数）。
- 无效的数字有： $3, 4, 7$。
- 这意味着 $X$ 的所有数位必须来自集合 $\{0, 1, 2, 5, 6, 8, 9\}$。

2）**旋转后不同**：旋转后的数必须和原数 $X$ 不同。

- 如果 $X$ 只包含 $\{0, 1, 8\}$，那么旋转后 $X$ 保持不变。
- 因此，为了使 $X$ 不同， $X$ 中 **必须包含至少一个** 来自集合 $\{2, 5, 6, 9\}$ 的数字。

**总结**：为了保证该数字合法，

- 合法性：**每一位** 都必须是 $\{0, 1, 8, 2, 5, 6, 9\}$ 中的一个。
- 差异性：该数字中必须包含 **至少一个** $\{2, 5, 6, 9\}$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def rotatedDigits(self, n: int) -> int:
        ans = 0
        # 合法的数字集合
        valid_set = {'0', '1', '8', '2', '5', '6', '9'}
        # 会发生变化的数字集合
        diff_set = {'2', '5', '6', '9'}
        
        for i in range(1, n + 1):
            s = str(i)
            
            # 检查是否每一位都在合法集合中
            is_valid = True
            for char in s:
                if char not in valid_set:
                    is_valid = False
                    break
            
            if not is_valid:
                continue
            
            # 检查是否包含至少一个旋转后会变大的数字
            has_diff = False
            for char in s:
                if char in diff_set:
                    has_diff = True
                    break
            
            if has_diff:
                ans += 1
        
        return ans
```

```Java
// java
class Solution {
    public int rotatedDigits(int n) {
        int ans = 0;
        
        for (int i = 1; i <= n; i++) {
            String s = String.valueOf(i);
            
            // 检查是否每一位都在合法集合中
            boolean isValid = true;
            for (char c : s.toCharArray()) {
                if (c != '0' && c != '1' && c != '8' && c != '2' && c != '5' && c != '6' && c != '9') {
                    isValid = false;
                    break;
                }
            }
            
            if (!isValid) {
                continue;
            }
            
            // 检查是否包含至少一个旋转后会变大的数字
            boolean hasDiff = false;
            for (char c : s.toCharArray()) {
                if (c == '2' || c == '5' || c == '6' || c == '9') {
                    hasDiff = true;
                    break;
                }
            }
            
            if (hasDiff) {
                ans += 1;
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(n\times k)$，其中 $k$ 表示数字的平均位数
- 空间复杂度： $O(k)$，表示临时字符串 $s$ 的大小
