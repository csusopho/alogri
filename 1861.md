[Problem: 1861. 旋转盒子](https://leetcode.cn/problems/rotating-the-box/description/)

### 方法：双指针

真实过程是：先旋转矩阵，然后石头 “向下” 滚动。但是，我们可以 **换个角度** 思考：

1. 在原矩阵中，先让石头向 “右侧” 滚动。
2. 接着，将矩阵顺时针旋转 $90$ 度。

在原始矩阵中，箱子的 “底部” 变成了旋转后的 “右边”。旋转 $90$ 度后，原本的 “向右” 方向就变成了 “向下” 方向。因此，在旋转 **之前**，先处理每一行，让其中的石头尽量往右边（即未来的下方）靠。

> $ps$：不换角度一样做法，只不过这种解法更易理解。

**如何模拟石头右移？**

从每一行的 **末尾（最右侧）** 开始向前遍历。并且，维护一个指针 $available$，表示当前最靠右的 **空闲** 位置。

- 如果遇到石头 `'#'`：将其移动到 $available$ 位置，然后 $available$ 左移一位。
- 如果遇到障碍物 `'*'`：障碍物不动，但它挡住了后面的石头，所以 $available$ 必须更新为障碍物左侧的位置。
- 如果遇到空位 `'.'`：不做操作，等待石头落入。

注意，上述操作的 **本质** 逻辑为：处于第 $r$ 行，左侧指针为第 $c$ 列元素，右侧指针为第 $available$ 列元素。左侧指针指向当前位置，右侧指针指向等待填充位置。

遍历这一行的过程中，通过双指针的交换，不断将左侧元素填充到右侧目标位置。

在代码实现时，有个 **细节**：发现石头时，先将原位置设为 `'.'`，再将目标位置设为 `'#'`。处理原地不动的情况，颠倒顺序会出错！

**旋转矩阵的本质是什么？**

仔细观察，顺时针旋转 $90$ 度后，原矩阵坐标 $(r, c)$ 会映射到新矩阵的 $(c, (m-1) - r)$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def rotateTheBox(self, boxGrid: List[List[str]]) -> List[List[str]]:
        m = len(boxGrid)
        n = len(boxGrid[0])
        
        # 模拟重力，让石头在每一行内向右掉落
        for r in range(m):
            # 当前石头可以掉落的最右侧位置
            available = n - 1
            # 从右向左遍历列
            for c in range(n - 1, -1, -1):
                if boxGrid[r][c] == '#':
                    # 发现石头，移动到 available 位置
                    boxGrid[r][c] = '.'
                    boxGrid[r][available] = '#'
                    available -= 1
                elif boxGrid[r][c] == '*':
                    # 发现障碍物，available 更新为障碍物左边
                    available = c - 1
        
        res = [['' for _ in range(m)] for _ in range(n)]
        # 搬到新位置
        for r in range(m):
            for c in range(n):
                # 原位置 (r, c) -> 新位置 (c, m - 1 - r)
                res[c][m - 1 - r] = boxGrid[r][c]
        
        return res
```

```Java
// java
class Solution {
    public char[][] rotateTheBox(char[][] boxGrid) {
        int m = boxGrid.length;
        int n = boxGrid[0].length;
        
        // 模拟重力，让石头在每一行内向右掉落
        for (int r = 0; r < m; r++) {
            // 当前石头可以掉落的最右侧位置
            int available = n - 1;
            // 从右向左遍历列
            for (int c = n - 1; c >= 0; c--) {
                if (boxGrid[r][c] == '#') {
                    // 发现石头，移动到 available 位置
                    boxGrid[r][c] = '.';
                    boxGrid[r][available] = '#';
                    available -= 1;
                } else if (boxGrid[r][c] == '*') {
                    // 发现障碍物，available 更新为障碍物左边
                    available = c - 1;
                }
            }
        }
        
        char[][] res = new char[n][m];
        // 搬到新位置
        for (int r = 0; r < m; r++) {
            for (int c = 0; c < n; c++) {
                // 原位置 (r, c) -> 新位置 (c, m - 1 - r)
                res[c][m - 1 - r] = boxGrid[r][c];
            }
        }
        
        return res;
    }
}
```

- 时间复杂度： $O(m\times n)$，其中 $m,n$ 为矩阵 $boxGrid$ 的行/列数，两次遍历
- 空间复杂度： $O(1)$，返回值 $res$ 一般不计入复杂度
