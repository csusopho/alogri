[Problem: 2840. 判断通过操作能否让字符串相等 II](https://leetcode.cn/problems/check-if-strings-can-be-made-equal-with-operations-ii/description/)

### 方法：排序 & 哈希

什么是叫做 $j - i$ 是偶数？比如 $3-1,4-2$，说明 $i$ 和 $j$ 的奇偶性相同，同为奇数或同为偶数。这意味着：**奇数下标的字符只能和奇数下标的字符交换，偶数下标的字符只能和偶数下标的字符交换。**

最终目标是让 $s1$ 和 $s2$ 相等，也就是说，只要 $s1$ 中所有奇数下标字符组成的 **集合** 与 $s2$ 中奇数下标字符 **集合** 相同，且 $s1$ 的偶数下标字符集合与 $s2$ 的相同，就可以通过有限次交换使两个字符串相等。

因此，最简单的想法就是 **排序**：提取奇数位和偶数位字符，排序后比较是否相等。

代码如下，已附加注释：

```Python
# python
class Solution:
    def checkStrings(self, s1: str, s2: str) -> bool:
        n = len(s1)
        # 分别存放 s1 和 s2 的偶数位和奇数位字符
        even1, odd1 = [], []
        even2, odd2 = [], []
        
        for i in range(n):
            if i % 2 == 0:
                even1.append(s1[i])
                even2.append(s2[i])
            else:
                odd1.append(s1[i])
                odd2.append(s2[i])
        
        # 排序
        even1.sort()
        even2.sort()
        odd1.sort()
        odd2.sort()
        
        # 对应的集合必须完全一致
        return even1 == even2 and odd1 == odd2
```

```Java
// java
class Solution {
    public boolean checkStrings(String s1, String s2) {
        int n = s1.length();
        // 分别存放 s1 和 s2 的偶数位和奇数位字符
        StringBuilder even1 = new StringBuilder(), odd1 = new StringBuilder();
        StringBuilder even2 = new StringBuilder(), odd2 = new StringBuilder();
        
        for (int i = 0; i < n; i++) {
            if (i % 2 == 0) {
                even1.append(s1.charAt(i));
                even2.append(s2.charAt(i));
            } else {
                odd1.append(s1.charAt(i));
                odd2.append(s2.charAt(i));
            }
        }
        
        char[] e1 = even1.toString().toCharArray();
        char[] e2 = even2.toString().toCharArray();
        char[] o1 = odd1.toString().toCharArray();
        char[] o2 = odd2.toString().toCharArray();

        // 排序
        Arrays.sort(e1);
        Arrays.sort(e2);
        Arrays.sort(o1);
        Arrays.sort(o2);
        
        // 对应的集合必须完全一致
        return Arrays.equals(e1, e2) && Arrays.equals(o1, o2);
    }
}
```

- 时间复杂度： $O(n\times logn)$，其中 $n$ 是串 $s1$ 的长度，快排
- 空间复杂度： $O(n)$，表示列表 $even,odd$ 的大小

### 哈希

或者，我们使用 **哈希表** 统计奇数位和偶数位字符出现的次数，最后比较频率是否一致。

本题只有小写字母，可以换成长度为 $26$ 的数组来替换传统 $Map$。

同时，分别统计 $s1$ 和 $s2$ 的偶数位 **频率差**，以及奇数位频率差，而不是传统的出现次数。在某些数据范围特别大的题目里，可能有奇效。

代码如下，已附加注释：

```Python
# python
class Solution:
    def checkStrings(self, s1: str, s2: str) -> bool:
        n = len(s1)
        # 使用数组模拟哈希表
        even_count = [0] * 26
        odd_count = [0] * 26
        
        for i in range(n):
            idx1 = ord(s1[i]) - ord('a')
            idx2 = ord(s2[i]) - ord('a')
            
            if i % 2 == 0:
                # 偶数位：s1 增加计数，s2 减少计数
                even_count[idx1] += 1
                even_count[idx2] -= 1
            else:
                # 奇数位：s1 增加计数，s2 减少计数
                odd_count[idx1] += 1
                odd_count[idx2] -= 1
        
        # 在偶数位/奇数位上的字符构成必须一致
        for i in range(26):
            if even_count[i] != 0 or odd_count[i] != 0:
                return False
        
        return True
```

```Java
// java
class Solution {
    public boolean checkStrings(String s1, String s2) {
        int n = s1.length();
        // 使用数组模拟哈希表
        int[] even_count = new int[26];
        int[] odd_count = new int[26];
        
        for (int i = 0; i < n; i++) {
            int idx1 = s1.charAt(i) - 'a';
            int idx2 = s2.charAt(i) - 'a';
            
            if (i % 2 == 0) {
                // 偶数位：s1 增加计数，s2 减少计数
                even_count[idx1]++;
                even_count[idx2]--;
            } else {
                // 奇数位：s1 增加计数，s2 减少计数
                odd_count[idx1]++;
                odd_count[idx2]--;
            }
        }
        
        // 在偶数位/奇数位上的字符构成必须一致
        for (int i = 0; i < 26; i++) {
            if (even_count[i] != 0 || odd_count[i] != 0) {
                return false;
            }
        }
        
        return true;
    }
}
```

- 时间复杂度： $O(n)$，一次遍历
- 空间复杂度： $O(|\Sigma|)$，其中 $|\Sigma|=26$ 表示 $even,odd$ 数组的大小
