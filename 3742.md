[Problem: 3742. 网格中得分最大的路径](https://leetcode.cn/problems/maximum-path-score-in-a-grid/description/)

### 方法：DP

说白了，还是从左上到右下，网格图，查找最大路径，联想 $DP$。

需要记录当前的位置 $(i, j)$ 以及当前已经消耗的费用 $c$。因此，**定义** $dp[i][j][c]$ 表示到达单元格 $(i, j)$ 且 **总花费恰好为 $c$** 时，所能获得的最大分数。

初始全部设为 $-1$，表示该状态目前不可达。

对于当前格子 $(i, j)$ 来说，设其值为 $v = grid[i][j]$：

- 当前花费增量 $cost\_inc$：如果 $v > 0$，则 $cost\_inc = 1$；否则为 $0$。
- 当前分数增量 $score\_inc$： $score\_inc = v$。

每个格子只能向右/下移动，状态 $(i, j, c)$ 可以从两个方向转移而来：

1. **从上方来**：$dp[i-1][j][c - cost\_inc]$
2. **从左方来**：$dp[i][j-1][c - cost\_inc]$

根据上述推导，得出 **状态转移方程**：

$$
dp[i][j][c] = \max(dp[i-1][j][c - cost\_inc], dp[i][j-1][c - cost\_inc]) + score\_inc
$$

前提条件： $c \ge cost\_inc$ 且前驱状态不为 $-1$。

计算 **起点** 的费用 $c_0$ 和分数 $s_0$。如果 $c_0 \le k$，则 $dp[0][0][c_0] = s_0$。

遍历结束后，如果 $dp[m-1][n-1]$ 中所有的值都是 $-1$，说明没有合法路径，返回 $-1$。否则，遍历所有 $dp[m-1][n-1]$，寻找所有费用 $c$ 里的最大分数。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxPathScore(self, grid: List[List[int]], k: int) -> int:
        m, n = len(grid), len(grid[0])
        # dp[i][j][c] 表示到达 (i, j) 且花费为 c 时的最大分数
        dp = [[[-1] * (k + 1) for _ in range(n)] for _ in range(m)]
        
        # 起点
        v0 = grid[0][0]
        c0 = 1 if v0 > 0 else 0
        s0 = v0
        if c0 <= k:
            dp[0][0][c0] = s0
        else:
            # 起点的花费就超过了 k
            return -1
        
        for i in range(m):
            for j in range(n):
                if i == 0 and j == 0:
                    continue
                
                v = grid[i][j]
                cost_inc = 1 if v > 0 else 0
                score_inc = v
                
                # 遍历所有可能的当前费用 c
                for c in range(cost_inc, k + 1):
                    # 从上方转移
                    res_up = -1
                    if i > 0:
                        res_up = dp[i-1][j][c - cost_inc]
                    
                    # 从左方转移
                    res_left = -1
                    if j > 0:
                        res_left = dp[i][j-1][c - cost_inc]
                    
                    # 取最大
                    best_prev = max(res_up, res_left)
                    
                    # 前驱状态可达，更新当前状态
                    if best_prev != -1:
                        dp[i][j][c] = best_prev + score_inc
        
        # 寻找所有费用 c 里的最大分数
        ans = -1
        for c in range(k + 1):
            if dp[m-1][n-1][c] > ans:
                ans = dp[m-1][n-1][c]
                
        return ans
```

```Java
// java
class Solution {
    public int maxPathScore(int[][] grid, int k) {
        int m = grid.length;
        int n = grid[0].length;
        // dp[i][j][c] 表示到达 (i, j) 且花费为 c 时的最大分数
        int[][][] dp = new int[m][n][k + 1];
        
        // 初始化 dp 数组为 -1
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                Arrays.fill(dp[i][j], -1);
            }
        }
        
        // 起点
        int v0 = grid[0][0];
        int c0 = v0 > 0 ? 1 : 0;
        int s0 = v0;
        if (c0 <= k) {
            dp[0][0][c0] = s0;
        } else {
            // 起点的花费就超过了 k
            return -1;
        }
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (i == 0 && j == 0) {
                    continue;
                }
                
                int v = grid[i][j];
                int cost_inc = v > 0 ? 1 : 0;
                int score_inc = v;
                
                // 遍历所有可能的当前费用 c
                for (int c = cost_inc; c <= k; c++) {
                    // 从上方转移
                    int res_up = -1;
                    if (i > 0) {
                        res_up = dp[i - 1][j][c - cost_inc];
                    }
                    
                    // 从左方转移
                    int res_left = -1;
                    if (j > 0) {
                        res_left = dp[i][j - 1][c - cost_inc];
                    }
                    
                    // 取最大
                    int best_prev = Math.max(res_up, res_left);
                    
                    // 前驱状态可达，更新当前状态
                    if (best_prev != -1) {
                        dp[i][j][c] = best_prev + score_inc;
                    }
                }
            }
        }
        
        // 寻找所有费用 c 里的最大分数
        int ans = -1;
        for (int c = 0; c <= k; c++) {
            if (dp[m - 1][n - 1][c] > ans) {
                ans = dp[m - 1][n - 1][c];
            }
        }
                
        return ans;
    }
}
```

- 时间复杂度： $O(n\times n\times k)$，其中 $m,n$ 是矩阵 $grid$ 的行/列数
- 空间复杂度： $O(n\times n\times k)$，表示数组 $dp$ 的大小
