[Problem: 1594. 矩阵的最大非负积](https://leetcode.cn/problems/maximum-non-negative-product-in-a-matrix/description/)

### 方法：三维DP & 优化

网格图，每一次只能向右/向下移动一步。这就是经典的网格 $DP$，示例题目是 [63](https://leetcode.cn/problems/unique-paths-ii/description/)，题解为 [传送门](https://leetcode.cn/problems/unique-paths-ii/solutions/3067500/chang-gui-san-jie-ji-yi-hua-sou-suo-dong-4y1r/)。或者去看看灵神的题解，从 $DFS$ 开始推导。

本题的关键点在于：**负负得正**。如果只记录当前格子的最大正乘积，当遇到负数时，之前的最小乘积（绝对值最大的负数）反而可能变成之后的最大乘积。

因此，我们需要在每个格子同时维护两个值：**最大积** 和 **最小积**。

假设当前处于 $(i, j)$ 位置，只能从它的上方 $(i-1, j)$ 或左方 $(i, j-1)$ 到达。

设 $dp\_max[i][j]$ 为到达 $(i, j)$ 的最大积，$dp\_min[i][j]$ 为到达 $(i, j)$ 的最小积。分情况讨论：最大正数只能是 $正数\times 正数$ 或者 $负数\times 负数$，因此

- 如果 $grid[i][j] > 0$：
  - 最大积 = $\max(dp\_max[i-1][j], dp\_max[i][j-1]) \times grid[i][j]$
  - 最小积 = $\min(dp\_min[i-1][j], dp\_min[i][j-1]) \times grid[i][j]$
- 如果 $grid[i][j] < 0$：
  - 最大积 = $\min(dp\_min[i-1][j], dp\_min[i][j-1]) \times grid[i][j]$
  - 最小积 = $\max(dp\_max[i-1][j], dp\_max[i][j-1]) \times grid[i][j]$
- 如果 $grid[i][j] = 0$：
  - 最大积 = $0$，最小积 = $0$

综上所述，**定义** $dp[i][j][k]$ 表示到达 $(i,j)$ 位置的最大积和最小积，其中第三维大小为 $2$，分别存储 `[最大值, 最小值]`。

状态转移上面已经表述。**初始化** $grid[i][0][?]$ 第一列时，由于只能向右或向下，所以这一列的值只能从上方转移。同理第一行 $grid[0][j][?]$，只能从左方转移。

**切记**：不能在 $DP$ 过程中取模！

取模运算会破坏数值的大小关系，例如 $10$ 和 $7$ 对 $6$ 取模后变成 $4$ 和 $1$，原本 $10>7$ 变成了 $4>1$，但如果是负数参与的乘法，逻辑会彻底混乱。所以必须 **全精度计算，最后再取模**。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxProductPath(self, grid: List[List[int]]) -> int:
        m, n = len(grid), len(grid[0])
        MOD = 10**9 + 7
        
        # dp[i][j][0] 存最大值, dp[i][j][1] 存最小值
        dp = [[[0.0, 0.0] for _ in range(n)] for _ in range(m)]
        
        # 初始化起点
        dp[0][0][0] = dp[0][0][1] = grid[0][0]
        # 初始化第一列，只能向下
        for i in range(1, m):
            val = dp[i-1][0][0] * grid[i][0]
            dp[i][0][0] = dp[i][0][1] = val
        
        # 初始化第一行，只能向右
        for j in range(1, n):
            val = dp[0][j-1][0] * grid[0][j]
            dp[0][j][0] = dp[0][j][1] = val
        
        # 填充其余格子
        for i in range(1, m):
            for j in range(1, n):
                curr = grid[i][j]
                # 获取来自上方和左方的候选值
                candidates = [
                    dp[i-1][j][0] * curr, dp[i-1][j][1] * curr,
                    dp[i][j-1][0] * curr, dp[i][j-1][1] * curr
                ]
                dp[i][j][0] = max(candidates)
                dp[i][j][1] = min(candidates)
        
        # 答案
        res = dp[m-1][n-1][0]
        # 先判断是否为负数
        if res < 0:
            return -1
        return int(res % MOD)
```

```Java
// java
class Solution {
    public int maxProductPath(int[][] grid) {
        int m = grid.length, n = grid[0].length;
        long MOD = 1000000007L;
        
        // dp[i][j][0] 存最大值, dp[i][j][1] 存最小值
        // 使用 long 防止溢出
        long[][][] dp = new long[m][n][2];
        
        // 初始化起点
        dp[0][0][0] = dp[0][0][1] = grid[0][0];
        
        // 初始化第一列，只能向下
        for (int i = 1; i < m; i++) {
            long val = dp[i - 1][0][0] * grid[i][0];
            dp[i][0][0] = dp[i][0][1] = val;
        }
        
        // 初始化第一行，只能向右
        for (int j = 1; j < n; j++) {
            long val = dp[0][j - 1][0] * grid[0][j];
            dp[0][j][0] = dp[0][j][1] = val;
        }
        
        // 填充其余格子
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                long curr = grid[i][j];
                // 获取来自上方和左方的候选值
                long v1 = dp[i - 1][j][0] * curr;
                long v2 = dp[i - 1][j][1] * curr;
                long v3 = dp[i][j - 1][0] * curr;
                long v4 = dp[i][j - 1][1] * curr;
                
                dp[i][j][0] = Math.max(Math.max(v1, v2), Math.max(v3, v4));
                dp[i][j][1] = Math.min(Math.min(v1, v2), Math.min(v3, v4));
            }
        }
        
        // 答案
        long res = dp[m - 1][n - 1][0];
        // 先判断是否为负数
        if (res < 0) {
            return -1;
        }
        return (int) (res % MOD);
    }
}
```

- 时间复杂度： $O(m\times n)$，其中 $m,n$ 是矩阵 $grid$ 的行数、列数，只进行一次遍历
- 空间复杂度： $O(m\times n)$，表示数组 $dp$ 的大小

### 优化

观察状态转移方程，可以发现，计算当前 $dp[i][j]$ 时只依赖于第 $i-1$ 行 $dp[i-1][j]$ 和当前行的前一个元素 $dp[i][j-1]$。因此采用滚动数组，优化掉第一维。

再将第三维拆开，最终只需要使用两个 **一维** 数组 $dpMax$ 和 $dpMin$ 来保存当前行的状态。

对于当前格子 $(i,j)$ 来说， $dpMax[j]$ 以及 $dpMin[j]$ 是上一行没更新的同一列的值，而 $dpMax[j-1]$ 以及 $dpMin[j-1]$ 则是同一行左边列更新过的值。它们 **四个数字** 就是上方和左方的候选值，对应到上面的转移方程。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxProductPath(self, grid: List[List[int]]) -> int:
        m, n = len(grid), len(grid[0])
        MOD = 10**9 + 7
        
        # 维护当前行的最大和最小积
        dp_max = [0.0] * n
        dp_min = [0.0] * n
        
        # 初始化起点
        dp_max[0] = dp_min[0] = grid[0][0]
        # 初始化第一行，只能向右
        for j in range(1, n):
            dp_max[j] = dp_min[j] = dp_max[j-1] * grid[0][j]
        
        # 处理剩下的行
        for i in range(1, m):
            for j in range(n):
                curr = grid[i][j]
                if j == 0:
                    # 第一列只能从上方下来
                    dp_max[0] = dp_min[0] = dp_max[0] * curr
                else:
                    # 记录更新前上一行同列的值 dp_max[j],dp_min[j]
                    # 以及当前行左边的值 dp_max[j-1]
                    v1 = dp_max[j] * curr
                    v2 = dp_min[j] * curr
                    v3 = dp_max[j-1] * curr
                    v4 = dp_min[j-1] * curr
                    
                    dp_max[j] = max(v1, v2, v3, v4)
                    dp_min[j] = min(v1, v2, v3, v4)
        
        # 先判断是否为负数
        res = dp_max[n-1]
        return int(res % MOD) if res >= 0 else -1
```

```Java
// java
class Solution {
    public int maxProductPath(int[][] grid) {
        int m = grid.length, n = grid[0].length;
        long MOD = 1000000007L;
        
        // 维护当前行的最大和最小积
        long[] dp_max = new long[n];
        long[] dp_min = new long[n];
        
        // 初始化起点
        dp_max[0] = dp_min[0] = grid[0][0];
        
        // 初始化第一行，只能向右
        for (int j = 1; j < n; j++) {
            dp_max[j] = dp_min[j] = dp_max[j - 1] * grid[0][j];
        }
        
        // 处理剩下的行
        for (int i = 1; i < m; i++) {
            for (int j = 0; j < n; j++) {
                long curr = grid[i][j];
                if (j == 0) {
                    // 第一列只能从上方下来
                    dp_max[0] = dp_min[0] = dp_max[0] * curr;
                } else {
                    // 记录更新前上一行同列的值 dp_max[j],dp_min[j]
                    // 以及当前行左边的值 dp_max[j-1]
                    long v1 = dp_max[j] * curr;
                    long v2 = dp_min[j] * curr;
                    long v3 = dp_max[j - 1] * curr;
                    long v4 = dp_min[j - 1] * curr;
                    
                    dp_max[j] = Math.max(Math.max(v1, v2), Math.max(v3, v4));
                    dp_min[j] = Math.min(Math.min(v1, v2), Math.min(v3, v4));
                }
            }
        }
        
        // 先判断是否为负数
        long res = dp_max[n - 1];
        return res >= 0 ? (int) (res % MOD) : -1;
    }
}
```

- 时间复杂度： $O(m\times n)$，一次遍历
- 空间复杂度： $O(n)$，表示两个 $dp$ 数组的大小
