[Problem: 1855. 下标对中的最大距离](https://leetcode.cn/problems/maximum-distance-between-a-pair-of-values/description/)

### 方法：二分 & 双指针

题意比较简单，这里不再赘述。

一个容易想到的贪心：如果 $nums2$ 中所有元素都大于 $nums1[0]$。为了让距离最大，肯定是使用最右侧的 $nums2$ 的元素。

**核心目标**：对于 $nums1$ 中的每一个 $i$，我们需要在 $nums2$ 中找到 **最右边** 的一个 $j$，使得 $nums2[j]\ge nums1[i]$。

由于 $nums2$ 是非递增的，意味着满足条件 $nums2[j]\ge nums1[i]$ 的 $j$ 一定集中在数组的左侧 —— **单调性**，因此联想到 “二分查找”。

具体步骤为：

遍历数组 $nums1$，当前的数值为 $target = nums1[i]$。

要找的 $j$ 必须满足 $j\ge i$。因此，在 $nums2$ 的下标区间 $[i, len(nums2) - 1]$ 中进行搜索。

- 由于 $nums2$ 是降序（相等）的，如果 $nums2[mid]\ge target$，说明从 $i$ 到 $mid$ 的所有位置都满足条件。但目标是 **最大距离**，所以应该保留当前 $mid$ 作为候选答案，并去 **右半部分** $[mid + 1, right]$ 继续找更大的 $j$。
- 如果 $nums2[mid] < target$，说明 $mid$ 及其右侧的所有位置都太小了，不满足条件。需要去 **左半部分** $[left, mid - 1]$ 寻找满足条件的 $j$。

边界情况：如果 $i$ 已经超过了 $nums2$ 的长度，或者 $nums2[i] < nums1[i]$，则该 $i$ 无法形成有效对。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxDistance(self, nums1: List[int], nums2: List[int]) -> int:
        n, m = len(nums1), len(nums2)
        ans = 0
        
        for i in range(n):
            # 边界
            if i >= m:
                break
                
            # 在 nums2 中找到最后一个 >= nums1[i] 的位置
            target = nums1[i]
            left = i
            right = m - 1
            # 当前 i 对应的最远有效 j
            best_j = -1
            
            while left <= right:
                mid = (left + right) // 2
                
                if nums2[mid] >= target:
                    # 满足
                    best_j = mid
                    # 尝试向右移动，寻找更大的 j
                    left = mid + 1
                else:
                    # 不满足，往左找
                    right = mid - 1
            
            # 找到，计算距离
            if best_j != -1:
                ans = max(ans, best_j - i)
        
        return ans
```

```Java
// java
class Solution {
    public int maxDistance(int[] nums1, int[] nums2) {
        int n = nums1.length, m = nums2.length;
        int ans = 0;
        
        for (int i = 0; i < n; i++) {
            // 边界
            if (i >= m) {
                break;
            }
                
            // 在 nums2 中找到最后一个 >= nums1[i] 的位置
            int target = nums1[i];
            int left = i;
            int right = m - 1;
            // 当前 i 对应的最远有效 j
            int best_j = -1;
            
            while (left <= right) {
                int mid = left + (right - left) / 2;
                
                if (nums2[mid] >= target) {
                    // 满足
                    best_j = mid;
                    // 尝试向右移动，寻找更大的 j
                    left = mid + 1;
                } else {
                    // 不满足，往左找
                    right = mid - 1;
                }
            }
            
            // 找到，计算距离
            if (best_j != -1) {
                ans = Math.max(ans, best_j - i);
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(n\times logm)$，其中 $n,m$ 是数组 $nums1,nums2$ 的长度，对每个 $nums1[i]$ 处理，在 $nums2$ 中二分
- 空间复杂度： $O(1)$，仅使用常数个额外变量

### 双指针

在上面的方法中，只利用了 $nums2$ 有序，没有利用 $nums1$ 有序这个条件。

举个例子，假如 $nums1[2]=2$，它对应的有效元素是 $nums2[4]=4$。数组 $nums1$ 非递增，一定有 $nums1[3]\le nums1[2]$，已知 $nums1[2]<nums2[4]$，所以这个新元素 $nums1[3]$ 对应的有效元素，在 $nums2$ 中，下标 **一定** 大于等于 $4$！

使用 **两个指针** $i$ 和 $j$ 分别指向 $nums1$ 和 $nums2$。

初始化 $i = 0$, $j = 0$, $ans = 0$。比较 $nums1[i]$ 和 $nums2[j]$：

- 如果 $nums1[i]\le nums2[j]$：
  - 这是一个 “有效下标对”（虽然还需要满足 $i \le j$，但如果 $i > j$，距离是负数，不会更新 $ans$）。
  - 因为要找最大的 $j-i$，且此时条件满足，我们可以尝试让 $j$ **向右移动**，看看能不能找到更大的距离。
  - 更新 $ans = max(ans, j - i)$。
- 如果 $nums1[i] > nums2[j]$：
  - 当前的 $nums2[j]$ 太小了，无法满足条件。由于 $nums2$ 是非递增的，往后移 $j$ 只会让值更小，更不可能满足 $nums1[i]\le nums2[j]$。
  - 因此，我们必须让 $i$ **向右移动**，使得 $nums1[i]$ 变小，从而可能再次满足条件。

代码如下，已附加注释：

```Python
# python
class Solution:
    def maxDistance(self, nums1: List[int], nums2: List[int]) -> int:
        n, m = len(nums1), len(nums2)
        i = 0
        j = 0
        ans = 0
        
        while i < n and j < m:
            # 满足
            if nums1[i] <= nums2[j]:
                ans = max(ans, j - i)
                # 增大 j 以获得更大的距离
                j += 1
            else:
                # 需要增大 i 来减小 nums1[i] 的值
                i += 1
        
        return ans
```

```Java
// java
class Solution {
    public int maxDistance(int[] nums1, int[] nums2) {
        int n = nums1.length, m = nums2.length;
        int i = 0;
        int j = 0;
        int ans = 0;
        
        while (i < n && j < m) {
            // 满足
            if (nums1[i] <= nums2[j]) {
                ans = Math.max(ans, j - i);
                // 增大 j 以获得更大的距离
                j += 1;
            } else {
                // 需要增大 i 来减小 nums1[i] 的值
                i += 1;
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(n+m)$，两个指针最多各自移动一遍
- 空间复杂度： $O(1)$
