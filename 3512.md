[Problem: 3512. 使数组和能被 K 整除的最少操作次数](https://leetcode.cn/problems/minimum-operations-to-make-array-sum-divisible-by-k/description/)

### 方法：模拟

什么数字能被 $k$ 整除？形如 $0,k,2k,\dots$ 形式，它们都是 $k$ 的倍数，也即 $\%k=0$。

题目要求判断数组总和，因此我们先求和，然后对 $k$ 取余，余数是多少，就需要 $-1$ 操作多少次。

代码如下，使用一行流的写法：

```Python
# python
class Solution:
    def minOperations(self, nums: List[int], k: int) -> int:
        return sum(nums) % k
```

```Java
// java
class Solution {
    public int minOperations(int[] nums, int k) {
        return Arrays.stream(nums).sum() % k;
    }
}
```

- 时间复杂度： $O(n)$，其中 $n$ 是数组 $nums$ 的长度
- 空间复杂度： $O(1)$，仅使用常数个额外变量
