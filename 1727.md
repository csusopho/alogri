[Problem: 1727. 重新排列后的最大子矩阵](https://leetcode.cn/problems/largest-submatrix-with-rearrangements/description/)

### 方法：贪心 & 优化

任意排序列，尽量让 $1$ 连续。解决这类 “最大矩形” 问题的核心思路，通常是 **将矩阵转化为“直方图”**，并利用 “贪心” 来处理。

> $ps$：因为能任意交换，完全可以参考 [84](https://leetcode.cn/problems/largest-rectangle-in-histogram/description/) 这一题，看一眼图片就能恍然大悟。

无论哪种解法，第一步都是相同的：**计算每个位置上方连续 1 的高度**。

假设我们固定子矩阵的 **底边** 在第 $i$ 行。对于第 $i$ 行的第 $j$ 列，它能向上延伸出的最大高度 $h[i][j]$ 取决于：

- 如果 $matrix[i][j] = 0$，则高度为 $0$。
- 如果 $matrix[i][j] = 1$，则高度为 $h[i-1][j] + 1$。

把矩阵想象成一个建筑群。如果我们站在第 $i$ 行往上看，每一列都是一根垂直向上的 “1” 组成的柱子。因为 **列可以随意交换**，为了让矩形面积最大，显然应该把 **最高的柱子都挪到一起**。

预处理高度：遍历矩阵，计算每个位置 $(i, j)$ 向上连续 $1$ 的数量。

对于每一行 $i$，得到了一个高度数组，例如 $[2, 0, 3, 1]$。**贪心策略**：由于可以重新排列列，将这一行的高度 **从大到小排序**，排序后得到 $[3, 2, 1, 0]$。

**计算面积**：

- 宽度为 $1$ 时，最大高度是 $3$，面积 = $3 \times 1 = 3$
- 宽度为 $2$ 时，为了保证是矩形，高度受限于较矮的那根，即高度为 $2$，面积 = $2 \times 2 = 4$
- 宽度为 $3$ 时，高度受限于 $1$，面积 = $1 \times 3 = 3$

**原地** 修改，将矩阵 $matrix$ 当作 $height$ 数组处理。

代码如下，已附加注释：

```Python
# python
class Solution:
    def largestSubmatrix(self, matrix: List[List[int]]) -> int:
        m, n = len(matrix), len(matrix[0])
        ans = 0
        
        # 预处理高度，原地修改
        for i in range(1, m):
            for j in range(n):
                if matrix[i][j] == 1:
                    matrix[i][j] += matrix[i-1][j]
        
        # 逐行排序并计算
        for i in range(m):
            # 拷贝当前行并降序排列
            cur_row = sorted(matrix[i], reverse=True)
            for j in range(n):
                if cur_row[j] == 0:
                    break
                # 高度是 cur_row[j], 宽度是 j + 1
                ans = max(ans, cur_row[j] * (j + 1))
        
        return ans
```

```Java
// java
class Solution {
    public int largestSubmatrix(int[][] matrix) {
        int m = matrix.length, n = matrix[0].length;
        int ans = 0;
        
        // 预处理高度，原地修改
        for (int i = 1; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (matrix[i][j] == 1) {
                    matrix[i][j] += matrix[i - 1][j];
                }
            }
        }
        
        // 逐行排序并计算
        for (int i = 0; i < m; i++) {
            // 拷贝当前行并降序排列
            int[] curRow = matrix[i].clone();
            Arrays.sort(curRow);
            for (int j = 0; j < n; j++) {
                int h = curRow[n - 1 - j];
                if (h == 0) {
                    break;
                }
                // 高度是 cur_row[j], 宽度是 j + 1
                ans = Math.max(ans, h * (j + 1));
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m\times n\times logn)$，其中 $m,n$ 是矩阵 $matrix$ 的行数、列数
- 空间复杂度： $O(n)$，表示中间变量 $curRow$ 的大小，忽略排序的栈开销 $O(logn)$

上面这种原地解法可能稍难理解，可以用独立的高度数组 $heights$ 进行记录。同时，可以使用 $>$ 比较代替 $max$ 比较，速度更快。

下面的 $Python$ 代码耗时 $111ms$，超过 $100\%$，已附加注释：

```Python
# python
class Solution:
    def largestSubmatrix(self, matrix: List[List[int]]) -> int:
        m, n = len(matrix), len(matrix[0])
        # 独立的高度数组
        heights = [0] * n
        ans = 0
        
        for i in range(m):
            # 更新高度
            for j in range(n):
                if matrix[i][j] == 1:
                    heights[j] += 1
                else:
                    heights[j] = 0
            
            # 拷贝并降序
            sorted_heights = sorted(heights, reverse=True)
            
            # 计算以当前行为底边的最大矩形面积
            for k in range(n):
                h = sorted_heights[k]
                if h == 0: 
                    break
                
                # 宽度是 k + 1
                cur_area = h * (k + 1)
                if cur_area > ans:
                    ans = cur_area
                    
        return ans
```

### 优化

在上述解法中，每行都进行了一次全排序。但观察发现：**第 $i$ 行的高度是基于第 $i-1$ 行变化的**，增量只能是 $+1$ 或者变成 $0$。

因此，可以不使用 $sort$，而是手动维护一个 **递增的高度列表**。

如何维护升序？将高度为 $0$ 的列放前面，高度递增的列放后面。上一行已排序的索引，在当前行更新后依然保持相对有序。

举个例子，假设上一行高度是 $[0, 2, 5]$。

- 如果这一行全是 $1$，高度变为 $[1, 3, 6]$，相对顺序不变。
- 如果某些位置变成 $0$，将它们移到最左边的 $inactive$，剩下的 $1$ 依然保持原来的相对大小关系 $active$。
- 通过 $inactive + active$ 的拼接，保证依旧是升序。

如何计算面积？由于 $sortedIdx$ **升序**，当我们遍历到 $i$ 时，第 $i$ 列到第 $n-1$ 列的所有高度都 **大于或等于** 当前高度 $heights[sortedIdx[i]]$。因此，以当前高度为准，能够构成的矩形宽度就是这一列及其右侧所有的列数，即 $n - i$。

这种解法速度更快，经实测 $Python$ 耗时 $107ms$，同样超过 $100\%$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def largestSubmatrix(self, matrix: List[List[int]]) -> int:
        n = len(matrix[0])
        heights = [0] * n
        # 存储列下标，并始终保持其对应的 heights 为升序排列
        sorted_indices = list(range(n))
        ans = 0
        
        for row in matrix:
            active_cols = []   # 存储当前行为 1 的列下标
            inactive_cols = [] # 存储当前行为 0 的列下标

            # 顺序遍历，保证 active_cols 内部依然高度升序
            for j in sorted_indices:
                if row[j] == 1:
                    heights[j] += 1
                    active_cols.append(j)
                else:
                    heights[j] = 0
                    inactive_cols.append(j)
            
            # 将高度为 0 的列放前面，高度递增的列放后面
            sorted_indices = inactive_cols + active_cols
            
            # 计算最大矩形面积
            for i in range(len(inactive_cols), n):
                height = heights[sorted_indices[i]]
                width = n - i
                cur_area = height * width
                if cur_area > ans:
                    ans = cur_area
            
        return ans
```

```Java
// java
class Solution {
    public int largestSubmatrix(int[][] matrix) {
        int n = matrix[0].length;
        int[] heights = new int[n];
        // 存储列下标，并始终保持其对应的 heights 为升序排列
        int[] sortedIndices = new int[n];
        for (int i = 0; i < n; i++) {
            sortedIndices[i] = i;
        }
        int ans = 0;
        
        for (int[] row : matrix) {
            int[] activeCols = new int[n];   // 存储当前行为 1 的列下标
            int[] inactiveCols = new int[n]; // 存储当前行为 0 的列下标
            int aPtr = 0, iPtr = 0;

            // 顺序遍历，保证 active_cols 内部依然高度升序
            for (int j : sortedIndices) {
                if (row[j] == 1) {
                    heights[j] += 1;
                    activeCols[aPtr++] = j;
                } else {
                    heights[j] = 0;
                    inactiveCols[iPtr++] = j;
                }
            }
            
            // 将高度为 0 的列放前面，高度递增的列放后面
            System.arraycopy(inactiveCols, 0, sortedIndices, 0, iPtr);
            System.arraycopy(activeCols, 0, sortedIndices, iPtr, aPtr);
            
            // 计算最大矩形面积
            for (int i = iPtr; i < n; i++) {
                int height = heights[sortedIndices[i]];
                int width = n - i;
                int curArea = height * width;
                if (curArea > ans) {
                    ans = curArea;
                }
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m\times n)$，内层循环都是线性时间
- 空间复杂度： $O(n)$
