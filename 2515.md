[Problem: 2515. 到目标字符串的最短距离](https://leetcode.cn/problems/shortest-distance-to-target-string-in-a-circular-array/description/)

### 方法：模拟

从 $startIndex$ 到 $target$ 的最短距离，有两种移动方向：**向右** 和 **向左**。

假设目标字符串 $target$ 所在的索引为 $i$。

- **绝对距离（不考虑环形）**：两个下标之间的直接差值是 $abs(i - startIndex)$。
- **环形距离（跨越首尾）**：数组是环形的，可以从一端走出去，从另一端绕回来。环形总长度是 $n$，那么另一条路径的距离就是 $n - abs(i - startIndex)$。

因此，我们只需要一次遍历，当前单词 $words[i]$ 等于 $target$ 时，就用上面的公式计算并更新答案。

由于本题的 $words$ 中可能存在多个 $target$，所以找到一个 $target$ 后不能立刻结束。

代码如下，已附加注释：

```Python
# python
class Solution:
    def closestTarget(self, words: List[str], target: str, startIndex: int) -> int:
        n = len(words)
        ans = n 
        found = False
        
        for i in range(n):
            if words[i] == target:
                found = True
                # 直接距离
                d1 = abs(i - startIndex)
                # 环形距离
                d2 = n - d1
                
                # 较小值
                cur_dist = min(d1, d2)
                if cur_dist < ans:
                    ans = cur_dist
        
        return ans if found else -1
```

```Java
// java
class Solution {
    public int closestTarget(String[] words, String target, int startIndex) {
        int n = words.length;
        int ans = n;
        boolean found = false;
        
        for (int i = 0; i < n; i++) {
            if (words[i].equals(target)) {
                found = true;
                // 直接距离
                int d1 = Math.abs(i - startIndex);
                // 环形距离
                int d2 = n - d1;
                
                // 较小值
                int cur_dist = Math.min(d1, d2);
                if (cur_dist < ans) {
                    ans = cur_dist;
                }
            }
        }
        
        return found ? ans : -1;
    }
}
```

- 时间复杂度： $O(n)$，其中 $n$ 是数组 $words$ 的长度
- 空间复杂度： $O(1)$，仅使用常数个额外变量
