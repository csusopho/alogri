[Problem: 1582. 二进制矩阵中的特殊位置](https://leetcode.cn/problems/special-positions-in-a-binary-matrix/description/)

### 方法：模拟 & 预处理

当前格子是 $1$，且本行本列没有其他 $1$。

直接 **模拟** 题意，遍历矩阵的每一行：

- 对于当前遍历到的第 $i$ 行，统计这一行中 $1$ 的数量。
- 如果 $1$ 的数量 **不等于 1**，说明这行不可能存在 “特殊位置”，直接跳过，看下一行。
- 如果 $1$ 的数量 **恰好等于 1**，记下这个 $1$ 所在的列索引 $colIdx$。
- **按需验证**：既然第 $i$ 行已经满足条件，此时再去验证第 $colIdx$ 列。遍历这一列，统计这一列中 $1$ 的个数。
- 如果发现这一列的 $1$ 也不止一个，说明它不满足条件，直接跳出循环。
- 如果这一列的 $1$ 也 **恰好只有 1 个**，那么位置 $(i, colIdx)$ 就是 “特殊位置”，答案加一。

代码如下，已附加注释：

```Python
# python
class Solution:
    def numSpecial(self, mat: List[List[int]]) -> int:
        m = len(mat)
        n = len(mat[0])
        ans = 0
        
        # 逐行扫描
        for i in range(m):
            ones_in_row = 0
            col_idx = -1
            
            # 统计当前行中 1 的数量
            for j in range(n):
                if mat[i][j] == 1:
                    ones_in_row += 1
                    col_idx = j
            
            # 恰好只有 1 个 1
            if ones_in_row == 1:
                ones_in_col = 0
                
                # 验证第 col_idx 列中 1 的数量
                for k in range(m):
                    if mat[k][col_idx] == 1:
                        ones_in_col += 1
                        # 剪枝
                        if ones_in_col > 1:
                            break
                
                # 该列也恰好只有一个 1
                if ones_in_col == 1:
                    ans += 1
        
        return ans
```

```Java
// java
class Solution {
    public int numSpecial(int[][] mat) {
        int m = mat.length;
        int n = mat[0].length;
        int ans = 0;
        
        // 逐行扫描
        for (int i = 0; i < m; i++) {
            int ones_in_row = 0;
            int col_idx = -1;
            
            // 统计当前行中 1 的数量
            for (int j = 0; j < n; j++) {
                if (mat[i][j] == 1) {
                    ones_in_row++;
                    col_idx = j;
                }
            }
            
            // 恰好只有 1 个 1
            if (ones_in_row == 1) {
                int ones_in_col = 0;
                
                // 验证第 col_idx 列中 1 的数量
                for (int k = 0; k < m; k++) {
                    if (mat[k][col_idx] == 1) {
                        ones_in_col++;
                        // 剪枝
                        if (ones_in_col > 1) {
                            break;
                        }
                    }
                }
                
                // 该列也恰好只有一个 1
                if (ones_in_col == 1) {
                    ans++;
                }
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m\times n+m^2)$，其中 $m,n$ 是矩阵 $mat$ 的行数和列数
- 空间复杂度： $O(1)$，仅使用常数个额外变量

### 预处理

上述步骤有些繁琐，为了方便计数，我们可以 **提前** 把每一行和每一列的 $1$ 的个数统计出来。

- 开辟一个一维数组 $rowSum$，记录每一行 $1$ 的个数。
- 开辟一个一维数组 $colSum$，记录每一列 $1$ 的个数。

在预处理完成后，遍历一次矩阵，对于任意一个位置 $(i, j)$，只要同时满足：

1. $mat[i][j] = 1$
2. $rowSum[i] = 1$
3. $colSum[j] = 1$

那么它就是一个特殊位置，答案加一。

代码如下，已附加注释：

```Python
class Solution:
    def numSpecial(self, mat: List[List[int]]) -> int:
        m, n = len(mat), len(mat[0])
        ans = 0

        # 统计每一行的 1 的个数
        row_sum = [sum(row) for row in mat]
        # 统计每一列的 1 的个数
        col_sum = [sum(col) for col in zip(*mat)]
        
        for i in range(m):
            for j in range(n):
                # 如果当前元素为 1，且它所在的行和列的 1 的总数都为 1
                if mat[i][j] == 1 and row_sum[i] == 1 and col_sum[j] == 1:
                    ans += 1
          
        return ans
```

```Java
// java
class Solution {
    public int numSpecial(int[][] mat) {
        int m = mat.length, n = mat[0].length;
        int ans = 0;

        // 统计每一行的 1 的个数
        int[] row_sum = new int[m];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                row_sum[i] += mat[i][j];
            }
        }

        // 统计每一列的 1 的个数
        int[] col_sum = new int[n];
        for (int j = 0; j < n; j++) {
            for (int i = 0; i < m; i++) {
                col_sum[j] += mat[i][j];
            }
        }
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                // 如果当前元素为 1，且它所在的行和列的 1 的总数都为 1
                if (mat[i][j] == 1 && row_sum[i] == 1 && col_sum[j] == 1) {
                    ans++;
                }
            }
        }
          
        return ans;
    }
}
```

- 时间复杂度： $O(m\times n)$，每个格子遍历三次
- 空间复杂度： $O(m+n)$，表示数组 $rowSum$ 和 $colSum$ 的大小
