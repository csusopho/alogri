[Problem: 2751. 机器人碰撞](https://leetcode.cn/problems/robot-collisions/description/)

### 方法：栈

什么时候会发生碰撞？只有当一个向右移动的机器人 $R$ 位于一个向左移动的机器人 $L$ 的左侧时，它们才可能发生碰撞。

- 如果两个机器人同向，永不碰撞。
- 如果左边是 $L$，右边是 $R$，它们背向而驰，永不碰撞。

也就是说，为了正确模拟碰撞，我们需要按照机器人的 **物理位置** 从左到右进行处理。

输入给出的 $positions$ 是乱序的。最好把位置、健康度、方向和 **原始索引** 打包在一起，放入 $robots$ 列表，这样方便处理。然后根据 $position$ 从小到大排序。

如何线性模拟这种一维的移动？使用 **栈**，存储所有 **向右** 的机器人。

遍历排序后的机器人，对于当前第 $i$ 个机器人来说：

- 如果是 $R$：它可能会撞到右边后续出现的 $L$，所以把它入栈。
- 如果是 $L$：
  - 栈为空，说明它左边没有向右走的机器人，它安全了。
  - 栈不为空，栈顶一定是距离它最近且向右走的机器人。这时发生碰撞。比较两者的健康度。根据生存规则更新健康度，并决定是 “继续撞向栈里的下一个机器人” 还是 “自己消失”。
- 当 $while$ 循环结束时，这个一直向左的机器人要么已经 “消失”，要么将栈内的向右机器人全部碰完并存活。如果仍然存活，就需要单独将它进行记录。要么使用额外列表，要么复用 $robots$。

碰撞结束后，剩下的机器人只有两种可能：

- 一直顶到最左侧的存活的 $L$
- 在栈内仍然向右的 $R$

根据它们的原始索引排序，最后只输出健康度。

代码如下，已附加注释：

```Python
# python
class Solution:
    def survivedRobotsHealths(self, positions: List[int], healths: List[int], directions: str) -> List[int]:
        # [位置, 健康度, 方向, 原始索引]
        n = len(positions)
        robots = []
        for i in range(n):
            robots.append([positions[i], healths[i], directions[i], i])
        
        # 按照位置从小到大排序
        robots.sort()
        # 存放向右移动的机器人信息
        stack = []
        
        # 遍历排序后的机器人
        for i in range(len(robots)):
            curr = robots[i]
            # 向右走，直接入栈
            if curr[2] == 'R':
                stack.append(curr)
            else:
                # 向左走，与栈顶向右走的机器人碰撞
                while stack and curr[1] > 0:
                    top = stack[-1]
                    
                    if top[1] < curr[1]:
                        # 栈顶消失
                        stack.pop()
                        curr[1] -= 1
                    elif top[1] > curr[1]:
                        # 栈顶健康度减 1
                        top[1] -= 1
                        curr[1] = 0
                    else:
                        # 两者都消失
                        stack.pop()
                        curr[1] = 0
                
                # 向左走的机器人依然存活
                if curr[1] > 0:
                    # 借用 robots 列表本身存储
                    robots[i] = curr
                else:
                    robots[i] = None # 标记为死亡
        
        survivors = []
        # 栈里剩下的都是向右走且没被撞掉的
        for r in stack:
            survivors.append(r)
        # 找到存活的向左走的机器人
        for i in range(len(robots)):
            if robots[i] is not None and robots[i][2] == 'L':
                survivors.append(robots[i])
        
        # 按照原始索引排序
        survivors.sort(key=lambda x: x[3])
        return [r[1] for r in survivors]
```

```Java
// java
class Solution {
    public List<Integer> survivedRobotsHealths(int[] positions, int[] healths, String directions) {
        // [位置, 健康度, 方向, 原始索引]
        int n = positions.length;
        List<int[]> robots = new ArrayList<>();
        for (int i = 0; i < n; i++) {
            robots.add(new int[]{positions[i], healths[i], directions.charAt(i), i});
        }

        // 按照位置从小到大排序
        robots.sort((a, b) -> Integer.compare(a[0], b[0]));
        // 存放向右移动的机器人信息
        List<int[]> stack = new ArrayList<>();

        // 遍历排序后的机器人
        for (int i = 0; i < robots.size(); i++) {
            int[] curr = robots.get(i);
            // 向右走，直接入栈
            if (curr[2] == 'R') {
                stack.add(curr);
            } else {
                // 向左走，与栈顶向右走的机器人碰撞
                while (!stack.isEmpty() && curr[1] > 0) {
                    int[] top = stack.get(stack.size() - 1);

                    if (top[1] < curr[1]) {
                        // 栈顶消失
                        stack.remove(stack.size() - 1);
                        curr[1] -= 1;
                    } else if (top[1] > curr[1]) {
                        // 栈顶健康度减 1
                        top[1] -= 1;
                        curr[1] = 0;
                    } else {
                        // 两者都消失
                        stack.remove(stack.size() - 1);
                        curr[1] = 0;
                    }
                }

                // 向左走的机器人依然存活
                if (curr[1] > 0) {
                    // 借用 robots 列表本身存储
                    robots.set(i, curr);
                } else {
                    robots.set(i, null); // 标记为死亡
                }
            }
        }

        List<int[]> survivors = new ArrayList<>();
        // 栈里剩下的都是向右走且没被撞掉的
        for (int[] r : stack) {
            survivors.add(r);
        }
        // 找到存活的向左走的机器人
        for (int i = 0; i < robots.size(); i++) {
            if (robots.get(i) != null && robots.get(i)[2] == 'L') {
                survivors.add(robots.get(i));
            }
        }

        // 按照原始索引排序
        survivors.sort((a, b) -> Integer.compare(a[3], b[3]));
        
        List<Integer> result = new ArrayList<>();
        for (int[] r : survivors) {
            result.add(r[1]);
        }
        return result;
    }
}
```

- 时间复杂度： $O(n\times logn)$，其中 $n$ 是数组 $positions$ 的长度，初始&结束都是快排，每个机器人最多入/出栈一次
- 空间复杂度： $O(n)$，存储机器人信息的列表 $robots$ 和栈空间 $stack$
