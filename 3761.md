[Problem: 3761. 镜像对之间最小绝对距离](https://leetcode.cn/problems/minimum-absolute-distance-between-mirror-pairs/description/)

### 方法：哈希

本题有两个容易混淆的点：

- 找的是 $reverse(nums[i]) = nums[j]$。遍历到索引 $j$ 时，我们需要知道 **之前** 是否存在一个索引 $i$，其数字反转后的值正好等于当前的 $nums[j]$。**只有** 前面元素才能反转！
- 数字 $A$ 反转能变为 $B$，**不一定** $B$ 反转后变为 $A$！比如数字 $120\to 21$，但是 $21\to 12$。

根据题目要求，反转需要忽略前导零。如何反转？

- 转为字符串，反转，再转回整数。
- 从低位到高位，先 $\times 10$ 再累加最低位，不过极易溢出。

这里 $Python$ 选择最简便的第一种做法，而 $Java$ 采用第二种。

**核心**：为了使距离 $|i - j|$ 最小，对于同一个目标值，只需要记录它 **最近一次** 出现的索引。

使用 **哈希表 $Map$** 来存储前面记录过的 “期望的值” 及其 “对应的索引”。也即，哈希键是 $reverse(nums[i])$，而哈希值是索引 $i$。

一次遍历，对于每个元素 $nums[j]$：

- **检查：** 如果 $nums[j]$ 在 $map$ 字典中，说明找到了一个镜像对。
  - 计算距离 $dist = j - map[nums[j]]$。
  - 更新最小距离：$ans = min(ans, dist)$。
- **记录：** 计算当前数字的反转值 $target = reverse(nums[j])$。
- **更新：** 将 $target$ 作为 $Key$，当前索引 $j$ 作为 $Value$ 存入字典。如果 $Key$ 已存在，直接覆盖。

在上面的步骤中，最关键的是 **贪心策略**：在字典中遇到重复的 $Key$ 时覆盖旧索引。因为我们要求的是 **最小** 绝对距离，对于后面的 $j$ 来说，越靠右的 $i$ 产生的距离一定越小。

总结思路：枚举右侧当前数字，将左侧已反转数字存入哈希，从而 $O(1)$ 查找是否有对应。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minMirrorPairDistance(self, nums: List[int]) -> int:
        ans = float('inf')
        # 记录 reverse(nums[i])，对应最近的索引 i
        last_seen_reverse = {}
        
        for j in range(len(nums)):
            cur_val = nums[j]
            
            # 检查
            if cur_val in last_seen_reverse:
                # 相同且最近
                dist = j - last_seen_reverse[cur_val]
                if dist < ans:
                    ans = dist
            
            # 反转
            reversed_val = int(str(cur_val)[::-1])
            # 存入
            last_seen_reverse[reversed_val] = j
        
        return ans if ans != float('inf') else -1
```

```Java
// java
class Solution {
    public int minMirrorPairDistance(int[] nums) {
        int ans = Integer.MAX_VALUE;
        // 记录 reverse(nums[i])，对应最近的索引 i
        Map<Integer, Integer> lastSeenReverse = new HashMap<>();

        for (int j = 0; j < nums.length; j++) {
            int curVal = nums[j];

            // 检查
            if (lastSeenReverse.containsKey(curVal)) {
                // 相同且最近
                int dist = j - lastSeenReverse.get(curVal);
                if (dist < ans) {
                    ans = dist;
                }
            }

            // 反转
            int reversedVal = reverse(curVal);
            // 存入
            lastSeenReverse.put(reversedVal, j);
        }

        return ans == Integer.MAX_VALUE ? -1 : ans;
    }

    private int reverse(int n) {
        int res = 0;
        while (n > 0) {
            res = res * 10 + n % 10;
            n /= 10;
        }
        return res;
    }
}
```

- 时间复杂度： $O(n\times L)$，其中 $n$ 是数组 $nums$ 的长度，而 $L$ 是数字的最大位数，每次变字符串+反转的时间为 $O(L)$
- 空间复杂度： $O(n)$，表示哈希表 $lastSeen$ 的大小
