[Problem: 48. 旋转图像](https://leetcode.cn/problems/rotate-image/description/)

### 方法：模拟

观察旋转前后，矩阵中四个 **对称点** 的位置关系。

对于一个 $n \times n$ 的矩阵，坐标为 $(i, j)$ 的点在旋转 $90$ 度后会移动到 $(j, n-1-i)$。如果我们连续追踪四个位置：

1. 左上 $(i, j)$  $\to$ 移到右上
2. 右上 $(j, n-1-i)$ $\to$ 移到右下
3. 右下 $(n-1-i, n-1-j)$ $\to$ 移到左下
4. 左下 $(n-1-j, i)$ $\to$ 移到左上

这四个位置构成了一个循环！不过，记得使用一个临时变量，不然某个值会被覆盖。

既然如此，只需要遍历矩阵的 **四分之一**。

- 行：遍历前一半 $n // 2$
- 列：遍历 $(n + 1) // 2$

下面 $+1$ 是处理 $n$ 为奇数时中心轴的情况。

代码如下，已附加注释：

```Python
# python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        n = len(matrix)
        
        # 遍历四分之一
        for i in range(n // 2):
            for j in range((n + 1) // 2):
                # 暂存左上角
                temp = matrix[i][j]
                # 左下 -> 左上
                matrix[i][j] = matrix[n - 1 - j][i]
                # 右下 -> 左下
                matrix[n - 1 - j][i] = matrix[n - 1 - i][n - 1 - j]
                # 右上 -> 右下
                matrix[n - 1 - i][n - 1 - j] = matrix[j][n - 1 - i]
                # 暂存的左上 -> 右上
                matrix[j][n - 1 - i] = temp
```

```Java
// java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        
        // 遍历四分之一
        for (int i = 0; i < n / 2; i++) {
            for (int j = 0; j < (n + 1) / 2; j++) {
                // 暂存左上角
                int temp = matrix[i][j];
                // 左下 -> 左上
                matrix[i][j] = matrix[n - 1 - j][i];
                // 右下 -> 左下
                matrix[n - 1 - j][i] = matrix[n - 1 - i][n - 1 - j];
                // 右上 -> 右下
                matrix[n - 1 - i][n - 1 - j] = matrix[j][n - 1 - i];
                // 暂存的左上 -> 右上
                matrix[j][n - 1 - i] = temp;
            }
        }
    }
}
```

- 时间复杂度： $O(n^2)$，其中 $n$ 为矩阵 $matrix$ 的边长，一次遍历
- 空间复杂度： $O(1)$，仅使用常数个额外变量

### 翻转

进一步观察，可以发现，顺时针旋转 $90$ 度其实等价于 **两步** 简单的矩阵变换：

1. **转置**：将矩阵的行和列对换（即沿对角线翻转）。
2. **左右镜像翻转**：将每一行的数据左右颠倒。

假设 $n=3$ 的矩阵：

```text
1 2 3
4 5 6
7 8 9
```

**第一步：转置**（`matrix[i][j]` 与 `matrix[j][i]` 交换）

```text
1 4 7
2 5 8
3 6 9
```

**第二步：左右翻转**（每一行进行 $reverse$）

```text
7 4 1
8 5 2
9 6 3
```

**结果：** 刚好就是顺时针旋转 $90$ 度的结果！

> $ps$：什么，你问我怎么想到的？那还真得靠数学直觉。这里不仅可以 “转置+左右翻转”，还可以 “上下翻转+对角线翻转”。拓展一下知识，防止下次遇到类似题目。

代码如下，已附加注释：

```Python
# python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        n = len(matrix)
        
        # 转置
        for i in range(n):
            for j in range(i + 1, n):  # j 从 i+1 开始，避免重复交换变回原样
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        
        # 翻转每一行
        for i in range(n):
            matrix[i].reverse()
```

```Java
// java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        
        // 转置
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) { // j 从 i+1 开始，避免重复交换变回原样
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
        
        // 翻转每一行
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n / 2; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[i][n - 1 - j];
                matrix[i][n - 1 - j] = temp;
            }
        }
    }
}
```

- 时间复杂度： $O(n^2)$，两次遍历
- 空间复杂度： $O(1)$
