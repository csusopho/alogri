[Problem: 3643. 垂直翻转子矩阵](https://leetcode.cn/problems/flip-square-submatrix-vertically/description/)

### 方法：模拟

什么叫做 **垂直翻转**？子矩阵的第一行与最后一行交换，第二行与倒数第二行交换，依此类推。行内元素的顺序不变！

具体来说，子矩阵有 $k$ 行，要进行如下交换：

- 第 $0$ 行（相对于子矩阵）与第 $k-1$ 行交换。
- 第 $1$ 行与第 $k-2$ 行交换。

**总结规律**：第 $i$ 行与第 $k-1-i$ 行交换。

**注意**：交换只需要进行到中间 $k/2$，否则换过去又会换回来。

也就是说，遍历 $i$ 从 $0$ 到 $k/2-1$。当前需要交换的两行在原矩阵中的实际索引 $row1 = x + i$，而 $row2 = x + k - 1 - i$，交换这两行中从 $y$ 到 $y + k - 1$ 这一段的元素。

代码如下，已附加注释：

```Python
# python
class Solution:
    def reverseSubmatrix(self, grid: List[List[int]], x: int, y: int, k: int) -> List[List[int]]:
        # 只需要遍历子矩阵高度的一半
        for i in range(k // 2):
            # 确定需要交换的两行索引
            r1 = x + i
            r2 = x + k - 1 - i
            # 确定列的范围
            grid[r1][y:y+k], grid[r2][y:y+k] = grid[r2][y:y+k], grid[r1][y:y+k]
        
        return grid
```

```Java
// java
class Solution {
    public int[][] reverseSubmatrix(int[][] grid, int x, int y, int k) {
        // 只需要遍历子矩阵高度的一半
        for (int i = 0; i < k / 2; i++) {
            // 确定需要交换的两行索引
            int row1 = x + i;
            int row2 = x + k - 1 - i;
            
            // 交换这两行中从 y 到 y + k - 1 这一段的元素
            for (int j = 0; j < k; j++) {
                // 当前处理的列索引
                int curCol = y + j;
                // 经典的交换三变量法
                int temp = grid[row1][curCol];
                grid[row1][curCol] = grid[row2][curCol];
                grid[row2][curCol] = temp;
            }
        }
        
        return grid;
    }
}
```

- 时间复杂度： $O(k^2)$，访问子矩阵中约一半的元素进行交换，操作次数与子矩阵的面积成正比
- 空间复杂度： $O(1)$，原地修改
