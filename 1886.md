[Problem: 1886. 判断矩阵经轮转后是否一致](https://leetcode.cn/problems/determine-whether-matrix-can-be-obtained-by-rotation/description/)

### 方法：模拟 & 优化

由于矩阵是 $n \times n$ 的，旋转 $4$ 次 $360$ 度后会回到原状，因此我们只需要检查这 **4 种状态** 即可。也即顺时针旋转 0度、90度、180度、270度。

**最直接** 的方法：编写一个函数，实现将矩阵顺时针旋转 $90$ 度的操作。然后调用这个函数 $4$ 次，每次旋转后都与 $target$ 进行比较。

顺时针旋转 $90$ 度有什么变化？假设原矩阵坐标为 $(i, j)$，

1. **转置**：行变列，即 $(i, j) \to (j, i)$。
2. **水平翻转**：每一行左右反转，即 $(j, i) \to (j, n-1-i)$。

所以，顺时针 $90$ 度的映射规律是： $newmat[j][n-1-i] = mat[i][j]$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def findRotation(self, mat: List[List[int]], target: List[List[int]]) -> bool:
        n = len(mat)

        # 旋转 90 度的函数
        def rotate(matrix):
            # 全 0 的新矩阵
            res = [[0] * n for _ in range(n)]
            for i in range(n):
                for j in range(n):
                    # 填充
                    res[j][n - 1 - i] = matrix[i][j]
            return res
        
        # 尝试 4 次旋转
        for _ in range(4):
            if mat == target:
                return True
            mat = rotate(mat)  # 更新 mat 为旋转后的矩阵
        
        return False
```

```Java
// java
class Solution {
    public boolean findRotation(int[][] mat, int[][] target) {
        int n = mat.length;

        // 尝试 4 次旋转
        for (int k = 0; k < 4; k++) {
            if (Arrays.deepEquals(mat, target)) {
                return true;
            }
            mat = rotate(mat);  // 更新 mat 为旋转后的矩阵
        }
        
        return false;
    }

    // 旋转 90 度的函数
    private int[][] rotate(int[][] matrix) {
        int n = matrix.length;
        // 全 0 的新矩阵
        int[][] res = new int[n][n];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                // 填充
                res[j][n - 1 - i] = matrix[i][j];
            }
        }
        return res;
    }
}
```

- 时间复杂度： $O(n^2)$，其中 $n$ 是矩阵 $mat$ 的边长，遍历四次
- 空间复杂度： $O(n^2)$，表示中间变量 $res$ 的大小

### 优化

实际上，可以 **不进行物理旋转**，而是直接根据坐标变换公式，一次性检查 $4$ 种可能的旋转结果。

对于原矩阵中坐标为 $(i, j)$ 的元素：

1. 旋转 $0$ 度：对应 $target[i][j]$
2. 旋转 $90$ 度：对应 $target[j][n-1-i]$
3. 旋转 $180$ 度：对应 $target[n-1-i][n-1-j]$
4. 旋转 $270$ 度：对应 $target[n-1-j][i]$

如果某个格子旋转 $90$ 度不能匹配，就说明所有格子都不能旋转 $90$ 度！以此类推，任意角度不能匹配，这个角度就可以 **永远不考虑**！

那么，我们只需要设定 $4$ 个布尔 **标记位**，分别代表 $4$ 种角度是否匹配。

遍历一次矩阵，如果某个位置不符合某种旋转，就把对应的标记设为 $False$。最后只要有一个标记为 $True$，则返回 $True$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def findRotation(self, mat: List[List[int]], target: List[List[int]]) -> bool:
        n = len(mat)
        # 假设四个角度都是可能的
        c0, c90, c180, c270 = True, True, True, True
        
        for i in range(n):
            for j in range(n):
                val = mat[i][j]
                # 检查 0 度
                if val != target[i][j]:
                    c0 = False
                # 检查 90 度
                if val != target[j][n - 1 - i]:
                    c90 = False
                # 检查 180 度
                if val != target[n - 1 - i][n - 1 - j]:
                    c180 = False
                # 检查 270 度
                if val != target[n - 1 - j][i]:
                    c270 = False
            
            # 所有角度都不可能了，提前退出
            if not (c0 or c90 or c180 or c270):
                return False
        
        return c0 or c90 or c180 or c270
```

```Java
// java
class Solution {
    public boolean findRotation(int[][] mat, int[][] target) {
        int n = mat.length;
        // 假设四个角度都是可能的
        boolean c0 = true, c90 = true, c180 = true, c270 = true;
        
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                int val = mat[i][j];
                // 检查 0 度
                if (val != target[i][j]) {
                    c0 = false;
                }
                // 检查 90 度
                if (val != target[j][n - 1 - i]) {
                    c90 = false;
                }
                // 检查 180 度
                if (val != target[n - 1 - i][n - 1 - j]) {
                    c180 = false;
                }
                // 检查 270 度
                if (val != target[n - 1 - j][i]) {
                    c270 = false;
                }
            }
            
            // 所有角度都不可能了，提前退出
            if (!(c0 || c90 || c180 || c270)) {
                return false;
            }
        }
        
        return c0 || c90 || c180 || c270;
    }
}
```

- 时间复杂度： $O(n^2)$，一次遍历
- 空间复杂度： $O(1)$，仅使用常数个额外变量
