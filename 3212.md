[Problem: 3212. 统计 X 和 Y 频数相等的子矩阵数量](https://leetcode.cn/problems/count-submatrices-with-equal-frequency-of-x-and-y/description/)

### 方法：二维 & 一维前缀和

所有子矩阵必须包含 **左上角** $(0,0)$，因此每一个子矩阵都可以唯一地由其 **右下角** $(i, j)$ 来确定。

如何快速计算任意矩形区域的元素和？二维前缀和原理/推导，可以看这篇 [图解](https://leetcode.cn/problems/range-sum-query-2d-immutable/solutions/2993541/tu-jie-er-wei-qian-zhui-he-tui-dao-ji-su-61oc/)。

将 $(0,0)$ 为左上角 $(i,j)$ 为右下角的子矩阵和记为 $S[i][j]$，它可以通过动态规划的思想计算：

根据 **容斥原理**，要计算 $S[i][j]$，可以利用已经算出的上方和左侧的结果：

$$
S[i][j] = S[i-1][j] + S[i][j-1] - S[i-1][j-1] + \text{grid}[i][j]
$$

其中 $S[i-1][j]$ 表示上方矩形的和， $S[i][j-1]$ 表示左侧矩形的和， $S[i-1][j-1]$ 表示左上方重叠部分的和（被加了两次，所以要减去一次）。

上面是原理，对于本题来说，定义两个二维数组：

- $preX[i][j]$：从 $(0, 0)$ 到 $(i, j)$ 矩形区域内 $X$ 的总数。
- $preY[i][j]$：从 $(0, 0)$ 到 $(i, j)$ 矩形区域内 $Y$ 的总数。

对于任意一个确定的右下角 $(i, j)$，该子矩阵满足条件的 **充要条件** 是：

1. $preX[i][j] == preY[i][j]$ （数量相等）
2. $preX[i][j] > 0$ （至少包含一个 $X$）

只需要一次遍历，更新前缀和，同时判断当前子矩阵是否满足。

代码如下，已附加注释：

```Python
# python
class Solution:
    def numberOfSubmatrices(self, grid: List[List[str]]) -> int:
        n = len(grid)
        m = len(grid[0])
        ans = 0
        
        # preX[i][j] 表示以 (0,0) 为左上角，(i-1, j-1) 为右下角的矩形中 X 的数量
        preX = [[0] * (m + 1) for _ in range(n + 1)]
        preY = [[0] * (m + 1) for _ in range(n + 1)]
        
        for i in range(n):
            for j in range(m):
                # 更新前缀和
                deltaX = 1 if grid[i][j] == 'X' else 0
                deltaY = 1 if grid[i][j] == 'Y' else 0
                
                preX[i + 1][j + 1] = preX[i][j + 1] + preX[i + 1][j] - preX[i][j] + deltaX
                preY[i + 1][j + 1] = preY[i][j + 1] + preY[i + 1][j] - preY[i][j] + deltaY
                
                # 判断
                if preX[i + 1][j + 1] == preY[i + 1][j + 1] and preX[i + 1][j + 1] > 0:
                    ans += 1
        
        return ans
```

```Java
// java
class Solution {
    public int numberOfSubmatrices(char[][] grid) {
        int n = grid.length;
        int m = grid[0].length;
        int ans = 0;
        
        // preX[i][j] 表示以 (0,0) 为左上角，(i-1, j-1) 为右下角的矩形中 X 的数量
        int[][] preX = new int[n + 1][m + 1];
        int[][] preY = new int[n + 1][m + 1];
        
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                // 更新前缀和
                int deltaX = (grid[i][j] == 'X' ? 1 : 0);
                int deltaY = (grid[i][j] == 'Y' ? 1 : 0);
                
                preX[i + 1][j + 1] = preX[i][j + 1] + preX[i + 1][j] - preX[i][j] + deltaX;
                preY[i + 1][j + 1] = preY[i][j + 1] + preY[i + 1][j] - preY[i][j] + deltaY;
                
                // 判断
                if (preX[i + 1][j + 1] == preY[i + 1][j + 1] && preX[i + 1][j + 1] > 0) {
                    ans++;
                }
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m\times n)$，其中 $m,n$ 是矩阵 $grid$ 的行数、列数
- 空间复杂度： $O(m\times n)$，表示数组 $preX,preY$ 的大小

### 优化

想一想 **拼积木** 的过程，一个大矩形，不就是多个小矩形拼接的结果？

因此，我们可以将 $a\times b$ 大小的矩阵，看作 $b$ 个高度为 $a$ 的竖形矩阵拼接而来。

替换上面二维的思路，维护一个长度为 $n$ 的一维数组 $colSum$：

- $colSum[j]$ 表示在当前行 $i$ 时，第 $j$ 列从第 $0$ 行到第 $i$ 行的 **纵向** 元素之和，即 $colSum[j] = sum(grid[0...i][j])$
- 处理到第 $i$ 行时：更新 $colSum[j] += grid[i][j]$。同时，计算 $colSum$ 的 **横向累计前缀和**。设 $rowSum$ 为当前行从第 $0$ 列到第 $j$ 列的 $colSum$ 累加值。这个 $rowSum$ 实际上就是子矩阵 $(0,0)$ 到 $(i,j)$ 的总和。

也就是说，二维的累加和，换成了多个一维累加和的 **总和**。（先按列纵向累加，再按行横向累加）

对于本题来说，将二维 $preX$ 换成一维 $colX$，其中 $colX[j]$ 存储第 $j$ 列中从第 $0$ 到当前行 $i$ 为止的 $X$ 的垂直总和。同理 $colY[j]$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def numberOfSubmatrices(self, grid: List[List[str]]) -> int:
        n, m = len(grid), len(grid[0])
        # colX[j] 存储第 j 列到当前行 i 为止的 X 的垂直总和
        colX = [0] * m
        colY = [0] * m
        ans = 0
        
        for i in range(n):
            row_sum_x = 0  # 当前行从 0 列到 j 列的 X 累计
            row_sum_y = 0

            for j in range(m):
                # 更新前缀和
                if grid[i][j] == 'X':
                    colX[j] += 1
                if grid[i][j] == 'Y':
                    colY[j] += 1
                
                # 统计列和
                row_sum_x += colX[j]
                row_sum_y += colY[j]
                if row_sum_x == row_sum_y and row_sum_x > 0:
                    ans += 1
        
        return ans
```

```Java
// java
class Solution {
    public int numberOfSubmatrices(char[][] grid) {
        int n = grid.length, m = grid[0].length;
        // colX[j] 存储第 j 列到当前行 i 为止的 X 的垂直总和
        int[] colX = new int[m];
        int[] colY = new int[m];
        int ans = 0;
        
        for (int i = 0; i < n; i++) {
            int rowSumX = 0;  // 当前行从 0 列到 j 列的 X 累计
            int rowSumY = 0;

            for (int j = 0; j < m; j++) {
                // 更新前缀和
                if (grid[i][j] == 'X') {
                    colX[j]++;
                }
                if (grid[i][j] == 'Y') {
                    colY[j]++;
                }
                
                // 统计列和
                rowSumX += colX[j];
                rowSumY += colY[j];
                if (rowSumX == rowSumY && rowSumX > 0) {
                    ans++;
                }
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m\times n)$
- 空间复杂度： $O(n)$
