[Problem: 61. 旋转链表](https://leetcode.cn/problems/rotate-list/description/)

### 方法：模拟

旋转链表的本质：**把链表的后面一部分切下来，接到链表的前面去。**

假设链表长度为 $n$，我们要将每个节点向右移动 $k$ 个位置：

1. **处理 $k$ 过大的情况**：如果 $k$ 大于 $n$，由于旋转 $n$ 次链表会回到原状（不变）。因此，实际需要移动的次数是 `k % n`。
2. **找到断开点**：旋转后，原链表的倒数第 $k$ 个节点将变成新链表的头节点。

因此，得到本题的思路：

- 先将链表连成一个 **环**（尾节点指向头节点）。
- 从原头节点出发，走 `n - (k % n) - 1` 步，找到新链表的 **尾节点**。
- 记录新尾节点的下一个节点为 **新头节点**，然后断开环。

---

以 $head = [1, 2, 3, 4, 5], k = 2$ 为例：

链表长度 $n=5$，计算 `k % n = 2 % 5 = 2`。

将末尾节点 $5$ 的 $next$ 指向节点 $1$。此时链表结构：`1 -> 2 -> 3 -> 4 -> 5 -> (回到1)`。

旋转 $k$ 个位置，意味着新的头节点是倒数第 $k$ 个。那么，新尾节点就是从原头节点开始算的第 $n - k$ 个节点。也即，从 $head$ 出发，走 $n - k - 1$ 步。

在这个例子中，得出 $n - k - 1 = 2$ 步。从 $1$ 出发走 $2$ 步，指针停在节点 $3$。

所以，新头节点 `newHead = 节点3.next`，即节点 $4$。断开 `节点3.next`，返回答案 $newHead$。

代码如下，已附加注释：

```Python
# python
class Solution:
    def rotateRight(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        # 边界情况
        if not head or not head.next or k == 0:
            return head
        
        # 计算链表长度
        # 找到原链表的尾节点
        n = 1
        old_tail = head
        while old_tail.next:
            old_tail = old_tail.next
            n += 1
        
        # 实际需要移动的步数
        k = k % n
        if k == 0:
            return head
        
        # 连成环
        old_tail.next = head
        
        # 找到新链表的尾节点：距离原头节点 n - k - 1 步
        # 新头节点就是新尾节点的下一个节点
        new_tail = head
        for _ in range(n - k - 1):
            new_tail = new_tail.next
        
        new_head = new_tail.next
        # 断开环
        new_tail.next = None
        return new_head
```

```Java
// java
class Solution {
    public ListNode rotateRight(ListNode head, int k) {
        // 边界情况
        if (head == null || head.next == null || k == 0) {
            return head;
        }

        // 计算链表长度
        // 找到原链表的尾节点
        int n = 1;
        ListNode old_tail = head;
        while (old_tail.next != null) {
            old_tail = old_tail.next;
            n++;
        }

        // 实际需要移动的步数
        k = k % n;
        if (k == 0) {
            return head;
        }

        // 连成环
        old_tail.next = head;

        // 找到新链表的尾节点：距离原头节点 n - k - 1 步
        // 新头节点就是新尾节点的下一个节点
        ListNode new_tail = head;
        for (int i = 0; i < n - k - 1; i++) {
            new_tail = new_tail.next;
        }

        ListNode new_head = new_tail.next;
        // 断开环
        new_tail.next = null;
        return new_head;
    }
}
```

- 时间复杂度： $O(n)$，其中 $n$ 为链表 $head$ 的节点数，两次遍历
- 空间复杂度： $O(1)$，仅使用常数个额外变量
