[Problem: 3567. 子矩阵的最小绝对差](https://leetcode.cn/problems/minimum-absolute-difference-in-sliding-submatrix/description/)

### 方法：暴力 & 滑窗

题意很清晰，在 $m\times n$ 的大矩阵中，遍历所有 $k\times k$ 的子正方形，寻找每个正方形内部 **不同值** 之间的最小绝对差。

限制正方形边长固定为 $k$，所以任意正方形都可以由它的 **左上角** 唯一确定。

自然地，想到使用 **暴力** 解法：

1. 遍历所有可能的左上角：对于 $m \times n$ 的矩阵，左上角 $(i, j)$ 的取值范围是 $0 \le i \le m-k$ 和 $0 \le j \le n-k$。
2. 提取子矩阵：对于每个起点 $(i, j)$，**提取** $k \times k$ 范围内的所有元素。
3. 去重与排序：
    - 将 $k^2$ 个元素放入集合 $Set$ 中去重。
    - 如果去重后只有一个元素，结果为 $0$。
    - 否则，将去重后的元素进行升序排序。
4. 计算差值：遍历排序后的数组，计算 **相邻** 两元素的差，记录最小值。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minAbsDiff(self, grid: List[List[int]], k: int) -> List[List[int]]:
        m, n = len(grid), len(grid[0])
        ans = [[0] * (n - k + 1) for _ in range(m - k + 1)]
        
        for i in range(m - k + 1):
            for j in range(n - k + 1):
                # 提取子矩阵并去重
                elements = set()
                for r in range(i, i + k):
                    for c in range(j, j + k):
                        elements.add(grid[r][c])
                
                # 只有一个值
                if len(elements) <= 1:
                    ans[i][j] = 0
                    continue
                
                # 排序
                sorted_elements = sorted(list(elements))
                min_diff = float('inf')

                # 寻找最小差
                for idx in range(1, len(sorted_elements)):
                    min_diff = min(min_diff, sorted_elements[idx] - sorted_elements[idx-1])
                ans[i][j] = min_diff
        
        return ans
```

```Java
// java
class Solution {
    public int[][] minAbsDiff(int[][] grid, int k) {
        int m = grid.length;
        int n = grid[0].length;
        int[][] ans = new int[m - k + 1][n - k + 1];
        
        for (int i = 0; i < m - k + 1; i++) {
            for (int j = 0; j < n - k + 1; j++) {
                // 提取子矩阵并去重
                Set<Integer> elements = new HashSet<>();
                for (int r = i; r < i + k; r++) {
                    for (int c = j; c < j + k; c++) {
                        elements.add(grid[r][c]);
                    }
                }
                
                // 只有一个值
                if (elements.size() <= 1) {
                    ans[i][j] = 0;
                    continue;
                }
                
                // 排序
                List<Integer> sortedElements = new ArrayList<>(elements);
                Collections.sort(sortedElements);
                int minDiff = Integer.MAX_VALUE;

                // 寻找最小差
                for (int idx = 1; idx < sortedElements.size(); idx++) {
                    minDiff = Math.min(minDiff, sortedElements.get(idx) - sortedElements.get(idx - 1));
                }
                ans[i][j] = minDiff;
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m \cdot n \cdot (k^2 \log(k^2)))$，其中 $m,n$ 是矩阵 $grid$ 的行数、列数，窗口总数约为 $m\cdot n$，每个窗口的大小为 $k^2$，对窗口使用快排
- 空间复杂度： $O(k^2)$，表示中间窗口 $elements$ 的大小

### 常数项优化

实际上，没必要使用 $set$ 去重，在遍历时 **跳过差值为 $0$ 的情况** 即可。

同时，使用 $<$ 判断代替 $min$ 函数速度更快。**提前剪枝**：如果 $diff$ 已经达到 $1$，对于整数矩阵来说不可能有更小的差值（不同值之间），可以直接结束。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minAbsDiff(self, grid: List[List[int]], k: int) -> List[List[int]]:
        m, n = len(grid), len(grid[0])
        res_m, res_n = m - k + 1, n - k + 1
        ans = [[0] * res_n for _ in range(res_m)]
        
        for i in range(res_m):
            for j in range(res_n):
                # 提取子矩阵
                elements = []
                for r in range(i, i + k):
                    elements.extend(grid[r][j : j + k])
                
                # 排序
                elements.sort()
                
                # 遍历寻找
                min_diff = float('inf')
                found_diff = False
                
                for idx in range(1, len(elements)):
                    diff = elements[idx] - elements[idx - 1]
                    # 只考虑不同值之间的差
                    if diff > 0:
                        if diff < min_diff:
                            min_diff = diff
                        found_diff = True
                        # 差值不可能小于 1
                        if min_diff == 1:
                            break
                
                # 所有元素都相同才是 0
                ans[i][j] = min_diff if found_diff else 0
        
        return ans
```

```Java
// java
class Solution {
    public int[][] minAbsDiff(int[][] grid, int k) {
        int m = grid.length;
        int n = grid[0].length;
        int resM = m - k + 1;
        int resN = n - k + 1;
        int[][] ans = new int[resM][resN];
        
        for (int i = 0; i < resM; i++) {
            for (int j = 0; j < resN; j++) {
                // 提取子矩阵
                List<Integer> elements = new ArrayList<>(k * k);
                for (int r = i; r < i + k; r++) {
                    for (int c = j; c < j + k; c++) {
                        elements.add(grid[r][c]);
                    }
                }
                
                // 排序
                Collections.sort(elements);
                
                // 遍历寻找
                int minDiff = Integer.MAX_VALUE;
                boolean foundDiff = false;
                
                for (int idx = 1; idx < elements.size(); idx++) {
                    int diff = elements.get(idx) - elements.get(idx - 1);
                    // 只考虑不同值之间的差
                    if (diff > 0) {
                        if (diff < minDiff) {
                            minDiff = diff;
                        }
                        foundDiff = true;
                        // 差值不可能小于 1
                        if (minDiff == 1) {
                            break;
                        }
                    }
                }
                
                // 所有元素都相同才是 0
                ans[i][j] = foundDiff ? minDiff : 0;
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m \cdot n \cdot (k^2 \log(k^2)))$，解法不变
- 空间复杂度： $O(k^2)$

经实测，这种 $Python$ 解法耗时 $27ms$，超过 $100\%$。

### 滑窗

当我们把窗口从 $(i, j)$ 向右移动到 $(i, j+1)$ 时：

- **移出**：左边那一列的 $k$ 个元素。
- **移入**：右边新的一列的 $k$ 个元素。

也就是说，我们需要一个数据结构，能快速执行：插入、删除、查询最小差。

因此，使用有序列表 $SortedList$ 维护升序。

具体来说，使用 **哈希表** $count$ 记录窗口内每个值出现的次数。只有当次数从 $0\to 1$ 或从 $1\to 0$ 时，才去更新 “唯一值集合”。**有序列表** $sl$ 存储当前窗口内的所有不同值。

如何寻找最小差值？

插入一个新值 $x$ 时，寻找它在 $sl$ 中的前驱 $prev$ 和后继 $next$，计算 $x-prev$ 和 $next-x$。而在本题中，数值 $k$ 较小，直接遍历有序列表反而更快。

代码如下，已附加注释：

```Python
# python
class Solution:
    def minAbsDiff(self, grid: List[List[int]], k: int) -> List[List[int]]:
        m, n = len(grid), len(grid[0])
        ans = [[0] * (n - k + 1) for _ in range(m - k + 1)]
        
        for i in range(m - k + 1):
            # 每一行开始时，重新初始化
            sl = SortedList()
            counts = {}
            
            def add(val):
                counts[val] = counts.get(val, 0) + 1
                if counts[val] == 1:
                    sl.add(val)
                
            def remove(val):
                counts[val] -= 1
                if counts[val] == 0:
                    sl.remove(val)
            
            def get_min_diff():
                if len(sl) <= 1:
                    return 0
                # 遍历寻找
                res = float('inf')
                for idx in range(1, len(sl)):
                    diff = sl[idx] - sl[idx-1]
                    if diff < res:
                        res = diff
                    if res == 1:  # 最小差不会小于 1
                        break
                return res
            
            # 初始化第一个窗口，起点是 (i, 0)
            for r in range(i, i + k):
                for c in range(k):
                    add(grid[r][c])
            ans[i][0] = get_min_diff()
            
            # 向右滑动
            for j in range(1, n - k + 1):
                # 移除左侧一整列
                for r in range(i, i + k):
                    remove(grid[r][j-1])
                # 添加右侧一整列
                for r in range(i, i + k):
                    add(grid[r][j+k-1])
                ans[i][j] = get_min_diff()
        
        return ans
```

```Java
// java
class Solution {
    private TreeSet<Integer> sl;
    private Map<Integer, Integer> counts;

    private void add(int val) {
        counts.put(val, counts.getOrDefault(val, 0) + 1);
        if (counts.get(val) == 1) {
            sl.add(val);
        }
    }

    private void remove(int val) {
        counts.put(val, counts.get(val) - 1);
        if (counts.get(val) == 0) {
            sl.remove(val);
        }
    }

    private int getMinDiff() {
        if (sl.size() <= 1) {
            return 0;
        }
        // 遍历寻找
        int res = Integer.MAX_VALUE;
        Integer prev = null;
        for (int curr : sl) {
            if (prev != null) {
                int diff = curr - prev;
                if (diff < res) {
                    res = diff;
                }
                if (res == 1) { // 最小差不会小于 1
                    break;
                }
            }
            prev = curr;
        }
        return res;
    }

    public int[][] minAbsDiff(int[][] grid, int k) {
        int m = grid.length;
        int n = grid[0].length;
        int[][] ans = new int[m - k + 1][n - k + 1];
        
        for (int i = 0; i < m - k + 1; i++) {
            // 每一行开始时，重新初始化
            sl = new TreeSet<>();
            counts = new HashMap<>();
            
            // 初始化第一个窗口，起点是 (i, 0)
            for (int r = i; r < i + k; r++) {
                for (int c = 0; c < k; c++) {
                    add(grid[r][c]);
                }
            }
            ans[i][0] = getMinDiff();
            
            // 向右滑动
            for (int j = 1; j < n - k + 1; j++) {
                // 移除左侧一整列
                for (int r = i; r < i + k; r++) {
                    remove(grid[r][j - 1]);
                }
                // 添加右侧一整列
                for (int r = i; r < i + k; r++) {
                    add(grid[r][j + k - 1]);
                }
                ans[i][j] = getMinDiff();
            }
        }
        
        return ans;
    }
}
```

- 时间复杂度： $O(m \cdot n \cdot k \log(k^2))$，内层滑动 $n$ 次，每次涉及 $2k$ 次平衡操作
- 空间复杂度： $O(k^2)$，表示列表 $sl$ 的大小

这种解法比暴力解法慢得多，可以将 $sl$ 换成普通列表，但是手动二分维护有序性，对应代码我放在下面的评论区，速度有小幅度提升。
