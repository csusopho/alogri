[Problem: 1980. 找出不同的二进制字符串](https://leetcode.cn/problems/find-unique-binary-string/description/)

### 方法：暴力 & 摩托尔对角线

长度为 $n$ 的二进制字符串总共有 $2^n$ 种组合，对应的十进制数值范围是 $0$ 到 $2^n - 1$。

因此，我们直接使用 **暴力**：枚举所有可能的 $2^n$ 组合，判断哪一个不存在 $nums$ 中。为了方便查找，使用哈希 $Set$ 将 $nums$ 中已有串进行存储。

代码如下，已附加注释：

```Python
# python
class Solution:
    def findDifferentBinaryString(self, nums: List[str]) -> str:
        n = len(nums)
        # 存到哈希中
        seen = set(nums)
        
        # 遍历所有十进制
        for i in range(2 ** n):
            # 转为二进制
            binary = bin(i)[2:].zfill(n)
            # 找到
            if binary not in seen:
                return binary
```

```Java
// java
class Solution {
    public String findDifferentBinaryString(String[] nums) {
        int n = nums.length;
        // 存到哈希中
        Set<String> seen = new HashSet<>(Arrays.asList(nums));
        
        // 遍历所有十进制
        for (int i = 0; i < (1 << n); i++) {
            // 转为二进制
            String binary = Integer.toBinaryString(i);
            while (binary.length() < n) {
                binary = "0" + binary;
            }
            
            // 找到
            if (!seen.contains(binary)) {
                return binary;
            }
        }
        return "";
    }
}
```

- 时间复杂度： $O(n^2)$，实际上达不到 $O(2^n)$，下面鸽巢原理会进行解释
- 空间复杂度： $O(n)$，表示哈希表 $seen$ 的大小，字符串长度为 $n$，严格就看作 $O(n^2)$

---

根据**鸽巢原理（抽屉原理）**：只需取十进制的 $0, 1, 2, \dots, n$ 这 $n+1$ 个数字，因为 $nums$ 里面只有 $n$ 个元素，所以这 $n+1$ 个数字中，**必定有一个数字不在 $nums$ 中**。

因此，不需要遍历所有 $2^n$ 个数字，只需要遍历 $0$ 到 $n$，将它们转换为二进制，看哪个不在 $nums$ 中即可。

代码如下，已附加注释：

```Python
class Solution:
    def findDifferentBinaryString(self, nums: List[str]) -> str:
        n = len(nums)
        # 转为十进制
        num_set = {int(s, 2) for s in nums}
        
        # 遍历 n+1 个数
        for i in range(n + 1):
            if i not in num_set:
                # 转换回字符串
                return format(i, f'0{n}b')
```

```Java
// java
class Solution {
    public String findDifferentBinaryString(String[] nums) {
        int n = nums.length;
        // 转为十进制
        Set<Integer> numSet = new HashSet<>();
        for (String s : nums) {
            numSet.add(Integer.parseInt(s, 2));
        }
        
        // 遍历 n+1 个数
        for (int i = 0; i <= n; i++) {
            if (!numSet.contains(i)) {
                // 转换回字符串
                String res = Integer.toBinaryString(i);
                while (res.length() < n) {
                    res = "0" + res;
                }
                return res;
            }
        }
        return "";
    }
}
```

- 时间复杂度： $O(n^2)$，将 $n$ 个长度为 $n$ 的字符串转为整数
- 空间复杂度： $O(n)$，表示哈希表 $set$ 的大小，这里存整数

### 摩托尔对角线

想要构造一个新的字符串 $ans$，只要保证跟 $nums$ 里的每一个字符串都不一样。

- 怎么保证 $ans$ 和 $nums[0]$ 不一样？ 只要让 $ans$ 的 **第 0 位** 和 $nums[0]$ 的 **第 0 位** 相反即可。
- 怎么保证 $ans$ 和 $nums[1]$ 不一样？ 只要让 $ans$ 的 **第 1 位** 和 $nums[1]$ 的 **第 1 位** 相反即可。
- $\dots$
- 怎么保证 $ans$ 和 $nums[i]$ 不一样？ 只要让 $ans$ 的 **第 i 位** 和 $nums[i]$ 的 **第 i 位** 相反即可。

**图形化理解：** 假设 $n = 3$, $nums =[000, 011, 101]$

提取对角线上的字符：

- $nums[0]$ = **0** 00
- $nums[1]$ = 0 **1** 1
- $nums[2]$ = 10 **1**

对角线字符串是 $011$，把对角线上的字符 **全部反转**，得到结果 $100$。

- $100$ 和 $000$ 不同，因为第 $0$ 位不同。
- $100$ 和 $011$ 不同，因为第 $1$ 位不同。
- $100$ 和 $101$ 不同，因为第 $2$ 位不同。

绝对完美避开了 $nums$ 中的所有字符串！

代码如下，已附加注释：

```Python
class Solution:
    def findDifferentBinaryString(self, nums: list[str]) -> str:
        res = []
        n = len(nums)
        
        # 遍历每一行
        for i in range(n):
            # 取出对角线元素并反转
            if nums[i][i] == '0':
                res.append('1')
            else:
                res.append('0')
        
        # 拼接
        return "".join(res)
```

```Java
// java
class Solution {
    public String findDifferentBinaryString(String[] nums) {
        StringBuilder res = new StringBuilder();
        int n = nums.length;
        
        // 遍历每一行
        for (int i = 0; i < n; i++) {
            // 取出对角线元素并反转
            if (nums[i].charAt(i) == '0') {
                res.append('1');
            } else {
                res.append('0');
            }
        }
        
        // 拼接
        return res.toString();
    }
}
```

- 时间复杂度： $O(n)$，一次遍历
- 空间复杂度： $O(n)$
