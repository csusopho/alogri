[Problem: 657. 机器人能否返回原点](https://leetcode.cn/problems/robot-return-to-origin/description/)

### 方法：模拟

建立一个 **二维直角坐标系**，机器人的初始位置在原点 $(x, y) = (0, 0)$。

根据题意，每种移动对坐标的影响如下：

- **U**：纵坐标增加 $1$，即 $y = y + 1$。
- **D**：纵坐标减少 $1$，即 $y = y - 1$。
- **L**：横坐标减少 $1$，即 $x = x - 1$。
- **R**：横坐标增加 $1$，即 $x = x + 1$。

机器人要回到原点 $(0, 0)$，必须满足两个条件：

1. **水平方向相互抵消**：向左移动的次数必须等于向右移动的次数，即 $x$ 的最终值为 $0$。
2. **垂直方向相互抵消**：向上移动的次数必须等于向下移动的次数，即 $y$ 的最终值为 $0$。

模拟坐标变换，一次遍历，实时更新机器人的坐标位置。

代码如下，已附加注释：

```Python
# python
class Solution:
    def judgeCircle(self, moves: str) -> bool:
        x, y = 0, 0
        
        # 模拟
        for move in moves:
            if move == 'U':
                y += 1
            elif move == 'D':
                y -= 1
            elif move == 'L':
                x -= 1
            elif move == 'R':
                x += 1
        
        # 判断
        return x == 0 and y == 0
```

```Java
// java
class Solution {
    public boolean judgeCircle(String moves) {
        int x = 0, y = 0;
        
        // 模拟
        for (char move : moves.toCharArray()) {
            if (move == 'U') {
                y += 1;
            } else if (move == 'D') {
                y -= 1;
            } else if (move == 'L') {
                x -= 1;
            } else if (move == 'R') {
                x += 1;
            }
        }
        
        // 判断
        return x == 0 && y == 0;
    }
}
```

- 时间复杂度： $O(n)$，其中 $n$ 是串 $moves$ 的长度
- 空间复杂度： $O(1)$，仅使用常数个额外变量
