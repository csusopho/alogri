[Problem: 3783. 整数的镜像距离](https://leetcode.cn/problems/mirror-distance-of-an-integer/description/)

### 方法：模拟

如何反转一个整数？

- 转为字符串，反转，再转回整数。
- 从低位到高位，先 $\times 10$ 再累加最低位，不过极易溢出。

这里采用第一种方法，评论区附加第二种。

代码如下，已附加注释：

```Python
# python
class Solution:
    def mirrorDistance(self, n: int) -> int:
        # 转为字符串
        s = str(n)
        # 反转并转换回整数
        reversed_n = int(s[::-1])
        # 计算
        return abs(n - reversed_n)
```

```Java
// java
public class Solution {
    public int mirrorDistance(int n) {
        // 转为字符串
        String s = String.valueOf(n);
        // 反转并转换回整数
        String reversedStr = new StringBuilder(s).reverse().toString();
        long reversedN = Long.parseLong(reversedStr);
        // 计算
        return (int) Math.abs((long) n - reversedN);
    }
}
```

- 时间复杂度： $O(log_2{n})$，变字符串+反转
- 空间复杂度： $O(1)$，或者 $O(log_2{n})$，仅使用常数个额外变量
