[Problem: 1622. 奇妙序列](https://leetcode.cn/problems/fancy-sequence/description/)

### 线段树 & 数学

根据题意可知，我们需要一个数据结构支持：

1. **区间加法**：对区间 $[0,R]$ 每个数增加 $inc$。
2. **区间乘法**：对区间 $[0,R]$ 每个数乘以 $m$。
3. **单点查询**：找出下标为 $idx$ 处的数值。

什么数据结构专门处理 “区间修改/查询”？**线段树**。

线段树的本质是将一个数组拆分成一段段的区间。对于本题，上述操作可以理解为：

- $append$: 在数组末尾（下标 $n$）放入一个数。
- $addAll$: 对区间 $[0, n-1]$ 进行加法更新。
- $multAll$: 对区间 $[0, n-1]$ 进行乘法更新。
- $getIndex$: 查询下标为 $idx$ 的点的值。

为了保证效率，线段树使用 **“懒标记”**：当我们修改一个区间时，不立刻更新该区间内所有的叶子节点，而是在对应区间节点上打一个 “记号”，等下次真正需要访问下面的节点时，再把这个记号传下去。

每个节点维护 **两个** 懒标记：乘法标记 $mul$ 和加法标记 $add$。当一个值 $x$ 依次收到乘法 $m$ 和加法 $inc$ 时，变换公式为：

$$
x = x \cdot m + inc
$$

乘法和加法的标记如何传递？假设一个节点原本的变换是 $f(x) = x \cdot m_{old} + a_{old}$。现在来了一个新的变换 $g(x) = x \cdot m_{new} + a_{new}$。合并后的变换应该是 $g(f(x))$：

$$
g(f(x)) = (x \cdot m_{old} + a_{old}) \cdot m_{new} + a_{new}
$$

展开后可得：

$$
x \cdot (m_{old} \cdot m_{new}) + (a_{old} \cdot m_{new} + a_{new})
$$

也就是说，新增一个乘法，乘法系数部分是累乘，加法尾缀也要累乘；新增一个加法，乘法部分不变，而加法尾缀累加（代码中 $apply$ 的逻辑）。

假如你想全面了解线段树，可以去看看 [维基百科](https://oi-wiki.org/ds/seg/)，有详细阐释，不过稍微有点难理解，对新手不太友好。推荐一篇我写过的 [题解](https://leetcode.cn/problems/booking-concert-tickets-in-groups/solutions/2933254/xian-duan-shu-yuan-li-mo-ban-xiang-xi-tu-dlmy/)，里面介绍了链表形式和数组形式的线段树，以及 **动态开点** 做法，包含 *原理*、*模版代码*、*经典例题*。

对区间进行加减时，使用 $Lazy$ 懒标记，单次更新时间从 $O(n)$ 变为 $O(logn)$。

这种 **正向思维** 比较好理解： $addAll$ 和 $multAll$ 就是对现有的区间整体打个标签。查询的时候，顺着树往下走，把路上积攒的标签（乘了多少、加了多少）依次应用在数字上即可。

在辅助线段树部分，套用模版中的 $pushDown$ 以及 $update$ 等函数。在主体部分，模拟题意，完成区间修改以及单点查询操作。

```Python
# python
class SegmentTree:
    def __init__(self, n):
        self.n = n
        self.mod = 10**9 + 7
        suit_len = 2 << self.n.bit_length()
        # 区间值
        self.tree_val = [0] * suit_len
        # 懒标记：乘法标记初始为 1，加法标记初始为 0
        self.lazy_mul = [1] * suit_len
        self.lazy_add = [0] * suit_len

    # 对一个节点应用 x * m + a 的变换
    def apply_transform(self, node, m, a):
        # 更新当前节点的值
        self.tree_val[node] = (self.tree_val[node] * m + a) % self.mod
        # 更新懒标记
        self.lazy_mul[node] = (self.lazy_mul[node] * m) % self.mod
        self.lazy_add[node] = (self.lazy_add[node] * m + a) % self.mod

    # 向下传递乘法和加法标记
    def push_down(self, node):
        m = self.lazy_mul[node]
        a = self.lazy_add[node]
        if m == 1 and a == 0:
            return
        
        # 传递给左、右子节点
        self.apply_transform(node * 2, m, a)
        self.apply_transform(node * 2 + 1, m, a)
        
        # 重置当前节点的标记
        self.lazy_mul[node] = 1
        self.lazy_add[node] = 0

    # 区间更新：对 [l, r] 范围应用 x * m + a
    def update_range(self, node, start, end, l, r, m, a):
        # 区间不重叠，直接返回
        if l > end or r < start:
            return
        # 当前区间完全包含在更新范围内
        if l <= start and end <= r:
            self.apply_transform(node, m, a)
            return
        
        # 下放标记，递归更新子节点
        self.push_down(node)
        mid = (start + end) // 2
        self.update_range(node * 2, start, mid, l, r, m, a)
        self.update_range(node * 2 + 1, mid + 1, end, l, r, m, a)

    # 单点操作：在末尾插入新数
    def point_set(self, node, start, end, idx, val):
        # 叶子节点
        if start == end:
            self.tree_val[node] = val
            self.lazy_mul[node] = 1
            self.lazy_add[node] = 0
            return
        # 递归向左右寻找
        self.push_down(node)
        mid = (start + end) // 2
        if idx <= mid:
            self.point_set(node * 2, start, mid, idx, val)
        else:
            self.point_set(node * 2 + 1, mid + 1, end, idx, val)

    # 单点查询：获取 idx 的值
    def query_point(self, node, start, end, idx):
        if start == end:
            return self.tree_val[node]
        
        self.push_down(node)
        mid = (start + end) // 2
        if idx <= mid:
            return self.query_point(node * 2, start, mid, idx)
        else:
            return self.query_point(node * 2 + 1, mid + 1, end, idx)

class Fancy:
    def __init__(self):
        self.max_n = 100000
        self.st = SegmentTree(self.max_n)
        self.length = 0  # 当前序列长度

    def append(self, val: int) -> None:
        # 向末尾插入值
        self.st.point_set(1, 0, self.max_n - 1, self.length, val)
        self.length += 1

    def addAll(self, inc: int) -> None:
        if self.length > 0:
            # 全部加上 inc (x * 1 + inc)
            self.st.update_range(1, 0, self.max_n - 1, 0, self.length - 1, 1, inc)

    def multAll(self, m: int) -> None:
        if self.length > 0:
            # 全部乘以 m (x * m + 0)
            self.st.update_range(1, 0, self.max_n - 1, 0, self.length - 1, m, 0)
    
    def getIndex(self, idx: int) -> int:
        if idx >= self.length:
            return -1
        return self.st.query_point(1, 0, self.max_n - 1, idx)
```

```Java
// java
class SegmentTree {
    int n;
    long mod = 1_000_000_007;
    // 区间值
    long[] tree_val;
    // 懒标记：乘法标记初始为 1，加法标记初始为 0
    long[] lazy_mul;
    long[] lazy_add;

    public SegmentTree(int n) {
        this.n = n;
        int suit_len = 4 * n;
        tree_val = new long[suit_len];
        lazy_mul = new long[suit_len];
        Arrays.fill(lazy_mul, 1);
        lazy_add = new long[suit_len];
    }

    // 对一个节点应用 x * m + a 的变换
    public void apply_transform(int node, long m, long a) {
        // 更新当前节点的值
        tree_val[node] = (tree_val[node] * m + a) % mod;
        // 更新懒标记
        lazy_mul[node] = (lazy_mul[node] * m) % mod;
        lazy_add[node] = (lazy_add[node] * m + a) % mod;
    }

    // 向下传递乘法和加法标记
    public void push_down(int node) {
        long m = lazy_mul[node];
        long a = lazy_add[node];
        if (m == 1 && a == 0) {
            return;
        }

        // 传递给左、右子节点
        apply_transform(node * 2, m, a);
        apply_transform(node * 2 + 1, m, a);

        // 重置当前节点的标记
        lazy_mul[node] = 1;
        lazy_add[node] = 0;
    }

    // 区间更新：对 [l, r] 范围应用 x * m + a
    public void update_range(int node, int start, int end, int l, int r, long m, long a) {
        // 区间不重叠，直接返回
        if (l > end || r < start) {
            return;
        }
        // 当前区间完全包含在更新范围内
        if (l <= start && end <= r) {
            apply_transform(node, m, a);
            return;
        }

        // 下放标记，递归更新子节点
        push_down(node);
        int mid = (start + end) / 2;
        update_range(node * 2, start, mid, l, r, m, a);
        update_range(node * 2 + 1, mid + 1, end, l, r, m, a);
    }

    // 单点操作：在末尾插入新数
    public void point_set(int node, int start, int end, int idx, int val) {
        // 叶子节点
        if (start == end) {
            tree_val[node] = val;
            lazy_mul[node] = 1;
            lazy_add[node] = 0;
            return;
        }
        // 递归向左右寻找
        push_down(node);
        int mid = (start + end) / 2;
        if (idx <= mid) {
            point_set(node * 2, start, mid, idx, val);
        } else {
            point_set(node * 2 + 1, mid + 1, end, idx, val);
        }
    }

    // 单点查询：获取 idx 的值
    public long query_point(int node, int start, int end, int idx) {
        if (start == end) {
            return tree_val[node];
        }

        push_down(node);
        int mid = (start + end) / 2;
        if (idx <= mid) {
            return query_point(node * 2, start, mid, idx);
        } else {
            return query_point(node * 2 + 1, mid + 1, end, idx);
        }
    }
}

class Fancy {
    int max_n = 100001;
    SegmentTree st;
    int length = 0; // 当前序列长度

    public Fancy() {
        st = new SegmentTree(max_n);
    }

    public void append(int val) {
        // 向末尾插入值
        st.point_set(1, 0, max_n - 1, length, val);
        length++;
    }

    public void addAll(int inc) {
        if (length > 0) {
            // 全部加上 inc (x * 1 + inc)
            st.update_range(1, 0, max_n - 1, 0, length - 1, 1, inc);
        }
    }

    public void multAll(int m) {
        if (length > 0) {
            // 全部乘以 m (x * m + 0)
            st.update_range(1, 0, max_n - 1, 0, length - 1, m, 0);
        }
    }

    public int getIndex(int idx) {
        if (idx >= length) {
            return -1;
        }
        return (int) st.query_point(1, 0, max_n - 1, idx);
    }
}
```

假设 $append$ 次数为 $n$，也即序列最大长度为 $n$，题目规定 $n\leq 10^5$。

- 时间复杂度：四种操作都是 $O(logn)$
- 空间复杂度： $O(2^{\left\lceil\log{n}\right\rceil +1})$，表述线段树数组 $tree$ 以及标记数组 $lazy$ 的大小

### 数学

所有的 $addAll$ 和 $multAll$ 操作本质上都是对序列中的元素进行 **线性变换**。假设一个数初始数值是 $x$，经过一系列操作后，它会变成：

$$
y = a \cdot x + b
$$

- **addAll** 操作：原本是 $y = a \cdot x + b$，加上 $inc$ 后变成 $y = a \cdot x + (b + inc)$。
- **multAll** 操作：原本是 $y = a \cdot x + b$，乘以 $m$ 后变成 $y = m \cdot (a \cdot x + b)$，也即 $(m \cdot a) \cdot x + (m \cdot b)$。

**结论一**：不管怎么操作，数字永远都符合 $a \cdot x + b$ 的格式。

$append$ 是在 **当前** 时间点新加入一个数字，它不应该受到 **之前** 发生的 $addAll$ 和 $multAll$ 的影响，但会受到 **之后** 所有操作的影响。但怎么知道，一个数字 $x$，到底操作了多少次？

因此，我们尽量 **一视同仁**。

从前往后进行 $append$ 时，总共的操作次数为 $n$，套用上面的公式后很轻松得到 **第一个** 数字操作后的 $a,b$ 系数。但是，第二个数字怎么计算？

其实，第二次数字是操作了 $n-1$ 次，相当于少操作一次，直接套用 $n$ 次操作会多出一次！由于 $n$ 加上 $-1$ 等于 $n-1$，因此联想到 **逆向操作** 一次！假设第二个数字原本值为 $x$，逆向操作一次变为 $t$，那么 $t$ 操作 $n$ 次就是正确值！

**结论二**：维护一对全局的系数 $(a, b)$，表示 **从序列开始到现在** 所有 $n$ 次操作的累积变换。同时，序列中新添加的值，不再是原始值 $x$，而是 “特殊值” $v'$。

这个特殊值 $v'$，在经过前面所有次全局变换 $y = a \cdot v' + b$ 后，这个值刚好等于新加入的 $val$：

$$
a \cdot v' + b \equiv val
$$

先向右移项，然后求解这个方程，得到 $v' \equiv (val - b) \cdot a^{-1}$。

其中 $a^{-1}$ 是 $a$ 在模 $M$ 意义下的 **乘法逆元**。由于 $M = 10^9 + 7$ 是质数，我们可以根据费马小定理，利用快速幂求得 $a^{-1} = a^{M-2} \pmod M$。

**总结**：让序列中存储 “虚假值”，它经过全局变换后，得到的是正确的目标值。这样一来，无需考虑每个数值到底经过几次变化，只需维护一次全局变换即可。

```Python
# python
class Fancy:
    def __init__(self):
        self.mod = 10**9 + 7
        self.nums = []
        # 全局增量变换公式: y = a * x + b
        self.a = 1
        self.b = 0

    def append(self, val: int) -> None:
        # 利用费马小定理计算乘法逆元
        inv_a = pow(self.a, self.mod - 2, self.mod)
        # 找到这个 v'
        v_prime = ((val - self.b) * inv_a) % self.mod
        self.nums.append(v_prime)

    def addAll(self, inc: int) -> None:
        # 增加，只更新b
        self.b = (self.b + inc) % self.mod

    def multAll(self, m: int) -> None:
        # 相乘，更新a和b
        self.a = (self.a * m) % self.mod
        self.b = (self.b * m) % self.mod

    def getIndex(self, idx: int) -> int:
        if idx >= len(self.nums):
            return -1
        # 应用当前的全局变换 (a, b)
        return (self.a * self.nums[idx] + self.b) % self.mod
```

```Java
// java
class Fancy {
    long mod = 1000000007L;
    List<Long> nums;
    // 全局增量变换公式: y = a * x + b
    long a;
    long b;

    public Fancy() {
        nums = new ArrayList<>();
        a = 1;
        b = 0;
    }

    public void append(int val) {
        // 利用费马小定理计算乘法逆元
        long inv_a = power(a, mod - 2);
        // 找到这个 v'
        long v_prime = ((val - b + mod) % mod * inv_a) % mod;
        nums.add(v_prime);
    }

    public void addAll(int inc) {
        // 增加，只更新b
        b = (b + inc) % mod;
    }

    public void multAll(int m) {
        // 相乘，更新a和b
        a = (a * m) % mod;
        b = (b * m) % mod;
    }

    public int getIndex(int idx) {
        if (idx >= nums.size()) {
            return -1;
        }
        // 应用当前的全局变换 (a, b)
        return (int) ((a * nums.get(idx) + b) % mod);
    }

    // 快速幂计算 (x^y) % mod
    private long power(long x, long y) {
        long res = 1;
        x %= mod;
        while (y > 0) {
            if ((y & 1) == 1) res = (res * x) % mod;
            y >>= 1;
            x = (x * x) % mod;
        }
        return res;
    }
}
```

- 时间复杂度：只有 $append$ 操作是 $O(logM)$，其中操作均为 $O(1)$，且 $M=10^9+7$
- 空间复杂度： $O(n)$，表示序列 $nums$ 的大小
